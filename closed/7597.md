## Summary

Current `main` already implements plugin-level `before_tool_call` and `after_tool_call` hooks, with typed payloads including `toolCallId`/`runId`, runtime wiring on tool execution paths, documentation, regression coverage, and a changelog entry showing the feature shipped in `v2026.2.9`.
