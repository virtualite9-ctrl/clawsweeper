## Summary

Current `main` already implements this PR’s behavior, and more. The new `splitTrailingAuthProfile()` helper preserves `@YYYYMMDD` model suffixes, still allows a trailing auth profile after the version suffix, and is wired into both shared model resolution and simple-completion selection. The added regression coverage on `main` includes the exact `@20251001` cases from this PR, so this PR is superseded.
