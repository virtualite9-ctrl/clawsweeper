## Summary

Current main does not show an OpenClaw bug where Feishu Developer Console personnel availability should unlock OpenClaw capabilities. The Feishu integration is an OpenClaw bot/channel with its own DM/group allowlists; Feishu-native calendar/meeting Agent capability publishing is outside this repo. If the intended path is OpenClaw paired-device calendar creation, that is controlled by existing owner-only and high-risk node-command configuration, not by Feishu app visibility settings.

## Best Possible Solution

Close this as support/external configuration, keep the secure OpenClaw defaults, and direct admins to configure the existing Feishu dm/group allowlists plus commands.ownerAllowFrom and gateway.nodes.allowCommands for paired-device calendar creation. Any Feishu-native calendar/meeting capability should be a separate plugin/ClawHub or plugin-API discussion.
