## Summary

Current main no longer treats Feishu as an external plugin-install path: Feishu is bundled, bare `feishu` installs resolve to the bundled source before npm, and the supported setup flow is now `openclaw channels login --channel feishu` on 2026.4.24+.

## Evidence
