## Summary

Close as ClawHub. #53580 is a new optional ACP provider plugin, not a core regression, and current main already exposes the relevant plugin/provider, ACP runtime, and agent-harness surfaces for this kind of experiment outside the core repository. The existing bundled `acpx` plugin covers ACP harness sessions, while the proposed ACP-as-provider bridge is better maintained and iterated as an external ClawHub/npm plugin. PR review also identified unresolved implementation bugs, so this should not be merged as a bundled plugin as-is.

## Best Possible Solution

Do not merge this as a bundled OpenClaw plugin. Encourage the author to publish the ACP provider experiment through ClawHub/npm using the existing provider and, if needed, trusted agent-harness or ACP runtime surfaces; any concrete missing generic SDK capability should be split into a small, reviewable API proposal.
