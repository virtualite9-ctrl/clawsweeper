## Summary

Issue #40908 asks for a service-specific Feishu Drive large-file upload tool. Current main still does not implement `feishu_drive.upload_file` or Feishu multipart upload, but the request is an optional Feishu/Drive tool capability that fits the ClawHub/plugin path rather than OpenClaw core. Related Feishu Drive upload cleanup already points the same capability to #48025/#19723, with `feishu_doc upload_file` confirmed as a different document-scoped surface.

## Best Possible Solution

Close this OpenClaw core issue and route the large-file Feishu Drive uploader to a ClawHub skill/plugin or community Feishu add-on using `api.registerTool` and the existing runtime config surface. Reopen or file a narrower OpenClaw issue only if that implementation exposes a concrete missing plugin API or security/product reason for bundled promotion.
