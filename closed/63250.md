## Summary

Current `main` no longer validates required plugin config for a bare disabled plugin entry, which matches the requested fix. The v2026.4.8 regression validated any present plugin entry; current `main` only validates enabled plugins or entries with an explicit `config` block, and the changelog records that fix under v2026.4.10.

## Evidence
