## Summary

Current `main` already fixes the blocking behavior this PR targeted: OpenClaw now races manual Codex OAuth input and immediately falls back to pasted redirect/code when the local callback never starts, so the reported hang is no longer present. This PR's extra port-owner diagnostics are incremental polish, not a missing fix on `main`.

## Evidence
