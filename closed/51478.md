## Summary

Current `main` already implements the OpenClaw-side fix for this report: sandbox `/workspace/...` media paths are remapped to the host sandbox root before validation, and outbound local-media roots include the active agent workspace (`workspace-<agentId>`). The shared message-action path applies that normalization before plugin/gateway dispatch, with regression coverage in source.
