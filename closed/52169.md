## Summary

Current `main` already allows same-origin microphone access on Gateway responses, which covers Control UI requests. The Web UI dictation button still uses browser-native `SpeechRecognition`, so the reported browser-side block is addressed on `main` by commit `665cc3d5371c275b32c2c400b54238e5d96f6761` even though it has not shipped in the latest tagged release yet.
