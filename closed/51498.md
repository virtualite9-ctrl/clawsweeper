## Summary

Current `main` already covers the reported failure mode: legacy cron stores are normalized by `openclaw doctor --fix`, cron admin read paths have regression coverage showing `cron list`/`cron status` stay responsive while jobs are running, and gateway `health` now serves a warm cached snapshot with background refresh instead of forcing every non-verbose call through a live probe.
