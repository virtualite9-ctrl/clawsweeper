## Summary

Feishu Sheets support is better handled as a native OpenClaw plugin published via ClawHub/npm, not as another core cleanup item. Current main already provides the relevant extension seams: plugin-distributed agent tools, plugin config/runtime config access, and an existing Feishu plugin pattern that reads Feishu account credentials from `api.config` for Feishu-specific tools.

## Evidence
