## Summary

Current main addresses the user-visible Control UI reload flash/lost-message problem with broader history-application and render-layer fixes. PR #63537’s exact predicate change is now stale: main intentionally reloads final chat events, including final events with no message payload, and tests assert that contract.

## Best Possible Solution
