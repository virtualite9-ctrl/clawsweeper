## Summary

Close as `clawhub`: this PR is a self-contained optional plugin, current main explicitly directs that class of work to external plugin distribution/ClawHub rather than bundling more core plugins, and the author already has a standalone `self-evolve` plugin repo available for users.

## Evidence

- **Vision scope for optional plugins:** Current main says OpenClaw is slimming core while expanding plugin capabilities, that plugins should be hosted and maintained in their own repository, that the bar for adding optional plugins to core is intentionally high, and that plugin promotion belongs in ClawHub. It also notes memory is an exclusive slot and the project plans to converge on one recommended path rather than keep growing bundled options.
