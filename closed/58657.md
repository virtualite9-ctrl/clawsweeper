## Summary

Current main no longer matches the reported durable-history failure mode: the bundled ACPX runtime is pinned to acpx 0.6.0, OpenClaw uses its JSON file session store under plugin state, and acpx 0.6.0 bounds persisted conversation/tool output instead of replaying raw `.stream*.ndjson` session updates as durable history.
