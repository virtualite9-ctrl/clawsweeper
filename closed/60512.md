## Summary

Current main routes `config.patch` through the normal config validator, and that validator now explicitly accepts `agents.defaults.compaction.customInstructions`, `postCompactionSections`, and `model`. The latest verifiable release tag (`v2026.4.23`) also contains those validator entries and matching generated schema/help metadata, so the 2026.4.2 mismatch is already resolved in shipped code.
