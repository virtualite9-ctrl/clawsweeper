## Summary

Close #69200 as duplicate/superseded by the narrower open fix PR #54899. Current main still sends Feishu streaming-card markdown content through the Card Kit update path without a Card Kit markdown sanitizer, so this is not fixed on main; the remaining truncation work is already tracked by #54899, which explicitly targets #26708 and proposes sanitizing streaming-card markdown/backticks/angle brackets. The related thinking-tag leakage mentioned in the body is separately addressed on current main for streamed partials by strict reasoning-tag stripping and a regression test.

## Best Possible Solution

Close this reopened umbrella issue as a duplicate of the focused open PR #54899. Maintainers should continue review there, with the likely code path in `extensions/feishu/src/streaming-card.ts`: sanitize Card Kit markdown before update/close payloads and add regression tests for unmatched backticks, bare angle brackets, and summaries.
