## Summary

Current main has dedicated handling for the cron LiveSessionModelSwitchError path: isolated cron runs force a fresh run session, stale runtime model/session state is stripped, model-switch retries are bounded, and fallback no longer rethrows LiveSessionModelSwitchError as an unknown outer-loop trigger.
