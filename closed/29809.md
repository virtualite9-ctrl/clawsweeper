## Summary

Current `main` already implements the behavior needed for this report: `gateway.controlUi.allowedOrigins` now accepts `"*"`, the gateway seeds missing Control UI origins for upgraded non-loopback installs, and the docs explicitly describe exact-origin configuration for remote Control UI deployments. The shipped `v2026.4.23` tag contains the wildcard allowlist support as well.

## Evidence
