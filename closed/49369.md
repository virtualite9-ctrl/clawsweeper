## Summary

Current `main` already carries the NVIDIA-prefixed model-id path needed for this report. The bundled NVIDIA catalog now stores Nemotron with upstream id `nvidia/nemotron-3-super-120b-a12b`, the docs instruct selecting it as `nvidia/nvidia/...`, model-ref parsing preserves nested ids after the first provider slash, and the OpenAI-compatible transport sends `model.id` verbatim in the request payload instead of stripping the upstream `nvidia/` segment.
