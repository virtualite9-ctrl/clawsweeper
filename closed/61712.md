## Summary

Current main and the latest release load configured Telegram as a channel plugin, start channel sidecars on gateway startup, and start Telegram polling from the Telegram gateway hook. The reported 2026.4.2 symptom where `channels.telegram.botToken` left the Channels table empty after a LaunchAgent restart is covered by the current startup and auto-enable path.
