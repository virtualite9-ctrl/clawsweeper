## Summary

Current `main` already fixes the missing-`model.input` crash path by normalizing omitted input metadata to `["text"]` before runtime use, adds regression coverage for that case, and already contains the null-safe guards on the scan/vision paths. This PR is no longer needed, and it also bundles an unrelated Feishu key-validation change that current `main` intentionally does not take.
