## Summary

Current `main` already contains targeted lifecycle guards for this class of post-abort/post-timeout pi-agent-core late-event failure: OpenClaw now suppresses late exec/tool updates after run teardown and waits for the embedded agent to become idle before flushing pending tool results during attempt/compaction cleanup. The fix is present on `main` but does not appear to be in a released tag yet.

## Evidence
