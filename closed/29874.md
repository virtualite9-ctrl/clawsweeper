## Summary

The request is a broad optional model-routing feature. Current main already supports provider-owned routing through model-provider plugins such as OpenRouter, Kilo Gateway, and LiteLLM, while core model selection remains explicit primary/fallback configuration. Per VISION.md, this kind of optional routing policy belongs in a provider/plugin distributed through ClawHub rather than new OpenClaw core behavior.
