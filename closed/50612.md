## Summary

Current main, and the inspected v2026.4.24 release tag, already fix the dashboard failure mode for supported gateway token SecretRefs/env-template refs: the dashboard resolves configured token refs through the shared SecretRef resolver, avoids embedding SecretRef-managed tokens in generated URLs, and has regression coverage proving no literal token fragment is emitted.
