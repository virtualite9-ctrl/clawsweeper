## Summary

Current `main` already implements this: the Ollama plugin declares and exports a memory embedding provider, the runtime resolves manifest-declared memory embedding providers for cold/standalone CLI paths, regression tests cover `getMemoryEmbeddingProvider("ollama")`, docs list `ollama` as a supported explicit memory provider, and the fix is recorded in the `2026.4.14` changelog.

## Evidence
