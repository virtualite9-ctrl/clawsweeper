## Summary

Passive execution-health warnings fit the existing plugin/hook architecture, not OpenClaw core: current direction says optional capability should ship as plugins, the runtime already exposes tool/turn observation hooks, and `extensions/skill-workshop` is an in-tree example of post-turn analysis driven by `agent_end` and tool-count thresholds.
