## Summary

Current main has removed the legacy Chrome extension relay path rather than keeping the broken HMAC relay flow. The browser server stub is now an intentional no-op, new `driver: "extension"` profiles are rejected, and doctor migrates legacy extension-relay configs to `existing-session` Chrome MCP attach.
