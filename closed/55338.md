## Summary

Current `main` already implements the requested outbound recovery: missing deliverable channel plugins are bootstrapped even when the active registry is non-empty but missing that channel, and regression tests cover the stale/partial-registry cases. The same implementation is present in release `v2026.4.22`.

## Evidence
