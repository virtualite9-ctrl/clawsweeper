## Summary

Current `main` already covers the underlying need with a narrower implementation: outbound media access now adds the active agent’s concrete workspace root and, when filesystem policy allows, can widen to the exact parent directory of explicit media sources. The PR’s broader `workspace-*` glob approach is therefore obsolete and weaker than the shipped behavior.
