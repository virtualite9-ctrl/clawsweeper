## Summary

Current `main` no longer delivers `openclaw message send --dry-run` traffic: dry-run send/poll paths return before gateway/direct delivery, plugin dispatch is skipped for dry-run, and regression tests cover both send and poll. The same implementation is already present in the shipped `v2026.4.23` release.
