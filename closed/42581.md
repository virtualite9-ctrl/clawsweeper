## Summary

Current OpenClaw already fixes the reported ACP voice-note regression through the newer `reply_dispatch`/`dispatch-acp` path: ACP-bound turns are routed through the bundled ACPX hook, and `dispatch-acp.ts` now runs media understanding before building the ACP prompt. The same implementation is present in released tag `v2026.4.23`, so this PR is stale against current main.

## Evidence
