## Summary

Current `main` already speaks gateway protocol v3, attaches device identity for token-auth gateway calls, and reads the challenge nonce from `payload.nonce`; the reported `openclaw cron run` handshake failure is obsolete on current code and appears shipped in `v2026.4.22`.
