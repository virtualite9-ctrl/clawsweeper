## Summary

Current `main` already provides the requested post-turn capture surface through the existing plugin hook API: `agent_end` for completed turns, `after_tool_call` for structured tool results, and `subagent_ended` plus subagent runtime helpers for subagent completion/indexing workflows. Bundled plugins already use these hooks for automatic capture, and the capability is present in the shipped `v2026.4.23` release.
