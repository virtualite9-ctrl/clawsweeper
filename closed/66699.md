## Summary

Current Telegram menu handling already covers this payload-budget failure: it trims descriptions, drops trailing commands if the payload still cannot fit, retries `setMyCommands` on `BOT_COMMANDS_TOO_MUCH`, logs the reduced registration, and the shipped `v2026.4.23` tree already contains that logic.

## Evidence
