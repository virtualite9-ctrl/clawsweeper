## Summary

Current `main` implements the requested behavior: text-only gateway sessions no longer drop inline images before fallback handling, and media-understanding now prefers explicit `agents.defaults.imageModel` routing over the active session model. The latest tagged release `v2026.4.23` still had the old drop path, so this appears fixed on `main` but not yet in a release.

## Evidence
