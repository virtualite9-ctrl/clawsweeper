## Summary

Current main fixes the reported Control UI/WebChat image attachment delivery path: the browser sends attachments in `chat.send`, the gateway accepts attachment-only turns, normalizes/parses image payloads, forwards inline images to the agent run, and offloads images for text-only session models instead of dropping them.
