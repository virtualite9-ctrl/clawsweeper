## Summary

Current main no longer matches the reported reproduction: channel config changes are planned as hot reloads or no-op runtime reads, not gateway SIGUSR1 restarts, so the described Discord-channel config update should not abort the active agent session on current `main`.

## Evidence

- **Docs: channel, agent, and binding changes are non-restart:** Gateway docs say `channels.*` does not require a restart, and `agent`, `agents`, `routing`, and `bindings` are also non-restart surfaces in hybrid mode.
  - file: [docs/gateway/configuration.md:530](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/docs/gateway/configuration.md#L530)

  - sha: [78a431ed224b](https://github.com/openclaw/openclaw/commit/78a431ed224b6f5b9d58672346551a5b411ad200)
- **Reload planner: Discord channel config is hot-reloadable:** Bundled channel plugins contribute `reload.configPrefixes`; the Discord plugin registers `channels.discord`, which the reload planner maps to hot reload with `restart-channel:discord`, not `restartGateway`.
  - file: [extensions/discord/src/shared.ts:111](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/extensions/discord/src/shared.ts#L111)
  - command: `nl -ba extensions/discord/src/shared.ts | sed -n '108,113p'`

- **Reload planner: bindings/agents are non-restart:** Core reload rules mark `agents`, `agent`, `routing`, `messages`, `session`, and `bindings` as `kind: "none"`, so the surrounding multi-agent routing config in the reported workflow is also not a gateway-restart trigger.
  - file: [src/gateway/config-reload-plan.ts:91](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/src/gateway/config-reload-plan.ts#L91)

- **Hot reload handler restarts only affected channels:** When a reload plan includes channel changes, the handler stops and starts the affected channel runtime; it does not emit a gateway restart signal for that path.
  - file: [src/gateway/server-reload-handlers.ts:166](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/src/gateway/server-reload-handlers.ts#L166)

  - file: [src/gateway/server.reload.test.ts:430](https://github.com/openclaw/openclaw/blob/78a431ed224b6f5b9d58672346551a5b411ad200/src/gateway/server.reload.test.ts#L430)
