## Summary

Current `main` already contains a shipped Control UI chat fix that matches the reported post-send lag regression: same-session history reloads are deferred until the active run ends, and that behavior is covered by tests. The changelog shows it shipped in `2026.4.15`, after the reported `2026.4.12` regression.
