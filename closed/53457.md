## Summary

Current main already treats truly global system-prompt mutation as a plugin concern, not a core `agents.defaults.*` config surface. The documented and implemented path is a global/community plugin using `before_prompt_build` plus `prependSystemContext`/`appendSystemContext`, which is applied across agent runs and preserved for the Codex harness too.
