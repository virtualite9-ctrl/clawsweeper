## Summary

Current `main` contains explicit plugin-loader reentry protection, a regression test for recursive snapshot loads during plugin initialization, and enforced gateway RPC timeouts. The changelog shows the relevant fix shipped in `v2026.4.7`, which matches the 2026.4.5 regression family referenced from this issue.
