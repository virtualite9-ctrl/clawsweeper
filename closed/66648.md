## Summary

Current main, and the shipped `v2026.4.23` code in this checkout, already contain the relevant session-targeting and UI filtering guards for async exec/system-event leakage. The report targeted `v2026.4.12`; later shipped code pins exec-event delivery to the originating topic/session, prevents system-event turns from retargeting session bindings, and Control UI ignores transcript updates for other sessions.

## Evidence
