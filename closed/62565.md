## Summary

This PR is a Sophia-specific deployment/persona rollout, not a reusable OpenClaw core change. Current main already ships the generic Render deployment path, while the proposed Sophia workspace/config assets fit better as an external bundle-style plugin or separate deploy repo; the one generic hook change here is not on main and should come back separately if maintainers want it.
