## Summary

PR #42339 adds a vendor-specific Resemble deepfake-detection command and config surface to OpenClaw core, but current main explicitly routes optional detection/vendor behavior to plugins, and current plugin APIs already support custom slash commands. This is better handled as a ClawHub/plugin implementation than a core built-in.
