## Summary

Current `main` no longer matches the reported macOS Talk symptom: it reloads resolved Talk config from gateway `talk.config`, carries the configured ElevenLabs `voiceId` into playback, retries failed local ElevenLabs playback through gateway `talk.speak`, and has regression coverage around the config/voice path. The exact fix appears to be on `main` but not yet tied to a shipped release tag.

## Evidence
