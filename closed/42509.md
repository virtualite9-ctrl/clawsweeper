## Summary

Close PR #42509. Current main already includes the canvas-host DOM hardening, and the remaining viewer change does not correspond to a reproducible XSS issue on current main because `iconMarkup` is only sourced from local hardcoded SVG helpers. The PR's proposed `template.innerHTML` change also does not materially improve security for untrusted input, matching the existing review feedback.
