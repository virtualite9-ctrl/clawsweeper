## Summary

Current main, and the v2026.4.24 tag, already suppress task terminal-summary notifications for cron task records with `notifyPolicy: "silent"`/`deliveryStatus: "not_applicable"`. Current isolated cron delivery also skips runner fallback when the message tool already sent to the same or bound target, which addresses the duplicate Telegram-style summary path described against 2026.4.23.
