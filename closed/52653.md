## Summary

Close as `clawhub`. The request is coherent and useful, but current main scope pushes reusable procedural workflows into skills/plugins, not a new core artifact. Existing main surfaces already cover the proposed area through workspace `SKILL.md` procedures, the Skill Workshop plugin, and the OpenProse multi-agent orchestration plugin.
