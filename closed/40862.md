## Summary

Current `main` already supports the requested acknowledge-then-process pattern for `sessions_send`: `timeoutSeconds: 0` returns immediately with `status: "accepted"`, and the follow-up A2A flow later waits for the target reply and delivers an announce message back to the visible channel/thread. The proposed `ackMode` parameter does not exist, but the underlying behavior request is implemented.

## Evidence
