## Summary

PR #60966 is a useful LiteRT-LM/Edge Gallery provider prototype, but it is an optional local model-provider integration rather than a core OpenClaw change. Current main already has the native plugin/provider surfaces needed for this to live as an external provider plugin published through ClawHub or npm, and the PR discussion itself frames the work as an experimental, not-yet-merge-ready skeleton with unresolved runtime and packaging issues.

## Best Possible Solution

Close this core PR and encourage extracting the LiteRT-LM research/skeleton into a standalone native provider plugin published through ClawHub or npm. Reopen core work only if the external plugin path reveals a concrete missing Plugin SDK seam, or if maintainers explicitly decide LiteRT-LM should become an OpenClaw-owned bundled provider.
