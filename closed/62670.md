## Summary

Current `main` already contains a targeted exec-update lifecycle fix for the reported `Agent listener invoked outside active run` crash. The code now suppresses late `tool_execution_update` callbacks once an exec run exits, fails, or is aborted, and the changelog records that exact crash as fixed in `#62821`. The fix appears to be on `main` after `v2026.4.23`, so it is implemented but likely not yet in the latest shipped release.

## Evidence

- **Exec runtime now suppresses late updates after run end:** `emitUpdate()` bails out when `updatesDisabled` is set, with inline comments explicitly naming the `"Agent listener invoked outside active run"` crash and explaining that updates must stop once the agent run is disposed.
