## Summary

Current main implements the Bonjour/CIAO crash-loop fix reported against v2026.4.24. The bundled Bonjour plugin now wires scoped process handlers into the advertiser, classifies CIAO probing/announcement cancellations as handled dependency cancellations, lets the global unhandled-rejection path consult plugin handlers before exiting, and bounds stuck advertiser restarts so the gateway keeps running.

## Best Possible Solution

Close this as implemented on main and keep the current Bonjour-scoped CIAO process-error handler, shared host handler registry, bounded restart/disable behavior, container auto-disable policy, docs, and regression tests. If a user still sees a Bonjour crash after installing a release that contains this mainline fix, track that as a fresh issue with new gateway logs, environment details, and Bonjour/plugin configuration.

## Evidence
