## Summary

Current main implements the requested Control UI usage visibility: the Overview dashboard has a cost card, the Model Auth card displays provider quota windows as percent left with reset hints, and the Usage tab fetches session usage/cost data with provider/model breakdowns. The implementation is present in release v2026.4.24, and the only current-main change after that release is unrelated Feishu work.
