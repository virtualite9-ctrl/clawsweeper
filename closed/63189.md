## Summary

Current `main` already contains a TUI-side fix for stale streaming state, and that fix shipped in `v2026.4.23`, which is newer than the reporter's `2026.4.5` version.

## Evidence

- **Shipped changelog entry:** The changelog explicitly says the TUI now has a client-side streaming watchdog that resets `streaming · Xm Ys` back to `idle` after 30s of silence when final chat events are lost or late, and this entry appears in the `v2026.4.23` release.
  - file: [CHANGELOG.md:648](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/CHANGELOG.md#L648)

  - sha: [a9797214338b](https://github.com/openclaw/openclaw/commit/a9797214338ba31c52c796adbb75afb16e0684a9)
- **Main-code implementation:** `createEventHandlers` defines `streamingWatchdogMs`, clears `activeChatRunId`, sets activity back to `idle`, and emits a system log telling the user to send a new message to resync if streaming goes silent.
  - file: [src/tui/tui-event-handlers.ts:76](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/tui/tui-event-handlers.ts#L76)

  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Delta-path arming:** On every `delta` event, the TUI now marks the run as `streaming` and arms the watchdog specifically to avoid the stale-status case when visible text does not change or the final event never arrives.
  - file: [src/tui/tui-event-handlers.ts:313](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/tui/tui-event-handlers.ts#L313)
  - command: `nl -ba src/tui/tui-event-handlers.ts | sed -n '306,320p'`
  - sha: [245451b6a9c5](https://github.com/openclaw/openclaw/commit/245451b6a9c529464e177afa65013adca0d63eb5)
- **Regression coverage:** Tests cover the exact stuck-indicator scenario: a run that only receives `delta` events resets to `idle` after the watchdog window, refreshes correctly on new deltas, and does not spuriously fire after normal finalization.
  - file: [src/tui/tui-event-handlers.test.ts:729](https://github.com/openclaw/openclaw/blob/245451b6a9c529464e177afa65013adca0d63eb5/src/tui/tui-event-handlers.test.ts#L729)
