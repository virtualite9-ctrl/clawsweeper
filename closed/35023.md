## Summary

Current `main` already contains the replay fix for this bug: the agent transcript sanitizer now preserves thinking blocks on the latest assistant turn, and the run loop applies that sanitizer on follow-up tool continuations. The new coverage includes github-copilot and Anthropic replay cases matching the reported turn-2 failure shape. The fix appears to have landed after `v2026.4.23`, so older release reports still make sense.

## Evidence
