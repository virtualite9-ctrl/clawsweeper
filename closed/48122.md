## Summary

Current main and release v2026.4.24 implement the requested Control UI break-glass behavior: `gateway.controlUi.dangerouslyDisableDeviceAuth=true` now bypasses Control UI device identity checks for operator sessions, with regression coverage for stale-device and device-less connects.

## Evidence
