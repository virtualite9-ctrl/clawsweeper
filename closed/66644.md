## Summary

Current `main` policy is that third-party plugin discovery and promotion belong in ClawHub or the plugin's own repository, not as standalone showcase docs in the OpenClaw core repo. This PR is docs-only discoverability for the EO plugin, so it is out of scope for core and can be closed as a ClawHub item.
