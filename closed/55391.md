## Summary

The request is for a domain-specific external memory/graph integration, and current main explicitly scopes optional memory/integration work to plugins rather than OpenClaw core. The repo already exposes memory-plugin seams that an Engramic/graph adapter could use, so this belongs as a ClawHub/community plugin, not a core issue.

## Evidence
