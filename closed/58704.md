## Summary

Current main blocks the WhatsApp self-chat echo path described in #58704 by tracking message IDs returned from gateway sends and dropping matching fromMe upserts before they can enter access control or be represented as inbound user content. Existing WhatsApp monitor tests cover both group and self-chat DM gateway-sent fromMe echoes.
