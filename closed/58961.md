## Summary

Current main no longer matches the reported Control UI behavior: subagent labels are persisted, completed subagent sessions resolve to structured done/failed/timeout/killed states rather than a generic terminated state, and subagent-backed task cancellation uses cancelled user-facing text. The relevant behavior is also present in the latest v2026.4.24 tag.
