## Summary

Current `main` already routes Discord modal-trigger button clicks through a direct `showModal()` path instead of the agent reply pipeline, and modal submissions are acknowledged before async dispatch. That matches the requested fix, so the reported timeout behavior is implemented away on current `main`.

## Evidence
