## Summary

Current `main` no longer uses the static provider env-var map or exposes a `bailian` provider id; auth env resolution is now manifest-backed for the supported `qwen` and `alibaba` provider surfaces, so this PR's March 2026 repro path is obsolete rather than mergeable.
