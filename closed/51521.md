## Summary

This is a useful optional prompt-shaping feature, but current main already exposes plugin prompt-build hooks that can add reasoning-language guidance without a new core OpenClaw setting. Core `/reasoning` remains a visibility control, and the project vision directs optional capabilities like this to ClawHub/plugins when no missing core API is identified.
