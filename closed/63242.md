## Summary

Current `main` has already reworked the CLI startup path that matched this report: routed read-only commands such as `gateway status` avoid plugin preload, plugin CLI registration is lazy by default, descriptor collection uses a dedicated non-activating metadata path, bundled full plugin entries are skipped during metadata capture, and the shipped `v2026.4.23` changelog explicitly calls out large bundled-plugin startup reductions from native dist loading.
