## Summary

`postCompactionPrompt` is not implemented on `main`, but the requested resume-after-compaction behavior already fits the existing plugin/ClawHub extension surface: OpenClaw exposes `after_compaction` hooks with session context, and plugins can launch silent embedded agent turns from that hook.
