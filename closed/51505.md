## Summary

Current `main` already emits live `stream: "thinking"` agent events from reasoning streams and the gateway forwards those non-tool agent events to WebSocket clients. The reported missing live thinking-agent stream matched older behavior, but that path is now implemented on `main`.

## Evidence
