## Summary

Current main no longer matches the reported regression: the live replay/session-sanitization path preserves the latest assistant thinking turn, Claude 4.5+/4.6 replay policy does not request thinking-block dropping, and regression tests cover latest-turn preservation across Anthropic-, Bedrock-, and Copilot-style replays. The same relevant code is already present in release v2026.4.23.

## Evidence

- **Live replay path only drops thinking when policy asks for it:** `sanitizeSessionHistory()` is the current replay/session-recovery sanitation path. It applies `dropThinkingBlocks()` only when `policy.dropThinkingBlocks` is true, so replay no longer unconditionally removes the latest assistant turn.
  - file: [src/agents/pi-embedded-runner/replay-history.ts:443](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/src/agents/pi-embedded-runner/replay-history.ts#L443)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Claude 4.5+/4.6 models preserve thinking blocks:** `shouldPreserveThinkingBlocks()` returns true for Claude Opus/Sonnet/Haiku 4.x+, and Anthropic replay policy only enables `dropThinkingBlocks` when that preservation check is false.
  - file: [src/plugins/provider-replay-helpers.ts:72](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/src/plugins/provider-replay-helpers.ts#L72)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Transcript policy uses the preserve-thinking decision:** Anthropic transcript policy sets `dropThinkingBlocks` to the inverse of `shouldPreserveThinkingBlocks(modelId)`, so current Claude 4.x replay does not request dropping thinking blocks during recovery.
  - file: [src/agents/transcript-policy.ts:114](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/src/agents/transcript-policy.ts#L114)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Latest assistant turn is explicitly preserved:** `dropThinkingBlocks()` is documented and implemented to keep thinking blocks in the latest assistant turn verbatim, and to preserve assistant turn structure even when earlier turns were thinking-only.
  - file: [src/agents/pi-embedded-runner/thinking.ts:58](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/src/agents/pi-embedded-runner/thinking.ts#L58)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Regression tests cover latest-turn preservation:** Session-history tests assert preservation of latest assistant thinking blocks for GitHub Copilot, Anthropic replay, and Amazon Bedrock replay, including a latest assistant turn whose content is only signed thinking.
  - file: [src/agents/pi-embedded-runner.sanitize-session-history.test.ts:897](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/src/agents/pi-embedded-runner.sanitize-session-history.test.ts#L897)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)

  - file: [src/plugins/provider-replay-helpers.test.ts:61](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/src/plugins/provider-replay-helpers.test.ts#L61)
