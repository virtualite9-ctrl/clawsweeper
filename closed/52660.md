## Summary

The request is coherent, but current main scopes this kind of spawn-time knowledge/context injection to plugin or skill customization rather than OpenClaw core. Core already exposes subagent and context-engine seams that can carry injected context, while the project vision explicitly keeps optional orchestration features out of core and pushes new skills toward ClawHub.
