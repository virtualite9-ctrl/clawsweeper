## Summary

Current `main` implements the reported tool-policy merge and session-scoped message-tool discovery: `tools.profile` is applied before `tools.allow`, `group:messaging` expands to `message`, Discord discovery adds `event-create` when `actions.events` is enabled, and the effective-session inventory path now forwards live Discord session context into tool resolution.
