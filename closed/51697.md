## Summary

Keep #51697 open. Current main still does not contain `extensions/synology-chat/README.md`, so the PR is not implemented on main. The repository does have a full public Synology Chat channel doc that covers install/setup/config, but that does not fully replace the contributor-facing extension README gap named by the PR. The submitted README also needs revision before merge because its setup command is inconsistent with the current CLI and the file does not match established channel README conventions noted in the PR review discussion.

## Best Possible Solution

Keep the PR open for revision or maintainer follow-up. The best path is a small `extensions/synology-chat/README.md` that uses `# @openclaw/synology-chat`, explains that current packaged builds bundle the plugin, uses valid `openclaw plugins install` and `openclaw channels add --channel synology-chat ...` commands, includes a minimal JSON5 config block, and links to `https://docs.openclaw.ai/channels/synology-chat` for full setup and troubleshooting.
