## Summary

This PR adds Feishu/Lark-specific permission-management actions to the bundled Feishu plugin, but the request is optional service-specific tool surface that can be implemented as a ClawHub/plugin package with the current plugin API. Current main has not shipped these actions, and no missing OpenClaw extension API was identified.

## Best Possible Solution

Close this repository PR and move the expanded Feishu permission-management actions to a ClawHub/npm companion plugin or maintained Feishu plugin fork. If maintainers later want this in the bundled Feishu plugin, reopen with explicit maintainer ownership and address the schema/action-contract review notes first.
