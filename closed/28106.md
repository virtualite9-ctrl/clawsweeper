## Summary

Issue #28106 proposes a cross-instance agent marketplace/protocol for discovery, negotiation, escrow, reputation, and delegated work. Current `main` and the project vision place this kind of optional orchestration on the plugin/bundle/MCP/ClawHub side, not in OpenClaw core; shipped plugin runtime surfaces already expose subagents, paired nodes, and task-flow primitives that external implementations can build on.
