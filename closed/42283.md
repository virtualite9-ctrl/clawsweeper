## Summary

Current `main` explicitly moved community plugin discovery and promotion to ClawHub, and the community plugins docs now say not to open docs-only PRs just to add a plugin listing for discoverability. This PR is a docs-only listing request for an already-published external plugin, so it is now out of scope for OpenClaw core docs cleanup and is best closed in favor of the ClawHub/plugin path.
