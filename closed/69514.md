## Summary

Keep open. Current `main` still installs npm-sourced plugin dependencies with plain `npm install --omit=dev`, and both code and tests preserve that behavior. The PR is addressing a real gap, but the supplied review thread also surfaces an unresolved safety concern in the latest draft, so conservative cleanup should not close it.

## Evidence
