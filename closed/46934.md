## Summary

Current main implements the reported proxy path: Gemini memory embedding calls use the shared memory-host remote HTTP helper, and that helper now switches guarded fetch into trusted env-proxy mode when HTTP_PROXY/HTTPS_PROXY would be used for the target while preserving strict mode for NO_PROXY/ALL_PROXY-only cases. The unreleased 2026.4.25 changelog records this as #71506; no shipped release tag is available yet.
