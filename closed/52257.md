## Summary

Current `main` already implements the requested Control UI approval-dialog viewport fix: the shared approval overlay is full-viewport and fixed-position, centers its card with flexbox, and bounds the card height to the viewport with mobile safe-area handling. The same implementation is present in the latest shipped release snapshot checked here.
