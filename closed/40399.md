## Summary

Issue #40399 proposes an opt-in quality-based rerun layer, but current main keeps core model routing narrow and explicitly pushes optional orchestration into plugin/ClawHub surfaces. The existing plugin runtime, hook, and CLI seams are sufficient to host this as a plugin/harness instead of expanding the core `agents.defaults.model` contract.
