## Summary

Keep open. Current main does not implement Venice video generation: the Venice plugin lacks a video-generation provider file, does not call registerVideoGenerationProvider, and its manifest does not declare a videoGenerationProviders contract. The PR is still potentially useful, but it needs normal review/fixes rather than cleanup closure.
