## Summary

Current `main` already routes `openai/gpt-5.4*` models through the OpenAI Responses transport and uses the non-legacy token-limit fields (`max_output_tokens` for Responses, `max_completion_tokens` for OpenAI-compatible chat completions). The latest available release snapshot in this checkout already contains the same implementation.

## Evidence
