## Summary

Current `main` already supports plugin-driven model selection for agent runs. The legacy `before_agent_start` hook result includes `modelOverride`/`providerOverride`, and new work is directed to the explicit `before_model_resolve` hook.

## Evidence

- **Legacy hook result includes model override fields:** `PluginHookBeforeAgentStartResult` is defined as prompt-mutation fields plus `PluginHookBeforeModelResolveResult`, and that result type exposes `modelOverride` and `providerOverride`.
