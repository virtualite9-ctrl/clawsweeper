## Summary

Current `main` already ships a newer Microsoft Teams auth implementation that supersedes this PR's schema/type/env-validation follow-up: the runtime now supports `secret` and `federated` auth, validates env-backed certificate and managed-identity inputs, documents the shipped flow, and covers it with tests. This PR's old review target is obsolete against the current architecture.
