## Summary

Keep open. Current `main` still ships a bundled `coding-agent` skill that explicitly recommends `codex --yolo` and permission-bypass execution patterns, while the exec docs still state that sandboxing is off by default and gateway/node host exec defaults to no-approval `security=full` + `ask=off`. That is strong evidence against `implemented_on_main`, `cannot_reproduce`, `clawhub`, `incoherent`, or `stale_insufficient_info` closure.
