## Summary

Current main still rejects `gateway.bind=lan` with Tailscale Serve/Funnel and does not contain a shipped legacy migration for that combination, while #41009 explicitly closes #40910 and adds the migration/regression coverage for the same upgrade path.

## Best Possible Solution
