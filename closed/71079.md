## Summary

Issue #71079 is a valid Feishu card UX/config request, and the supporting comment reinforces that the header/footer hardcoding is real. Current main still lacks the requested config surface, but the same remaining Feishu card header/footer customization work is already tracked by open issue #60748, with open PR #54979 covering the header-only subset.

## Best Possible Solution

Close #71079 as a duplicate/superseded tracker and consolidate the remaining product discussion under open issue #60748. Maintainers can then review PR #54979 for the header-only subset and either extend it or add a follow-up for `channels.feishu.card.footer` behavior, schema, docs, generated metadata, and dispatcher tests.
