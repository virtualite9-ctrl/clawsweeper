## Summary

Keep open. Current `main` still documents system-wide Linux systemd services as a supported deployment shape, and `openclaw update` still promises a gateway restart, but the updater/service restart path remains centered on user-scoped systemd units and still emits `systemctl --user` guidance. There is not strong evidence that #62557 is fixed or obsolete.
