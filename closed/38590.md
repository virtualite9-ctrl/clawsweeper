## Summary

Current main addresses the reported plaintext API-key storage problem through the SecretRef system rather than a dedicated macOS-only Keychain source. Static provider API keys and auth-profile keys can now be stored as refs to env/file/exec providers, including an exec helper that can call macOS `security`, and `secrets apply/configure` can scrub plaintext from `auth-profiles.json`.

## Evidence
