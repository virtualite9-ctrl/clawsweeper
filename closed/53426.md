## Summary

Current `main` already addresses this post-self-update plugin refresh bug. The update CLI now respawns into the updated install before running plugin refresh, and regression tests cover that exact flow. The latest shipped tag available in this checkout, `v2026.4.22`, also contains the same post-core respawn path.

## Evidence
