## Summary

Current main already gets the requested DAVE package through the Discord plugin’s existing `@discordjs/voice` dependency, and the shipped `v2026.4.23` release contains that resolved dependency. This PR’s root `package.json` change is now redundant and targets the wrong manifest surface on main.
