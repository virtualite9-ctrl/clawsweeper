## Summary

Current main does not match the reported sessionManager-scoped pollution path. The safeguard runtime is still object-keyed, but production run and compaction paths open a run-local SessionManager before wiring the safeguard, compaction target resolution is centralized before model loading, and model-snapshot is only replay-sanitization bookkeeping rather than a model-selection source.
