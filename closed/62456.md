## Summary

Atlas Cloud is a valid provider-plugin idea, but current OpenClaw policy and docs point this kind of vendor-specific video provider to the external plugin path (ClawHub/npm), not a new bundled core extension.

## Evidence

- **Vision scope anchor:** OpenClaw's vision says optional capability should usually ship as plugins, plugin authors should host and maintain them in their own repository, and the bar for adding optional plugins to core is intentionally high.
  - file: [VISION.md:55](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/VISION.md#L55)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Contribution policy:** The contribution guide says most new features are not accepted in core and should be third-party plugins instead using the plugin SDK.
  - file: [CONTRIBUTING.md:88](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/CONTRIBUTING.md#L88)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Provider plugin docs:** The public plugin guide explicitly says you do not need to add a plugin to the OpenClaw repository and should publish it to ClawHub or npm instead.
  - file: [docs/plugins/building-plugins.md:16](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/docs/plugins/building-plugins.md#L16)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Community plugin path:** The community plugin docs define third-party providers as community-maintained packages published on ClawHub or npm and installed with `openclaw plugins install`.
  - file: [docs/plugins/community.md:9](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/docs/plugins/community.md#L9)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
- **Current shipped video providers:** Current user-facing docs say OpenClaw ships 14 video-generation backends and list them; Atlas Cloud is not among them, so this is not already implemented on `main`.
  - file: [docs/tools/video-generation.md:10](https://github.com/openclaw/openclaw/blob/ecac696643c604c6d854341bfe6424bfa31423e9/docs/tools/video-generation.md#L10)
  - sha: [ecac696643c6](https://github.com/openclaw/openclaw/commit/ecac696643c604c6d854341bfe6424bfa31423e9)
