## Summary

Current main already runs inbound media understanding on the direct reply path before session processing, emits the transcription hook when a transcript is produced, and also preserves media-understanding on the ACP path when inbound media arrives without prior processing. The same implementation is present in shipped release v2026.4.23, with regression coverage for the transcribed direct-delivery path.

## Evidence
