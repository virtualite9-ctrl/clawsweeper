## Summary

Current main already gives Control UI users visible feedback for queued follow-ups: busy sends are stored in chatQueue, rendered as a visible queued-message panel with the original text and controls, and documented/shipped in v2026.4.24. This addresses the reported user-visible disappearance even though the pending item is shown as queue UI rather than a permanent transcript bubble while pending.
