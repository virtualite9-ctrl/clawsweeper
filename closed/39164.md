## Summary

Current `main` already provides first-class exec/bash output compaction through the bundled opt-in `tokenjuice` plugin. That covers the issue's underlying goal of reducing noisy exec output in agent context, even though the shipped implementation uses post-exec tool-result middleware instead of a PATH-detected `rtk rewrite` preprocessor.
