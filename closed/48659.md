## Summary

Keep PR #48659 open. Current main has absorbed some related Microsoft Teams media hardening, but the central PR scope is not fully implemented: there is no channel archive plugin, no recent channel focus sidecar/readable label persistence, no channel_deleted plugin hook, no JSON escape repair middleware, and channel replies still default to top-level when requireMention=false. The PR also has substantive review feedback that should be addressed during a rebase/split rather than auto-closing it as obsolete.

## Best Possible Solution

Keep the PR open but ask for a focused rebase/split. The best maintainer path is to preserve the useful pieces not yet on main: JSON escape repair, thread-default reply policy alignment, recent-channel focus sidecar with readable Graph labels, a documented channel_deleted/plugin metadata seam if still needed, and the archive plugin if it meets the bundled-plugin bar. Each slice should address the existing bot review comments and include targeted tests/docs/changelog updates.
