## Summary

Current `main` already supersedes this PR. The `nodes run` action it patches is gone, `system.run` is now blocked from `nodes invoke`, and shell execution on paired nodes runs through `exec host=node`, which derives a longer gateway timeout and forwards the command timeout into `node.invoke`. The same superseding structure is present in the shipped `v2026.4.23` release, so this PR is obsolete rather than pending work.
