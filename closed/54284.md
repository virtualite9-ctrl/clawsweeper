## Summary

Current `main` already contains a targeted fix for the reported `openai-codex` OAuth regression: refreshes are now serialized with a cross-agent per-profile lock, and `refresh_token_reused` triggers recovery from fresher stored credentials or a single retry. The fix is present on `main` but not in the tagged release `v2026.4.22`.

## Evidence
