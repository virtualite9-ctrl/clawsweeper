## Summary

#71999 is a duplicate/superseded tracker for open PR #72009. Current main still has the hardcoded two-minute typing TTL and no public `typingTtlSeconds` config, so this is not implemented-on-main. The remaining work is already represented by #72009, which directly says `Closes #71999` and includes the runtime, config, docs, generated schema, and test changes for the requested behavior.

## Best Possible Solution

Close #71999 as superseded by the linked implementation PR #72009, while letting maintainers review, land, or consolidate #72009 with the older overlapping PR #42105. The shipped solution should include the runtime default change, `typingTtlSeconds` config contract, docs/help/generated schema updates, and focused TTL tests together.
