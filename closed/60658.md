## Summary

Keep open. Current `main` has partially superseded the draft with safer overflow handling, but it has not implemented this PR’s two concrete changes: the prompt-time tool-result guard is still installed, and there is still no dedicated `checks-fast-prompt-cache` / `test:prompt-cache` lane. The draft also has an unresolved regression risk called out in review, and current `main` now contains alternative preflight truncation logic that would require the PR to be redesigned rather than cleanup-closed.
