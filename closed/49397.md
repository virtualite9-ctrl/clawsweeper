## Summary

Close as `clawhub`. Current `main` does not include a bundled Serper integration, and the repo’s current architecture/docs explicitly route optional web-search providers to external plugins published on ClawHub or npm rather than adding more bundled/core support by default.

## Evidence

- **Vision: optional capability should be plugins, not core by default:** `VISION.md` says core stays lean, optional capability should usually ship as plugins, the preferred path is external plugin distribution, and the bar for adding optional plugins to core is intentionally high.

  - file: [docs/plugins/building-plugins.md:11](https://github.com/openclaw/openclaw/blob/f3c37a946cc49d26e5fe65a1043f7286a86aa4a4/docs/plugins/building-plugins.md#L11)
