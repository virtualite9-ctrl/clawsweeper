## Summary

Current main already covers the reported failure modes: Telegram/message delivery waits for gateway delivery results and protects in-flight sends from stall restarts; WhatsApp/channel restarts are bounded by backoff and max attempts; session tools expose discoverable aliases/listing; xAI tools stay disabled without configured or env auth.
