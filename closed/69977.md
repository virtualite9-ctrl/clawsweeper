## Summary

#69977 should close as duplicate/superseded by the open canonical macOS Gateway RSS regression tracker #70717. #69977 adds useful 2026.4.20 MacBook Air data showing the earlier session-maintenance and dreaming fixes were insufficient, but #70717 tracks the same remaining fresh-start/post-4.15 RSS regression with richer plugin-count/version data and the active #70730/current-main startup-scoping discussion.

## Best Possible Solution

Close #69977 as duplicate/superseded and keep #70717 as the canonical tracker for the post-4.15 macOS Gateway RSS/startup regression. Validate the current main startup-scoping/cold-registry fix there, and open a narrower follow-up only if current main still reproduces a distinct memory-core, memory-wiki, or channel-specific leak after startup plugin scope is bounded.
