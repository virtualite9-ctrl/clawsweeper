## Summary

Current `main` already fixes this: plugin runtime now exposes the shared `VERSION` resolver instead of doing its own source-tree-relative `package.json` lookup, and there is explicit regression coverage for `runtime.version` alignment. The latest shipped tag available in this checkout, `v2026.4.22`, also contains the fix.
