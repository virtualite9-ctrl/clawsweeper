## Summary

Current main carries Telegram forum topic IDs into plugin tool context as the trusted ambient delivery thread ID, so plugin tools no longer need the model to manually supply a `topicId` parameter to know the active topic. Main intentionally does not mutate plugin tool params; plugins should read `toolCtx.deliveryContext?.threadId`.
