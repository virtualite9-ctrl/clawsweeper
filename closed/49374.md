## Summary

Current main already routes webchat heartbeat visibility through `channels.defaults.heartbeat` and suppresses the reported heartbeat poll/ack text in the webchat UI when `showOk` is false. The same behavior is documented and present in the shipped `v2026.4.22` release.

## Evidence
