## Summary

Reviewed PR #42772 against current `main` (`459d2770`) on April 25, 2026. The reported Control UI/WebChat real-time refresh bug no longer reproduces on the current code path because session-key aliases are normalized to the gateway-provided canonical `mainSessionKey` before `handleChatEvent` applies its strict equality filter.

## Evidence
