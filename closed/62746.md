## Summary

Current `main` already contains a targeted exec-update suppression fix for this gateway crash, plus regression coverage and a changelog entry calling out the exact `Agent listener invoked outside active run` failure.

## Evidence

- **Runtime guard for late exec updates:** `createExecProcessHandle` now gates `emitUpdate()` behind `updatesDisabled`, `session.exited`, and `session.backgrounded`, with an inline comment naming the exact `Agent listener invoked outside active run` crash and explaining that updates must stop once the agent run is disposed.
  - file: [src/agents/bash-tools.exec-runtime.ts:563](https://github.com/openclaw/openclaw/blob/3ffd944e6ba6dfe4c7c4c275848f1c0a1f246e28/src/agents/bash-tools.exec-runtime.ts#L563)
