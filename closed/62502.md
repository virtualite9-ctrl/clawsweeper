## Summary

Current `main` already converts Windows absolute plugin import paths to `file://` URLs before dynamic loading, includes regression coverage for drive-letter and UNC paths, and the changelog records the `ERR_UNSUPPORTED_ESM_URL_SCHEME` onboarding/plugin-loader fix as shipped in `v2026.4.7`.
