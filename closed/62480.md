## Summary

Current main contains the plugin-registry reuse changes that stop compatible runtime/helper paths from reloading and re-registering workspace plugins, and the changelog shows those fixes shipped in v2026.4.23. That matches the reported symptom of an external memory plugin repeatedly registering after 2026.4.5.

## Evidence
