## Summary

Keep open. Current `main` does not implement the nested Claude `{"type":"result","result":"..."}` unwrapping described in PR #66819, the linked bug #66743 is still open, and there is no strong evidence that the PR was superseded or shipped elsewhere.

There is no nested-result unwrap helper or Claude-specific gating in this path.
