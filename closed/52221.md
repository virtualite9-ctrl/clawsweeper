## Summary

Current `main` already fixes the reported 0% context monitor behavior for OpenAI-compatible local backends. OpenClaw now requests streaming usage, recovers llama.cpp-style timing metadata, persists fresh prompt/context snapshots into session state, and lets `/status` fall back to transcript usage when session metadata is missing or stale.

## Evidence
