## Summary

Close as `clawhub`. Current `main` already exposes the hook surface needed for an agent-tracing plugin, but project direction and plugin docs say optional plugins should be maintained externally and distributed via ClawHub/npm rather than added as new bundled plugins in the core repo. The PR discussion also already surfaced unresolved correctness issues, which further argues against bundling this into `main`.
