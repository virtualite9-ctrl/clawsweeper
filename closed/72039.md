## Summary

PR #72039 adds a new bundled Fleet Orchestrator plugin and dispatch layer. Current main does not already contain it, but the requested capability is optional orchestration/plugin product work that fits the documented external plugin and ClawHub path, and the current plugin API already exposes the route/CLI/service/tool seams this stack uses or plans to use.

## Best Possible Solution

Close this PR as ClawHub/plugin-scope work. The best path is to continue Fleet Orchestrator as an external native plugin published through ClawHub or npm using the existing plugin APIs; if that work uncovers a truly missing core seam, open a small focused SDK/API proposal instead of landing the whole orchestration layer in the main repo.
