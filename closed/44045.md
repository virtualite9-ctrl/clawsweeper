## Summary

Current main already fixes the reported `/status` / `session_status` stale-zero context display by enabling transcript-usage fallback on the real status paths, preferring transcript prompt totals over sparse stored totals, and covering that behavior in tests; the changelog says this shipped in 2026.4.5.

## Evidence
