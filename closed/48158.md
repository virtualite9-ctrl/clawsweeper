## Summary

Current `main` already handles the long-message cases this PR targets: Discord and Telegram both expose chunkers to the shared outbound planner, and Discord forum sends already split long text into one starter thread message plus follow-up messages. The PR's March 2026 root-cause analysis no longer matches the shipped code on `main`.
