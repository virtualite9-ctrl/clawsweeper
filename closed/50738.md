## Summary

Current main implements the OpenRouter LLM request path strongly enough to close this issue: the bundled OpenRouter provider installs the stream wrapper, the wrapper resolves and forwards attribution headers at request time, and the embedded runner now preserves the runtime API key for provider-owned stream functions. The related #51056 discussion was useful because it identified the broader auth/header forwarding path; current main has regression coverage for that path too.
