## Summary

The report describes a registry/moderation problem in the skills marketplace, not an OpenClaw core bug. Current main routes skill discovery/install flows through ClawHub, OpenClaw’s security policy sends ClawHub vulnerabilities to the `openclaw/clawhub` repo, and the threat model already classifies malicious-skill publication and moderation evasion as ClawHub risks.
