## Summary

Keep open. Current main does not implement the PR’s requested runtime behavior: bundled plugin runtime dependency checks still only trust the plugin install root, and the existing tests still assert that package-root node_modules must not satisfy a plugin. The PR also contains additional useful follow-up work for eager-install version conflicts that is not present in current main.
