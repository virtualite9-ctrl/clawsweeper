## Summary

Close PR #62807 as already implemented/superseded on current `main`. The heartbeat portion is obsolete because OpenClaw no longer rolls heartbeat transcript entries back via file truncation; current main keeps transcripts append-only and filters heartbeat prompt/ack pairs at context-build and session-extraction time. The Kimi onboarding portion is also already present on main. The review comments were useful because they identified a real logic bug in this PR’s proposed re-resolve path, so this branch is not a merge candidate even aside from being superseded.
