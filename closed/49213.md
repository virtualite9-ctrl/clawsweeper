## Summary

Current `main` no longer contains the reported hardcoded Codex `--cwd` invocation in `extensions/acpx/src/runtime.ts`; that file now delegates session handling to the embedded `acpx/runtime` package, and the bundled plugin pin has moved from the reported `acpx` `0.1.16` to `0.5.3`.

## Evidence
