## Summary

Current main already exposes session-binding adapter registration through supported public plugin-sdk subpaths, and the root `openclaw/plugin-sdk` barrel is intentionally kept narrow. The underlying capability requested by PR #48381 is therefore already implemented on main, just on the documented subpaths rather than the root barrel.
