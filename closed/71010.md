## Summary

The issue asks for official discovery/promotion of the external WeiXin/WeChat channel plugin. Current main documents `@tencent-weixin/openclaw-weixin` as an external plugin, provides ClawHub/npm install and channel-plugin metadata surfaces, and VISION.md puts plugin discovery, official publisher status, provenance, and promotion in ClawHub rather than the core repo. The related PR #71129 attempted a core catalog entry but was closed unmerged, so there is no shipped core implementation to preserve or close as fixed.

## Best Possible Solution

Keep WeiXin/WeChat as an external channel plugin and handle official discovery, verification, publisher status, and promotion through ClawHub. If maintainers later want a core catalog exception, it should be a narrow maintainer-approved PR that only updates the official catalog metadata and tests.
