## Summary

Current main implements the requested Bedrock bearer-token auth. The native `amazon-bedrock` Converse transport uses `@mariozechner/pi-ai@0.70.2`, whose Bedrock stream reads `AWS_BEARER_TOKEN_BEDROCK`, sets the AWS SDK bearer token, and forces `httpBearerAuth`; OpenClaw also surfaces that env marker before AWS access keys/profile. The Mantle provider work remains useful but is no longer the only coverage for Claude/Converse.
