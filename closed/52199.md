## Summary

Current `main` already covers the reported cleanup: CLI and agent memory search now share the local memory-manager search path, gateway/secret-resolution failures are surfaced explicitly, upstream embedding HTTP failures include status/body, empty Telegram group allowlists are handled as warning/drop behavior instead of startup-fatal behavior, gateway singleton/port-conflict handling is implemented, and custom proxy embedding config is documented.
