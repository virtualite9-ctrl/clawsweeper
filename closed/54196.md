## Summary

Current `main` already contains working Control UI attachment plumbing for both drag-and-drop and file-picker uploads, and the same implementation is present in the shipped `v2026.4.22` release. The issue's claimed `this`-binding and variable-shadowing root causes do not match the current source.

## Evidence
