## Summary

Current `main` now has first-class `before_prompt_build` and `agent_end` plugin hooks wired through the agent loop, and bundled plugins on `main` already implement the requested read/write memory loop (`memory-lancedb` for recall/capture and `skill-workshop` for procedural self-improvement). The landing commit is `51dd4f288f67ef4df4c7a121f078f508535df1fd`, which post-dates the issue and the latest tag `v2026.4.23`, so this is implemented on `main` but not yet in a tagged release.

## Evidence

- **Core hook runner exposes both requested phases:** `runBeforePromptBuild(...)` merges prompt-build hook results, and `runAgentEnd(...)` delivers completed-run events to plugins.
