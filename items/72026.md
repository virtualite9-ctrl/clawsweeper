## Summary

Keep open. Current main still has the #72019 duplicate subagent task path: the task is embedded in the child system prompt and also sent as `[Subagent Task]: ${task}` in the first child user message. PR #72026 is an active focused fix attempt with tests; overlapping PR #72053 may be the stronger canonical fix, but it is still open and unmerged, so this should remain a maintainer choice rather than an automatic cleanup close.

## Best Possible Solution

Keep #72026 open for maintainer review while consolidating the overlapping #72019 fixes. The best final implementation should keep the full delegated task in exactly one model-visible payload location, preserve multiline/structured task text, cover the final `message` plus `extraSystemPrompt` gateway payload with regression tests, and either remove or explicitly deprecate any unused `task` parameter. If #72053 lands first, close #72026 as superseded by that shipped fix.
