## Summary

Keep open. Current `main` does not implement this hybrid-search change: the memory-core plugin still does a single full-query FTS lookup in hybrid mode, while `buildFtsQuery` still joins multi-word tokens with `AND`. That matches the PR's reported failure mode for terms split across chunks. The PR is stale against the moved code layout, but the underlying bug/request is still plausible and not clearly fixed on `main`.
