## Summary

Issue #37701 is already fixed on current main and was shipped after the reported 2026.3.2 build. The changelog entry under 2026.3.8 explicitly calls out the macOS overlay exclusivity crash fix, current overlay presentation code no longer uses shared visibility mutation in `OverlayPanelFactory.present`, and there is now a repeated Talk overlay smoke test covering the launch/present path.
