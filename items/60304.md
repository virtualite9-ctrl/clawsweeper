## Summary

Keep open. Current `main` still lacks the Xiaomi-specific normalization this PR proposes: the Xiaomi plugin has no stream wrapper, the shared OpenAI-completions parser still turns `reasoning_content` into `thinking` blocks, and the only automatic reasoning-only retry path is gated to GPT-5 OpenAI lanes, not Xiaomi. The PR discussion also points to only minor review comments, so there is not strong evidence to close it as stale or superseded.

## Evidence
