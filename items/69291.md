## Summary

Keep #69291 open. Current main has overlapping prompt guidance, but it does not fully implement the requested global, systematic Agent Behavior Principles component across all users. The related #71937 RFC is closed, design/research-only, and treated #69291 as a related issue to consider rather than as a shipped replacement.

## Best Possible Solution

Keep the issue open for maintainer prompt/product judgment. A good implementation would reconcile the proposal with `buildExecutionBiasSection`, `GPT5_BEHAVIOR_CONTRACT`, provider prompt contributions, plugin prompt hooks, and `docs/concepts/system-prompt.md`, then add focused system-prompt tests and docs rather than layering a large duplicate component onto the current prompt.

## Evidence

- **Global prompt partially overlaps the request:** `buildExecutionBiasSection` already tells agents to ask for the one missing blocker, continue until done or blocked, retry weak lookups, live-check mutable state, and provide final evidence. This overlaps with the proposal but is not the requested four-principle component and does not cover all proposed shortest-path/surgical/read-back rules.
