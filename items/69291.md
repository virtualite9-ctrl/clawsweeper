## Summary

Keep #69291 open. Current main has overlapping prompt guidance, but it does not fully implement the requested global, systematic Agent Behavior Principles component across all users. The related #71937 RFC is closed and treated #69291 as a related existing issue, not as a canonical shipped replacement.

## Best Possible Solution

Keep the issue open for maintainer prompt/product judgment. A good implementation would reconcile the request with `buildExecutionBiasSection`, `GPT5_BEHAVIOR_CONTRACT`, provider prompt contributions, and `docs/concepts/system-prompt.md`, then add focused system-prompt tests and docs rather than layering a large duplicate component on top of existing guidance.

## Risks / Open Questions

- Implementing the proposal verbatim could duplicate or conflict with the existing compact Execution Bias, Tool Call Style, Safety, and GPT-5 overlay guidance.
- Some proposed bullets, such as always stating a plan before multi-step work or reporting after each step, may conflict with current prompt guidance to avoid routine narration and keep progress updates brief.
