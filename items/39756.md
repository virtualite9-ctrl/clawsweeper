Closing from a Codex review: current `main` already fixes this. The relevant code on main is from commit `7d471837369885c12d540d6bafa319ff41129b89`. I could not confirm a shipped release containing it from the checked-out tags/context, so I’m closing this as implemented on `main` rather than as a released fix.
