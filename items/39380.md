## Summary

Current `main` already handles Control UI config unions well enough to avoid the Agent List `Unsupported schema node` failure. The generated agent schema still exposes `agents.list[].runtime` as an `anyOf` union, but the Control UI analyzer now treats renderable unions as supported and the renderer falls back to a JSON textarea instead of flagging the field as unsupported. The same implementation and regression tests are present in the shipped `v2026.4.22` release.
