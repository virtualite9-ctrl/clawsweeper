## Summary

Keep open. The exec-approvals half of the report is obsolete on current `main`, but the transcript half is still present: current `main` still reads whole transcript files into memory for idempotency checks and `sessions.compact maxLines`, and the latest release `v2026.4.22` still contains those same full-file reads.

## Evidence
