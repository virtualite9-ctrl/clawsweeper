## Summary

Keep open. Current `main` contains the previously shipped #52270 timer re-arm fix and older wake/restart guards, but there is no later heartbeat-specific fix on `main` or in `v2026.4.23` that clearly resolves this Linux long-runtime stall, and the issue discussion includes a second report on `2026.4.9` saying the problem persisted.
