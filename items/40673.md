## Summary

Keep open. Current `main` still lacks the requested documentation for disable semantics, and I did not find code, tests, or history in this checkout showing a LaunchAgent-specific fix for post-restart raw-stream logging. The code does treat `OPENCLAW_RAW_STREAM=0` or unset as disabled, so the report still points to a plausible restart/environment propagation bug rather than an item already implemented on `main`.

## Evidence
