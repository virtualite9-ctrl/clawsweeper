## Summary

Keep open. Current `main` still has the silent-empty-output gap this PR is trying to fix: cron retries interim acknowledgements, but not empty sanitized finals, and an empty synthesized reply still falls through as a non-error/no-delivery outcome. The PR discussion also surfaced a credible unresolved CLI session-reuse regression that still needs maintainer judgment before merge.
