## Summary

Keep open. Current `main` still omits `memory_search` / `memory_get` from the built-in tools table, and the repo treats those tool ids as first-class OpenClaw tools. There is some wording nuance because they come from the active memory plugin and can be disabled, but that is not strong evidence to close the PR as invalid or already implemented.
