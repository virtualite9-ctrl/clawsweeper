## Summary

PR #39465 targets an old `src/channels/dock.ts` Discord threading path that is no longer present on `main`. Current `main` already resolves Discord direct-message targets through the plugin-based routing/threading stack, including fallback to DM user identity and `OriginatingTo`, and that implementation is already present in the released `v2026.4.22` tree.
