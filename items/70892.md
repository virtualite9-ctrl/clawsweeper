## Summary

Keep open. Current `main` already covers part of the request through existing skill filtering (`skills.allowBundled` and agent skill allowlists), but it does not expose config gates for the hardcoded CLI quick-reference, reply-tag/output-directive, or messaging prompt blocks. Only the heartbeat section currently has a user-facing `includeSystemPromptSection` toggle.

## Evidence

- **CLI/reference and messaging blocks are still hardcoded in the full prompt:** `buildAgentSystemPrompt()` still appends `## OpenClaw CLI Quick Reference`, then `buildAssistantOutputDirectivesSection(...)`, then `buildMessagingSection(...)` in the normal/full prompt path. Those sections are only omitted by internal `promptMode=minimal`, not by any user-facing config toggle.
