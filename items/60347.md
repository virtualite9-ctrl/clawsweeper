## Summary

Keep open. Current `main` does not implement `agents.defaults.memorySearch.local.gpu` in config, runtime resolution, node-llama typing, or docs, so this is not `implemented_on_main`. The PR discussion is concrete and useful, and no other close reason fits. The branch also looks stale against current `main` because local embeddings now have additional `contextSize` plumbing and a dedicated `node-llama` wrapper/tests/docs surfaces that the PR would need to update.
