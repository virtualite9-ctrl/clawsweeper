## Summary

Keep open. Current `main` has refactored web search into `src/web-search/runtime.ts` plus secrets-runtime resolution, but it does not already implement the PR's proposed execute-time/create-time diagnostic logging. The PR also targets an older code path and would need rework against the current runtime/secrets architecture.
