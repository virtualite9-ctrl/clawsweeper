## Summary

Current `main` does not implement PR #60875's typed-failure downgrade, and the narrower bug path described in the PR still exists: workspace discovery is scoped to `.openclaw/extensions`, but any directory there with an `index.*` entry is still admitted as a plugin candidate, then a missing manifest is still promoted to an error diagnostic that flows into config validation and gateway startup logging.
