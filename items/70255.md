## Summary

Keep open: current `main` still routes `openclaw hooks` `list`/`info`/`check`/`enable`/`disable` and bare `hooks` through the generic `runHooksCliAction` path, with no one-shot post-output exit helper, while those commands still build plugin diagnostics and can execute plugin `register()` code.
