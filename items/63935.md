## Summary

Current `main` already routes embedded Gemini runs through the provider-owned Google transport path instead of forcing them through the generic OpenAI-style path, and the OpenAI Responses context wrapper is a no-op for `google-generative-ai`. The same relevant stream-routing files are present in release `v2026.4.23`, with regression tests covering the provider-owned Google path.
