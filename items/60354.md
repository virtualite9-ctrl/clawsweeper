## Summary

Current `main` no longer matches the reported regression: Linux docs explicitly support system-level gateway units, `openclaw gateway` is the foreground runner for those units, doctor gates custom service overwrites behind `--force`, and gateway service auth/runtime now avoids requiring inline `Environment=` secrets by using config-first gateway auth plus `EnvironmentFile` support.
