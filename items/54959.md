## Summary

Keep open. Current `main` has new OpenShell backend documentation, but it does not implement this PR's requested in-sandbox deployment guide or its Platforms-tab docs surface. The strongest evidence is that `main` documents OpenShell as a managed sandbox backend with the Gateway still running on the host, while this PR is specifically about running OpenClaw inside an OpenShell sandbox behind proxy/policy constraints.
