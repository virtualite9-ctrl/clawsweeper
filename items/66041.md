## Summary

Keep this PR open. Current main still has the Telegram ack-reaction mismatch: unmentioned group messages can process when `requireMention` is false, but the default `group-mentions` ack gate still rejects them.

## Best Possible Solution

Review and land this PR or an equivalent Telegram-owned fix that treats the default `group-mentions` ack scope as `group-all` only for Telegram groups already configured with `requireMention: false`, while keeping the focused regression test and Telegram docs note.
