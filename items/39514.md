## Summary

Current `main` already treats Discord `@everyone`/`@here` as mention-gating hits and ignores bot-authored broadcasts for loop prevention, but it does not implement the PR's proposed opt-in `everyoneMentionsBot` config at account/guild/channel scope. Because the behavior on `main` is global/default-on rather than configurable, this PR is not clearly implemented-on-main and still contains distinct potentially useful work.
