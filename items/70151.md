## Summary

Keep open. Current `main` still appears to have the stale-snapshot empty-turn path described in PR #70151, and the same behavior is still present in the latest release `v2026.4.23`. I did not find an equivalent landed fix or regression coverage for the late-assistant/session-file recovery case.
