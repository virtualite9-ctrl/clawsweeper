## Summary

Keep PR #70151 open. Current main still builds terminal payloads from the attempt snapshot and immediately returns the incomplete-turn error when payloads are empty; it does not poll the persisted session for a late assistant reply. The PR is focused, draft, and still contains useful unmerged behavior plus regression coverage.

## Best Possible Solution

Keep the PR open and adapt the narrow fix to current main: before returning the terminal empty-turn error, refresh from the persisted session or otherwise have the attempt path return a guaranteed-current assistant message, build payloads from that refreshed assistant, and keep a regression test for the stale-snapshot/late-assistant case.
