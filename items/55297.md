## Summary

Keep open. Current `main` implements a core/package auto-updater and post-update plugin sync, but the request asked for auto-update of both plugins and skills. ClawHub skills still require an explicit `openclaw skills update` action, and there is no `skills.autoUpdate` or `plugins.autoUpdate` config surface matching the issue request.
