## Summary

Keep open. Current `main` still resolves bare `session_status` model overrides with `defaultProvider: currentProvider`, so a cross-provider bare model like `deepseek-chat` would still be interpreted against the active provider. The unique-provider inference helper exists elsewhere on `main`, but it is not wired into `src/agents/tools/session-status-tool.ts`, and the PR's regression coverage is not present.
