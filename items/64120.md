## Summary

Keep open. Current `main` still lacks the WhatsApp DM voice-note audio-preflight path described in this PR, while Telegram and Discord retain analogous preflight transcription flows. The PR discussion also shows the branch absorbed multiple real review fixes, so this is not already-implemented, obsolete, or incoherent cleanup candidate.

## Evidence

There is no WhatsApp audio preflight or transcript substitution in this path on current `main`. That means `<media:audio>` still flows through as the inbound body for voice notes.
