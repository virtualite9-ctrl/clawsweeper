## Summary

Keep open. Current `main` still shows subagent sessions in the chat session picker, and the PR patch described in the discussion has unresolved correctness problems rather than being already implemented or obsolete.

## Evidence

That is the opposite of what this PR proposes, so `main` has not already moved to the requested behavior.
