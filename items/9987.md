## Summary

Keep #9987 open. Current main has backend support for listing pending exec approvals via `exec.approval.list`, including command payloads, but still lacks the requested user-facing `openclaw approvals pending` command, CLI JSON/table rendering, CLI tests, and CLI docs. The two related implementation PRs, #10013 and #10341, were closed unmerged, so this issue remains the canonical tracker.

## Best Possible Solution

Keep #9987 open until OpenClaw adds `openclaw approvals pending` in `src/cli/exec-approvals-cli.ts`, backed by the existing `exec.approval.list` gateway method, with human table output showing command details, `--json` output for automation, focused CLI tests, and docs at `docs/cli/approvals.md` / `docs/cli/index.md`. Treat `--watch` as either part of that command or an explicit follow-up with maintainer product judgment.
