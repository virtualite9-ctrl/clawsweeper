## Summary

Keep #9987 open. Current main exposes backend `exec.approval.list` plumbing with tests proving command payloads are present, but the requested user-facing `openclaw approvals pending` CLI command, CLI tests, and CLI docs are still missing. The related implementation PRs #10013 and #10341 are closed unmerged, and the existing issue discussion already identifies this as the canonical tracker.

## Best Possible Solution

Keep #9987 open until OpenClaw adds `openclaw approvals pending` in `src/cli/exec-approvals-cli.ts`, backed by existing `exec.approval.list`, with table output showing command details, `--json` output for automation, focused CLI tests, and docs at `docs/cli/approvals.md` / CLI index. Treat `--watch` as either part of the first implementation or an explicit follow-up.
