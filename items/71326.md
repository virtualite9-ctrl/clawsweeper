## Summary

Keep open. This is a recent, detailed report against core gateway-host exec behavior. Current main still skips script preflight for gateway YOLO mode and runs non-sandbox exec through a fresh shell child process; I found no source, docs, tests, changelog entry, related issue, or PR evidence that adds a stale-read/cache-coherence mitigation or supersedes the report.

## Best Possible Solution

Keep this issue open for maintainer investigation. The useful next step is a focused macOS/APFS repro across v2026.4.19, v2026.4.20, v2026.4.24, and current main using gateway-host exec with security=full and ask=off. If confirmed, the fix likely belongs in the core host-exec/runtime path or in a documented gateway-host filesystem consistency diagnostic, with regression coverage around atomic rename followed by reads from separate exec calls.
