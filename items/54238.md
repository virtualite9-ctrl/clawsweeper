## Summary

Keep open. Current `main` still does not thread `remoteCdpTimeoutMs` / `remoteCdpHandshakeTimeoutMs` into `createTargetViaCdp`, so this PR is not already implemented, and there is not strong enough evidence to call it obsolete or non-reproducible.
