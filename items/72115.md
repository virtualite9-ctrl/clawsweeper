## Summary

Keep open. Current main does not contain the Skill Workshop hardening this PR proposes: path containment is still string-prefix based, apply still prepares once before writing, and the plugin SDK skills runtime does not expose prompt-budget preview/enforcement helpers. The PR is a live implementation candidate rather than obsolete, though it needs maintainer review because the diff is large and includes some unrelated-looking CI/test/support changes.

## Best Possible Solution

Keep this PR open as the current implementation candidate. Ask the author to rebase on current `main`, narrow or justify the unrelated-looking changes, and run the plugin SDK/API plus targeted Skill Workshop tests; then review the filesystem-write and SDK-boundary changes for merge rather than closing the work as obsolete.
