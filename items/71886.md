## Summary

Keep open. Current main still serves the Control UI CSP with `connect-src 'self' ws: wss:` and no `https://tweakcn.com`, while the bundled custom theme importer fetches `https://tweakcn.com/r/themes/{id}` directly from the browser. The PR is a narrow fix for an active main-branch behavior gap, not an implemented, duplicate, or ClawHub-only item.

## Best Possible Solution

Keep the PR open and either merge it after normal maintainer validation or recreate the same narrow fix on main: allow only `https://tweakcn.com` in the Control UI `connect-src` directive and add a regression assertion for the CSP header. Do not treat broader configurable CSP issue #47565 as superseding this focused bundled-feature fix.
