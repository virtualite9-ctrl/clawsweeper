## Summary

Keep open. Current main still serves a Control UI CSP with `connect-src 'self' ws: wss:` only, while the bundled custom-theme importer performs a browser fetch to `https://tweakcn.com/r/themes/{id}`. The PR is a narrow fix to a real current-main mismatch, with a regression assertion matching the affected header; the Greptile review also treated it as a minimal, well-scoped CSP change.

## Best Possible Solution

Keep the PR open for maintainer review and likely merge or apply the same narrow change on main: add the exact HTTPS tweakcn origin to Control UI `connect-src` and keep the regression assertion. The broader configurable CSP request should remain separate; it is not a reason to close this focused bug-fix PR.
