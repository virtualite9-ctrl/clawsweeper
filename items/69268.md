## Summary

Keep PR #69268 open. Current main does not include the PR's two core behavior changes: wrapped/default Anthropic sessions are still treated as custom stream functions instead of being forced through the boundary-aware Anthropic transport, and the runner still lacks the PR's missing-user-tail/trailing-assistant transcript cleanup helpers. The PR discussion is also active and substantive, with author follow-ups addressing bot/Codex review findings, so this is not conservative cleanup material.

## Best Possible Solution

Keep the PR open for maintainer review or a successor patch. The best path is to land a narrowly reviewed version that routes the pi-coding-agent default wrapper through the boundary-aware Anthropic transport, adds finalized-prompt-aware trailing non-user cleanup, always writes back sanitized transcript state, and preserves or deliberately updates the custom stream-function contract with focused regression tests.
