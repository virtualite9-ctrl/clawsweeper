## Summary

Keep PR #69268 open. Current main has the direct Anthropic adaptive-thinking payload shape, but it does not implement the PR's two central fixes: wrapped pi-coding-agent stream functions are still treated as custom and bypass boundary-aware transport, and the runner still lacks the proposed trailing-assistant/missing-user-tail cleanup path. The PR discussion also contains concrete production symptoms, real Opus 4.7 verification, and follow-up commits addressing multiple review findings, so this is not stale or superseded cleanup.

## Best Possible Solution

Keep the PR open and review it as a plausible agents/Anthropic bug fix. The best path is to land or adapt the boundary-aware Anthropic routing and trailing non-user transcript cleanup in `src/agents/pi-embedded-runner` and `src/agents/pi-embedded-helpers`, with an explicit maintainer decision on custom stream-function precedence and fresh regression tests for wrapped Anthropic streams and assistant-tail cleanup.
