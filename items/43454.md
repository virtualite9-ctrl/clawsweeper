## Summary

Keep open. Current main still does not expose `subagent:ended`, `agent:turnComplete`, or tool-call-threshold events through the workspace/internal hook system. The Plugin SDK has adjacent lifecycle hooks, and open PR #18889 partially overlaps agent/tool lifecycle boundaries, but neither fully supersedes this issue’s requested workspace hook API and subagent completion bridge.

## Best Possible Solution

Keep this issue open for maintainer API judgment. The likely path is either a narrow internal-hook bridge for `subagent:ended` plus well-defined turn/tool-count lifecycle events, or explicit maintainer guidance that these workflows should use Plugin SDK hooks such as `subagent_ended`, `after_tool_call`, and `agent_end` instead of expanding workspace/internal hooks.
