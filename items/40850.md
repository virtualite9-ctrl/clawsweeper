## Summary

Current `main` does not implement this PR's proposed multi-Node-environment warning. The installer now reports the active Node/npm paths and enforces that the active shell ends up on a supported Node version, but there is no `warn_multiple_node_environments()`-style scan or any equivalent warning about multiple installed Node managers. That means this PR is not safely closable as implemented-on-main or obsolete.
