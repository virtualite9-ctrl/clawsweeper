## Summary

Keep open. Current `main` still implements `generic_repeat` as warning-only, and both unit and e2e tests explicitly lock in that behavior. The PR also carries unrelated Mattermost and MiniMax changes; one of those MiniMax changes is already on `main`, while the core loop-detection change has an unresolved regression concern about blocking repeated calls without no-progress evidence.
