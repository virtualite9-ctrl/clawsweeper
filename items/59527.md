## Summary

Current `main` already implements the reported Linux browser-path behavior: the browser runtime carries `browser.executablePath` into resolved config, prefers that explicit path before auto-detection, recognizes `/usr/bin/google-chrome-stable` on Linux, and surfaces the effective configured/detected path in browser status. The same implementation is present in release `v2026.4.23`.

## Evidence
