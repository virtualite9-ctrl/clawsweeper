## Summary

Current `main` already ships the requested Control UI settings overhaul: the Config tab defaults to a Quick Settings card layout, the advanced form is grouped into category-based accordion navigation, and the form layer has section metadata plus search/filter support. That is strong enough to treat this broad UX request as implemented.
