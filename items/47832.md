## Summary

Current `main` already refreshes persisted skill snapshots when the cached snapshot version is stale, including the existing-session paths described in #47832. The agent command path now recomputes `needsSkillsSnapshot` from both new-session state and `shouldRefreshSnapshotForVersion(...)`, and session-update/cron paths use the same version-based invalidation.

## Evidence
