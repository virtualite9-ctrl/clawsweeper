## Summary

Keep #22358 open. Current main has subagent lifecycle hooks and improved delivery plumbing, but it still does not expose the requested post-subagent-completion, pre-announcement plugin hook that can inspect the completion output, see delivery success/failure, and enrich or suppress the announcement. The issue discussion also includes a recent maintainer/Codex comment explicitly keeping this exact scope open.

## Best Possible Solution

Keep #22358 open until OpenClaw adds a public plugin hook at the post-subagent-completion/pre-announcement point, with completion output, child/session metadata, delivery result/failure visibility, and explicit announcement enrichment or suppression semantics. If maintainers prefer a broader design, fold this exact scope into a canonical lifecycle/delivery issue instead of closing it as already covered.
