## Summary

Keep open. Current `main` does not already implement this PR’s user-visible behavior, and the proposed PR still appears incomplete: typing TTL is still the default 2 minutes on `main`, timeout failures still use the generic failure copy, and the fallback runner can include an extra configured-primary attempt that matches the review concern about under-budgeting typing TTL.
