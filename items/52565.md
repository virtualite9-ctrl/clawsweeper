## Summary

Keep open. The issue body's claimed root cause is not supported by the code I inspected, but I also do not have strong evidence that current `main` fixed the reported UI behavior. The attach and voice buttons are still wired on `main`, the claimed `Ctrl+Shift+U` shortcut does not exist in the repo, and the same button wiring was already present in the latest shipped release snapshot I could inspect, so this does not meet the bar for a conservative close.
