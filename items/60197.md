## Summary

Current main already honors the `/tools/invoke` request `sessionKey` for `sessions_spawn({ runtime: "acp" })`, and ACP run-mode children are intentionally parent-owned background sessions that report completion back to the requester session rather than running as peer chat in that requester's session key.
