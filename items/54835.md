## Summary

Current `main` already fixes the duplicate plugin re-registration path this PR targeted by reusing compatible active registries and caching snapshot/discovery loads, so the PR is obsolete and its logger-muting approach is no longer the right fix.

## Evidence

  - file: [src/plugins/runtime/runtime-registry-loader.ts:76](https://github.com/openclaw/openclaw/blob/443b837bd521b0485c87bb7359f370a4ccec10fa/src/plugins/runtime/runtime-registry-loader.ts#L76)
