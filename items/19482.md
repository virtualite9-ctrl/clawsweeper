## Summary

Keep #19482 open. Current main still unconditionally recreates `HEARTBEAT.md` in `ensureAgentWorkspace({ ensureBootstrapFiles: true })`, which breaks the documented/runtime distinction where a missing heartbeat file keeps heartbeat active but an effectively empty file skips as `empty-heartbeat-file`. The PR remains a narrow fix for a real current-main behavior gap, though its negative regression test should be tightened before landing.

## Best Possible Solution

Keep the PR open and either land it after tightening the negative regression fixture, or apply the same fix directly on main: only seed `HEARTBEAT.md` for brand-new workspaces while preserving missing-file heartbeat semantics for existing workspaces.
