## Summary

Keep open. Current `main` does not implement a shared inbound gateway-receipt timestamp: the common inbound context, hook payloads, and agent-facing conversation metadata still expose only a single `timestamp` field, and there are no `receivedAt`/`gatewayReceivedAt` fields anywhere under `src/` or `docs/`.

## Evidence

- **Shared inbound context has only one timestamp field:** `MsgContext` defines `Timestamp?: number` and no separate gateway receipt timestamp field.
  - file: [src/auto-reply/templating.ts:158](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/auto-reply/templating.ts#L158)
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Inbound hook canonicalization forwards only `ctx.Timestamp`:** `deriveInboundMessageHookContext()` maps `ctx.Timestamp` into `canonical.timestamp`; there is no second receipt-time field captured from ingress.
  - file: [src/hooks/message-hook-mappers.ts:114](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/hooks/message-hook-mappers.ts#L114)
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Plugin hook contracts still expose only `timestamp`:** Both `PluginHookInboundClaimEvent` and `PluginHookMessageReceivedEvent` define `timestamp?: number` and generic `metadata`, with no typed `receivedAt`/`gatewayReceivedAt` field.
  - file: [src/plugins/hook-message.types.ts:16](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/plugins/hook-message.types.ts#L16)
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Agent-facing inbound metadata includes only `timestamp`:** `buildInboundUserContextPrefix()` builds `conversationInfo` with `timestamp: timestampStr` and no gateway receipt timestamp alongside it.
  - file: [src/auto-reply/reply/inbound-meta.ts:200](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/auto-reply/reply/inbound-meta.ts#L200)
