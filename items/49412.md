## Summary

Current `main` does not provide strong evidence that #49412 is fixed or obsolete. The duplicate-warning path is still active for bundled-vs-separately-discovered plugin IDs, and tests on `main` explicitly lock in that warning. `main` only suppresses duplicates when both candidates resolve to the same physical plugin directory, and it adds a doctor repair path for legacy bundled `plugins.load.paths`, which is a mitigation but not proof that this startup warning is gone.
