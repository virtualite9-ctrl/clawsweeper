## Summary

Current main already fixes PR #61824's user-visible bug through a different implementation: logging settings are read directly from the active openclaw.json path and logger resolution uses that result for logging.file. The fix is covered by focused tests and recorded in the unreleased changelog as fixing #61295.

## Best Possible Solution
