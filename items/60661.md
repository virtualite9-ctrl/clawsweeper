## Summary

Keep open. Current `main` still forwards `timeout` into the gateway-owned `agent` request, still falls back to embedded on any gateway error, and does not implement the PR's accepted-ownership guard in `src/gateway/call.ts`. The PR as written also remains incomplete because the imported accepted-ownership symbols are not exported on `main`, matching the existing review comments.
