## Summary

Current main already covers the practical request through the bundled `qwen` provider, global agent model defaults, and built-in usage tracking. The only obsolete part is the requested "Free"/portal-style surface: OpenClaw now documents that the old Qwen free-tier OAuth path was removed, and `qwen3.6-plus` is supported via the standard pay-as-you-go Qwen endpoint instead.
