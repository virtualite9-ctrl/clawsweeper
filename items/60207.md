## Summary

Keep open. Current `main` still has separate normal-reply and heartbeat outbound paths, heartbeat duplicate suppression is scoped only to prior heartbeat sends, and the shared outbound delivery layer only guards queue ownership/retry delivery rather than suppressing semantically duplicate content across reply sources.
