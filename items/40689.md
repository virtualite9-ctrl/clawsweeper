## Summary

Keep open. Current `main` still does not implement a `tool:*` event in the internal Hooks system the issue asks for. Internal hooks remain limited to `command`, `session`, `agent`, `gateway`, and `message` events, while tool-call interception exists only in the separate plugin hook system (`before_tool_call` / `after_tool_call`). That means the requested internal hook capability is not implemented on `main` well enough to close this issue.
