## Summary

Current `main` no longer exposes the private `qa` CLI from packaged installs, rejects loading that CLI outside a source checkout, and keeps completion-cache generation alive even if a subcommand registration fails. The 2026.4.9 npm packaging regression described here is therefore no longer present on `main`.
