## Summary

Keep open. Current `main` does not implement declarative per-DM/per-group/per-topic reply TTS voice overrides. The repo still exposes global outbound TTS at `messages.tts`, shared channel routing only for `channels.modelByChannel`, Telegram DM/group/topic config without any TTS fields, and a Discord-only `voice.tts` override limited to Discord voice playback.
