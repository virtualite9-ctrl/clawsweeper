## Summary

Current `main` already avoids the reported bundled-skill root-escape warning without copying whole `skills/` trees. The runtime overlay copies each `SKILL.md` into `dist-runtime`, skill discovery only validates the skill directory plus `SKILL.md` realpaths, and the existing regression test on `main` still intentionally expects companion files under `skills/` to remain symlinked. That makes this PR's broader copy-tree change obsolete against current behavior.
