## Summary

This PR should stay open. Current main still has the flat-only dated-memory matcher, so nested dated memory files still bypass path-based temporal decay and can be treated as evergreen memory; the linked #67594 remains unresolved unless this PR or an equivalent fix lands.

## Best Possible Solution

Keep this PR open for maintainer review and landing, or apply an equivalent memory-core fix that broadens DATED_MEMORY_PATH_RE for nested dated memory files while preserving evergreen behavior and the added regression coverage. After that lands, #67594 can close as implemented.
