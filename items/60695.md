## Summary

Keep open. Current `main` still has no `.devcontainer`, and this PR's proposed config is not strong enough to treat as ready cleanup: it adds only `mcr.microsoft.com/devcontainers/universal:2` with empty `features`, while the repo currently requires Node `>=22.14.0` and `pnpm@10.33.0` for the documented source workflow.
