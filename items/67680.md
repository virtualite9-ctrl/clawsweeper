The linked implementation PR #67694 is still open and unmerged with closing syntax for this issue, so this issue should remain the tracker while maintainers review, replace, or close that implementation candidate. Current main still has the bare `markdown-it-task-lists` import on the Control UI markdown render path and lacks the proposed UI build preflight/Vite config changes or a production bundle regression guard.
