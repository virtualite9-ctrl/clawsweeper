## Summary

Keep open. Current `main` still intentionally limits `context1m` to Anthropic-owned paths, and this PR only changes context-window accounting. It does not add the provider-specific transport/header behavior needed to actually request Anthropic's 1M beta on proxied providers like `github-copilot`, so there is not strong evidence this PR is the correct fix.
