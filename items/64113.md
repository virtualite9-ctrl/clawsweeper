## Summary

Keep open. Current `main` does not include a bundled Yep provider, and the repo still actively supports bundled web-search plugins through the same plugin SDK pattern this PR uses. The docs and vision prefer external plugin distribution, but they stop short of making new bundled provider PRs auto-closeable; without a maintainer scope rejection or an existing ClawHub plugin linked in the discussion, there is not strong enough evidence for a conservative close.
