## Summary

Keep #71135 open. Current main does not implement the session working-context capsule or an equivalent persisted machine-readable working-state path, and the linked #67511 umbrella remains open. The PR is recent, draft, narrowly scoped, and contains concrete implementation/tests for a real gap, so it is not a conservative auto-close candidate even though it does not wire runtime/status/compaction behavior yet.
