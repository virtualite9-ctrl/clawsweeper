## Summary

Keep open. Current `main` still lacks the Bun-specific `ERR_MODULE_NOT_FOUND` handling this PR adds, and Bun remains a documented optional CLI/dev runtime, so this is still a live user-facing gap rather than an already-fixed or out-of-scope change.
