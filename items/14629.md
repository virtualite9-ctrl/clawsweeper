## Summary

Current main still has the sanitizer gap described in #14629: final tags are removed as plain text, duplicate collapse is still blank-line paragraph-only, and existing tests do not cover same-line, single-newline, or partial-prefix duplicate cleanup. The prior April 26 maintainer-side Codex comment also explicitly kept this issue open for the same surface.

## Best Possible Solution

Keep #14629 open. The best path is a narrow sanitizer change in `src/agents/pi-embedded-helpers/sanitize-user-facing-text.ts` that handles same-line/single-newline repeats, the reported `</final>` boundary duplicate shape, and conservative partial-prefix duplicates, with focused tests in `src/agents/pi-embedded-helpers.sanitizeuserfacingtext.test.ts` that also preserve legitimate repetition and ordinary formatting.
