## Summary

Keep open. Current main still lacks Feishu normal-reply `message_sent` / internal `message:sent` hook emission, while PR #72138 contains targeted Feishu-owned work for the open linked regression #72125.

## Best Possible Solution

Keep #72138 open for maintainer review and validation. The best path is a Feishu-owned fix that emits canonical plugin `message_sent` and internal `message:sent` hooks inside `createFeishuReplyDispatcher`, passes routed session/mirror metadata from `bot.ts`, and preserves the streaming-card `messageId` and streaming-plus-media failure semantics now present in the PR head. Close linked #72125 only after this or an equivalent fix lands on main.
