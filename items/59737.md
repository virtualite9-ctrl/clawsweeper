## Summary

Keep open. Current `main` still uses a single comprehensive PR template, does not include a `.github/PULL_REQUEST_TEMPLATE/` chooser directory, and `CONTRIBUTING.md` does not document a quick-fix vs full-change template flow. The request is therefore not implemented on `main`, and no other close reason fits.
