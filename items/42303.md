- The PR would set HOMEBREW_* and prepend Linuxbrew PATH entries for all images, including builds where OPENCLAW_INSTALL_BREW is not enabled.
- The PR is based on an older Dockerfile shape and adds COPY/build steps in the runtime area rather than preserving current main's multi-stage runtime-assets flow.
- The added OPENCLAW_SKIP_BUILD error points users at openclaw build, but current CLI descriptors do not register a build command.
