## Summary

Current main already has proactive pre-prompt compaction, proactive pre-compaction memory flush, docs describing that behavior, and targeted tests. The issue’s claim that OpenClaw only compacts reactively after a model-side overflow is obsolete against current main and the latest release tree checked here.
