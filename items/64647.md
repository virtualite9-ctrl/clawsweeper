## Summary

Keep this PR open. Current main still does not expose `agents.list[].memory` or resolve memory backend/QMD config per agent, while the PR body and requester comment describe a focused implementation candidate for the closed rollout request in #64612.

## Best Possible Solution

Keep this PR open as the active implementation candidate. The best maintainer path is to refresh it on current main, fix any remaining typecheck issue, preserve the intended boundary (`agents.list[].memory.backend` and `agents.list[].memory.qmd.*` with `memory.citations` global), regenerate schema artifacts as needed, and land it with the included schema/runtime/docs regression coverage or an equivalent maintainer-owned patch.
