## Summary

Keep open. Current `main` already contains the core `requireAuth: true` change for `/bot-approve`, so this PR is not cleanly 'implemented elsewhere' in the sense of being redundant; it still carries useful follow-up work that is not on `main`, notably the QQ-only framework guard and the `/bot-approve ?` help-path fix raised in review.
