## Summary

## Risks / Open Questions

- The specific regression test proposed in this PR is not present on current `main`, so future safety still depends on `src/routing/session-key.ts` staying browser-safe.
- Two Control UI entry points still import `src/routing/session-key.js`, so a future server-only dependency added there could reintroduce the class of bug unless guarded elsewhere.
