## Summary

Issue #24597 is the same remaining product request as canonical open issue #18160, which tracks direct exec/shell cron jobs that bypass the LLM while preserving OpenClaw cron observability and delivery. Current main still does not implement the feature, and open PR #47628 is the active implementation attempt; #51276 attempted the same `payload.kind: "exec"` path but was closed unmerged.

## Best Possible Solution

Close #24597 as a duplicate of canonical open issue #18160. Keep the remaining design and implementation work focused on #18160 and PR #47628, with an additive cron payload/protocol/runtime/UI/docs change that runs deterministic scripts without creating an agent/LLM turn while preserving cron observability and delivery semantics.
