## Summary

Keep #40044 open. Current main at 631552c554095b0c520376db9131dac0f5427f94 still does not expose the current session UUID in the system-prompt `Runtime:` line, and the PR remains a focused additive agents prompt change rather than obsolete, duplicate, or ClawHub-scoped work.

## Best Possible Solution
