## Close Comment

Closing from a Codex review because I could not reproduce the reported failure path on current `main` (`b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503`). The state-dir migration hook is currently reached through doctor preflight (`src/commands/doctor-config-preflight.ts:66`), not the gateway restart path.

If you still hit this on a newer build, please open a fresh issue with the exact command path that produced the warning, `gateway-restart.log`, `openclaw gateway status --deep`, and whether `openclaw doctor` or any wrapper script ran before restart.
