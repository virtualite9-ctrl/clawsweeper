## Summary

Current main still matches the reported duplication: the subagent system prompt embeds the task text, the initial child user message also includes the same task, and `spawnSubagentDirect` sends both to the gateway as `extraSystemPrompt` plus `message`. The linked fix PRs are open and unmerged, so the issue should remain open as the canonical bug tracker until a fix lands.

## Best Possible Solution

Keep #72019 open until one focused fix lands. The likely best code path is to keep the full task in exactly one place, add a regression test covering the final `agent` call payload, and preserve multiline task formatting if the remaining copy lives in the system prompt.
