## Summary

Keep open. Current main still sends the native subagent task through both the extra system prompt and the first child user message. The linked ClawHub skill is a useful workaround, and #72028 / #72053 are active implementation candidates, but no fix is present on main yet.

## Best Possible Solution

Keep this issue open and review/land one canonical fix, likely #72053 or #72028 after maintainer choice. The fix should single-source the native subagent task text, preserve multiline task formatting, and add a focused regression test that inspects both the extra system prompt and the first gateway message.
