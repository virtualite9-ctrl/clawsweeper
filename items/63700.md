## Summary

Keep this issue open. Current main still exposes `openclaw infer model run` as a prompt-only command and forwards only text through both local and gateway paths. Separate image/audio/video infer commands partially overlap, but they do not provide the requested custom prompt plus repeatable file attachments for multimodal model inference.

## Best Possible Solution

Keep this issue open and implement the missing core CLI capability. Add repeatable `--file` handling to `src/cli/capability-cli.ts`, MIME-detect and validate supported media, forward image files to local `agentCommand` via `images`/`imageOrder`, forward gateway runs via `attachments`, make an explicit decision for audio behavior, then update `docs/cli/infer.md` and add focused CLI tests for single image, multiple images, unsupported media, and local/gateway forwarding.
