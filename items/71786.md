## Summary

Keep PR #71786 open. Current main already has the non-mutating `toSorted()` change, but the Pi TUI type-contract cleanup is still real: `paged-select` keeps a stale local `CustomUiContext` render signature and `/files` still passes object values through `SelectItem.value`. The latest PR head addresses much of that remaining gap, though it still needs maintainer cleanup before merge.

## Best Possible Solution

Keep the PR open for maintainer review or contributor revision. The best path is a narrow cleanup that uses the real Pi `ExtensionCommandContext` and `SelectItem` contracts, keeps `container.render(width)` returning `string[]`, maps object payloads through string keys plus lookup maps for all affected callers, and removes whitespace/no-final-newline churn before merge.
