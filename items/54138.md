## Summary

Keep open. Current `main` still has the heartbeat-specific passthrough in `src/auto-reply/reply/agent-runner-payloads.ts`, and the shared assistant-text sanitizers only strip XML tool markup and title-case downgraded `[Tool Call: ...]` markers, not uppercase bracket blocks like `[TOOL_CALL]...[/TOOL_CALL]`. I did not find code or test evidence that this exact heartbeat leak is fixed on `main`.
