## Summary

Keep open. Current `main` does not implement this PR’s expanded MiniMax VLM routing: core code still treats `MiniMax-VL-01` as the only MiniMax VLM model, MiniMax chat models remain text-only in the bundled config/onboarding path, and the VLM HTTP request still does not forward a selected model id. The PR also has a substantive unresolved review concern around routing `MiniMax-M2.7`/`MiniMax-M2.7-highspeed` through an endpoint that currently receives only `prompt` and `image_url`.
