## Summary

Keep open. Current `main` does not include this optimization, and none of the allowed close reasons fit. The code path in `ui/src/ui/app-gateway.ts` still allocates a new array on every event, while `ui/src/ui/app-settings.ts` still aliases `eventLog` to `eventLogBuffer` on debug-tab refresh.
