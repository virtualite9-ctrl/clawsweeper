## Summary

Keep open. Current `main` still does not implement the PR’s `params.num_ctx` threading for configured Ollama models, so this is not an `implemented_on_main` cleanup candidate. The PR discussion also shows the author already addressed the main review nits (merge precedence, canonical key handling, fractional guard), which makes this still-useful work rather than stale noise.
