## Summary

Keep #20237 open. Current main implements several pieces: WebUI cron list/edit/run/history, delete-after-run UI exposure, browser Web Push subscription, and a chat context notice with a Compact action. The core requested behavior still remains: native in-app toast/popups for cron results, cron popup action buttons, and snooze/reschedule controls are not present in the current Control UI or cron RPC surface.

## Best Possible Solution

Keep this open until the missing WebUI notification work is implemented or explicitly descoped. A good implementation path would add a native Control UI toast/popup layer, wire cron run completion events into actionable result notifications, add snooze/reschedule actions either through cron.update scheduling or a dedicated validated RPC if needed, and decide whether the context warning should remain inline or also emit toast-style prompts.
