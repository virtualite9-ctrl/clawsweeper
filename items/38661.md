## Summary

Keep open. Current main still lacks the TUI Ctrl+C active-run hint requested by #38501, and this PR contains a concrete patch for that remaining user-facing bug. The PR should not merge as-is because the bundled web_search rename conflicts with current main policy/tool-catalog behavior, while Feishu, bootstrap-cache, and Telegram portions are already handled or superseded on main.

## Best Possible Solution

Keep the PR open as useful contributor evidence for #38501, but do not merge the bundled branch as-is. The maintainer path is a focused TUI-only patch in `src/tui/tui.ts` that appends `run still active (Esc or /abort to stop)` when `activeChatRunId` is set, plus a small regression test for the status text. Preserve current main's `web_search`, Feishu media routing, bootstrap-cache rollover invalidation, and Telegram slash-session behavior.
