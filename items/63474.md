## Summary

Keep open. Current main does not implement the PR's stale `installs`/`plugins` channel-key repair, and the linked bug remains a real Feishu upgrade path concern. The submitted PR is not ready as-is because review comments identify reachability gaps for non-Feishu and account-only stale keys, but the underlying work is still actionable in this repository rather than obsolete cleanup.

## Best Possible Solution

Keep the PR open or replace it with a maintainer-approved fix that adds a doctor-owned legacy migration for stale `channels.*.installs` and `channels.*.plugins` keys, including nested `accounts.*`, with rules broad enough to trigger for all affected channel configs and regression coverage through the validated migration path. The linked issue #63101 should remain the canonical bug until that repair ships.
