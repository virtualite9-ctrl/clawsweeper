## Summary

Keep this issue open. Current main has partial defenses for tagged reasoning, reasoning-only retries, and final-tag enforcement, but it still does not cover the broader reported leak shape: Gemini reasoning/planning text delivered as ordinary channel text, including multilingual `Reasoning:`-style preambles. The related final-tag fix #48593 and broader preamble sanitizer candidate #60551 are still unmerged, so this remains active maintainer work.

## Best Possible Solution

Keep this issue open as the umbrella for Gemini plain-text reasoning leakage. The best path is a narrow, tested shared delivery sanitizer and/or Gemini provider/stream fix that covers untagged planning text, appended reasoning blocks, multilingual reasoning headers, and tag-fragment variants without stripping normal user-facing prose. Review #60551 for the broader preamble cleanup and keep #48593 scoped to final-tag fragments.
