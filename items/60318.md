## Summary

PR #60318 is superseded by merged PR #56924, which fixed the same Gateway Access token/password visibility-toggle layout problem on main. Current main contains the flex/min-width/box-sizing fix in the Overview access form, and the changelog records the shipped Control UI/Overview fix in v2026.4.5.
