## Summary

Current `main` already hardens the Claude/Anthropic tool-schema normalization path, but the separate upgrade report is not clearly implemented away: OpenClaw still documents and surfaces `gateway restart` as a required/manual post-update step, so there is not strong enough evidence to conservatively close the combined issue.

## Evidence
