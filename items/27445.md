## Summary

Keep #27445 open. Current main has adjacent subagent announce routing improvements, but it still does not expose the requested native `announceTarget` config default or `sessions_spawn` per-call override, and top-level dormant native completions still use the external channel delivery path by design.

## Best Possible Solution

Keep #27445 open as a core subagent announce policy request. The likely maintainer path is to add a public parent/channel announce policy to native `sessions_spawn` and `agents.defaults.subagents`, update the strict config schema, generated/help docs and https://docs.openclaw.ai/tools/subagents, and cover top-level parent-only routing, current channel delivery, active requester steering, nested subagent delivery, and interactions with #33478 callback/thread hardening.
