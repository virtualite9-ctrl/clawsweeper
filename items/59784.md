## Summary

Current `main` only reclassifies certain temporary 402 quota-window messages to `rate_limit`. The reported Claude Code OAuth shape (`"You've hit your limit · resets 6pm (UTC)"`) is not covered by the current matchers, so a 402 carrying that text still falls through to `billing`, which matches the issue report and can block model fallback.
