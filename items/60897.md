## Summary

Keep open. Current main and the latest shipped release still use the same Control UI cache-hit calculation, the repo only has generic troubleshooting for low cache reads, and the later changelog entries are broad prompt-cache improvements rather than a clear, issue-specific fix for this MiniMax/Control UI report.
