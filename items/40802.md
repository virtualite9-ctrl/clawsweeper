## Summary

Keep open. I did not find strong evidence that #40802 is already fixed or obsolete on current `main`. The Feishu DM reply path still routes direct replies through the same target-resolution/send code, and those core files are byte-identical to the latest shipped release `v2026.4.22`, so there is no maintainer-grade proof of an implemented fix.
