## Summary

Keep open. Current `main` still uses lazy placeholder commands without disabling Commander’s built-in help handling, and the current tests do not cover the reported `lazy-subcommand --help` regression. The discussion shows this PR’s fix remains unmerged and was later re-spun as #69930, so there is no strong basis to close this PR as already implemented or obsolete.
