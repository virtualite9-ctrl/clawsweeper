## Summary

Current main has a first-class external web-search provider plugin surface, runtime resolution for plugin-owned web_search backends, and explicit plugin-first scope guidance. An OmniRoute or generic REST search adapter belongs as a ClawHub/community plugin, not a new OpenClaw core backend mode.
