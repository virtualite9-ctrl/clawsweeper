## Summary

Keep #38467 open. Current main still exposes WhatsApp directory peers from configured allowFrom entries only, not from Baileys contacts.upsert, and the WhatsApp socket setup does not register a contacts.upsert handler. The related PR #33924 was closed unmerged and covered outbound allowSendTo gating, not persisted contact-directory lookup.
