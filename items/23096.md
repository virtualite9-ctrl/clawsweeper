## Summary

Keep PR #23096 open. Current main has the generic SecretRef env/file/exec architecture, but it still does not include the Bitwarden/Vaultwarden docs, bundled bw exec resolver, or Bitwarden-specific tests proposed by this PR. The PR was rewritten after #26155 to fit the merged exec-provider baseline, and the discussion includes maintainer signal to treat it as a focused follow-up rather than stale cleanup.

## Best Possible Solution
