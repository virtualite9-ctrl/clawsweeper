## Summary

Current `main` already implements Feishu message reactions: the channel advertises `react`/`reactions`, executes add/remove/list against Feishu message-reaction APIs, and handles inbound reaction webhook events. The implementation is on `main` but does not appear to be in the latest tagged release `v2026.4.22`, so the fix is unreleased.
