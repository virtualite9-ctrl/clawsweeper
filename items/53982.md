## Summary

Keep open. Current `main` still tells users to register the Nextcloud Talk bot with only `--feature reaction`, and the send path still turns HTTP 401 into the misleading `check bot secret` error. I found no current docs or code on `main` that mention the required `--feature response` flag or detect this specific failure mode.
