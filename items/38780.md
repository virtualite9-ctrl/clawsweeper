## Summary

Keep PR #38780 open. Current main has related restart-continuation, session-delivery, incomplete-turn retry, and bootstrap `continuation-skip` substrate, but it does not implement the PR's central self-elected continuation surface: `continue_work`, `continue_delegate`, `request_compaction`, response-token fallbacks, continuation config, delegate queue/chaining, context-pressure runtime, or gateway trigger fields. ## Best Possible Solution

Keep PR #38780 open for explicit maintainer/product review. If maintainers reject the product direction or want it outside core, close it deliberately as a maintainer decision rather than automated cleanup.
