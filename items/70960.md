In OpenClaw core, make Control UI agent/session selectors consistently prefer `agents.list[].name` before identity-derived names, and restore or intentionally replace the chat header/topbar token and cost usage surface using existing session usage fields. Route the QClaw Windows installer deletion report to the QClaw/Electron installer owner with the supplied install log and registry details.
