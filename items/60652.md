## Summary

Current `main` still reproduces the report by source inspection: Discord advertises `send`, `poll`, and reaction actions, but not `upload-file`, while the Discord send handler still accepts `media`/`path`/`filePath`. The message tool schema/action enum is built from that discovery output, so the discoverability gap remains actionable on `main`.
