## Summary

Keep this PR open. Current main still returns early for fresh persisted token totals before token-threshold evaluation, so the token-based proactive compaction re-fire requested by this PR is not implemented. The linked bug remains open, this PR is unmerged, and the overlapping #66716 is also open rather than a shipped superseding fix.
