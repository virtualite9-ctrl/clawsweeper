## Summary

Keep open. Current `main` still implements Kimi `web_search` as Moonshot chat-completions with native `$web_search`, and it still returns the model's final natural-language `message.content` as a successful search result whenever tool-calling stops. That matches the issue's reported failure mode rather than fixing it. Docs now clearly describe Kimi as an AI-synthesized answer provider, but there is still no code guard that rejects non-grounded fallback text or requires structured search results.
