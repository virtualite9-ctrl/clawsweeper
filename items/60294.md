## Summary

Keep open. Current `main` does not implement Android Chinese localization: the app still ships only the default `values/` resource set, and major Android UI surfaces are hard-coded in English instead of using localized string resources.

## Evidence

- **Android resources only define the app name:** `apps/android/app/src/main/res/values/strings.xml` contains only `<string name="app_name">OpenClaw Node</string>`, and the Android resource tree has no `values-zh*` locale directory.
