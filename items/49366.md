## Summary

Keep open. Current `main` and the latest release still merge config-backed, state-dir `.env`, and auth-profile env-ref values into the gateway install environment, and that environment is written into service metadata. The PR’s security hardening is not implemented on `main`, while the PR discussion also shows the later review regression around auth-profile env refs was addressed in the PR branch.
