## Summary

Current main already contains targeted fixes for the reported `npm -g` + `memory-lancedb` + Feishu regression path, and those fixes are present in release `v2026.4.22`. The code now preserves restrictive `plugins.allow` during plugin install, repairs missing bundled runtime dependencies before loading enabled bundled plugins (including Feishu setup/runtime paths), and makes `memory-lancedb` retry failed LanceDB loads while surfacing unsupported Intel macOS clearly instead of poisoning later turns.
