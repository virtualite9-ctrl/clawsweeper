## Summary

Keep this PR open. Current main still does not expose the proposed Plugin SDK `api.runtime.llm.complete()` capability, and context-engine runtime contexts still do not receive an LLM helper. The PR remains a plausible implementation candidate for the open Plugin SDK inference gap tracked by #45865, with maintainer design review still needed around active session agent/model/auth selection.

## Best Possible Solution

Keep this PR open for maintainer review or replacement by a revised implementation. The best path is an additive `api.runtime.llm.complete()` seam with exported SDK types, context-engine injection only where session/agent binding is well-defined, public docs/API-baseline updates, and focused tests proving active-agent model/auth resolution. If maintainers prefer a different API shape, keep #45865 as the canonical product/API discussion.
