## Summary

Keep open. Current `main` still injects a main-session system event for hook runs where `deliver: false`, so this is not implemented on `main`. The proposed PR also appears incomplete on current code: `runCronIsolatedAgentTurn` can return `status: "error"` without throwing, so a blanket `value.deliver !== false` suppression would also hide some real hook failures instead of preserving error surfacing.
