## Summary

Keep open. Current `main` only partially addresses the report: Control UI/config-response masking exists, and SecretRef can avoid storing some credentials inline, but file-log redaction is still incomplete and config/backups are still written as plaintext JSON on disk. The issue thread also already converted this into an active enhancement tracker with open follow-up work, so there is not strong evidence for an automated close.
