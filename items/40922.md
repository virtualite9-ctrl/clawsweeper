## Summary

Current `main` already includes Simplified Chinese localization for the Control UI, a persisted language selector in Overview settings, browser-language detection for Chinese locales, translated sidebar labels, docs, and tests. The changelog records this as shipped in `v2026.4.5`.

## Close Comment

Closing as implemented on `main` after a Codex review. The requested Control UI Chinese localization is already present: `zh-CN` is a supported locale with browser-language detection in `ui/src/i18n/lib/registry.ts:13` and `ui/src/i18n/lib/registry.ts:94`; the UI has a persisted language picker in **Overview -> Gateway Access -> Language** at `ui/src/ui/views/overview.ts:252`; and the Simplified Chinese bundle already translates the sidebar/control labels in `ui/src/i18n/locales/zh-CN.ts:148` and `ui/src/i18n/locales/zh-CN.ts:653`. This is also documented in `docs/web/control-ui.md:94`, and the changelog records multilingual Control UI support as shipped in `v2026.4.5` at `CHANGELOG.md:1164` and `CHANGELOG.md:1179`. If there is still a specific screen on current `main` that remains untranslated, please open a new focused issue with the exact page and strings.
