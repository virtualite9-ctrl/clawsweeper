## Summary

Keep open. Current `main` does not contain the PR's core behavior: cron still applies one outer timeout to the whole isolated run, the detached cron path does not propagate a deadline into `runCronIsolatedAgentTurn`, and `runWithModelFallback` still has no pre-attempt stop hook to skip a doomed fallback attempt. I found no matching implementation or history on `main`, so this is not safe to close as already implemented or obsolete.

## Evidence
