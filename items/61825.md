## Summary

Current `main` already wires WhatsApp login through ambient proxy-aware agents, documents that behavior, and has regression coverage for `HTTPS_PROXY`. This matches the reporter’s explicit-`HttpsProxyAgent` diagnosis closely enough to treat the issue as fixed on `main`, but not yet in the latest tagged release.
