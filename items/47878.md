## Summary

Keep open. Current `main` still does not contain the requested `read`/`write`/`edit` tool inventory docs, so this is not `implemented_on_main`. The PR idea is still valid, but the proposed `read` wording is stale against current runtime behavior and needs refresh rather than closure.
