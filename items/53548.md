## Summary

Keep open. Current `main` still hard-couples `mode:"session"` to `thread:true` for both native subagents and ACP spawns, tests still assert that rejection, and docs still describe persistent sessions as thread-bound-only. I found no evidence in the checked-out code that the requested decoupling has landed.
