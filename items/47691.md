## Summary

Keep open. Current `main` does not implement Telegram group multi-bot mention routing or message fan-out between agents. The routing docs still say one inbound message selects one agent, broadcast fan-out is documented as WhatsApp-only, and the Telegram handler explicitly says bots cannot see other bot messages in groups.
