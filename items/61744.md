## Summary

Keep open. Current `main` shows the WhatsApp directory behavior is intentionally config-backed rather than live-discovered, so the empty peers/groups portion is mostly a product/docs expectation mismatch. But the repeated `status 499` reconnect path is still present on `main` and is treated in code/docs as an active runtime problem path, not a shipped fix.
