## Summary

Keep open. Current main has the basic IDENTITY.md avatar resolver and a 2 MB local avatar limit that matches the reporter's later finding, but the linked open PR #38923 still tracks this exact issue and current tests do not prove the reported non-default Windows/webchat flow is fully covered.
