## Summary

Keep open. Current `main` has session selection and a Sessions tab, but it does not implement the session-sidebar/history-preservation behavior described here. `/new` still resets the current key in place and archives the old transcript as `.reset`, and the PR’s own review comments identified unresolved wiring bugs that would leave the proposed sidebar non-functional.
