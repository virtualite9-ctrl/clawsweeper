## Summary

Keep PR #72108 open. Current main has not implemented the proposed `src/**/*.test-helpers.ts` exclusion, and the reported helper file is still included by the plugin SDK declaration config while returning `vi.fn()`-backed values from a Vitest test fixture.

## Best Possible Solution

The remaining work is an actionable build-config fix, not something already implemented on main.
