Keep #35152 open. Current main still lacks the requested on-demand `openclaw doctor` duplicate-installation diagnostic. Main has an adjacent installer-time warning for duplicate npm global installs, and doctor has narrower gateway service checks, but neither implements this PR's binary/version/systemd inventory in doctor.
