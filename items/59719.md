## Summary

Keep open. Current `main` still does not implement the PR’s core behavior for background `exec` liveness/status, and the PR discussion surfaced a real correctness problem around non-exec `runtime:"cli"` tasks that `main` now covers separately in task-maintenance logic/tests. This is not safely closable as implemented, obsolete, or incoherent.
