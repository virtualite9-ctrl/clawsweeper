## Summary

Keep this issue open. Current main still lacks the requested gateway startup warning for legacy CLAWDBOT_*/MOLTBOT_* environment variables, and open PR #53667 is an unmerged implementation candidate with closing syntax for this issue.

## Best Possible Solution

Keep this issue open and review, land, or replace PR #53667. The best fix is a focused gateway startup/runtime-auth diagnostic that detects CLAWDBOT_*/MOLTBOT_* keys early, emits clear OPENCLAW_* replacement guidance plus doctor or migration pointers, updates misleading auth errors, and adds regression tests while preserving the intentional removal of legacy fallback behavior.
