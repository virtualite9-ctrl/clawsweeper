## Summary

Keep open. Current `main` and the latest shipped release intentionally advertise `adaptive` for Claude 4.6 and reserve `xhigh`/`max` for Claude Opus 4.7, while this PR changes a now-deprecated hook to expose `xhigh` for Anthropic 4.6. That means the PR is outdated and conflicts with tested shipped behavior, but there is not strong enough evidence for an auto-close reason.
