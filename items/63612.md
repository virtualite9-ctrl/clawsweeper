## Summary

Keep open. Current main still has the reported crash path: pre-prompt compaction calls OpenClaw's token estimator, which still delegates malformed replay messages directly to pi-coding-agent's unsafe `estimateTokens()` implementation. The linked PR #63636 is relevant and appears to be the right direction, but it is open/unmerged in the provided context, so the issue is not implemented on main yet.

## Best Possible Solution

Keep #63612 open and land #63636 or an equivalent maintainer-owned fix. The best implementation is a shared safe token estimator in `src/agents/compaction.ts` used by preemptive compaction and compaction metrics, plus regression tests covering malformed assistant content, missing block `text`/`thinking`/`name`, missing or non-array toolResult/custom content, missing bashExecution command/output, and missing branch/compaction summary fields.
