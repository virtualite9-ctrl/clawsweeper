## Summary

Keep open. Current main has partial overlap through the existing authenticated `update.run` WebSocket RPC and `hello-ok.server.version`, but it does not satisfy the issue’s requested `version` field on `GET /health`, nor the requested HTTP `POST /admin/update` shape with `target_version` support. The Crab-related use case therefore still has meaningful remaining work.
