## Summary

Current `main` already has iMessage outbound-echo suppression in the monitor path, caches outbound message IDs/text on send, and includes regression tests for both reflected outbound messages and `is_from_me` self-chat echoes. The changelog shows this shipped in `v2026.4.10`, so the issue is obsolete against current OpenClaw.
