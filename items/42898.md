## Summary

Keep this PR open. Current main still classifies dotted read-only gateway actions by checking the full action string against flat read-only verbs, so `config.schema.lookup` remains treated as mutating even though it is a valid read-scope gateway method. The PR is still a focused implementation candidate, and the related payload-level PR #42962 does not supersede the classifier fix.

## Best Possible Solution

Keep this PR open for maintainer review or landing. The best path is a gateway-only classifier fix with regression coverage for `config.schema.lookup`, `config.get`, a read-only dotted gateway method, mutating writes such as `config.patch`/`config.apply`, missing actions, and unsupported cron/canvas dotted actions. After a classifier fix lands, #42856 can close as implemented; #42962 should only proceed if maintainers still want an additional payload-specific severity policy.
