## Summary

Current main already fixes #67891. The bundled webhooks plugin now registers routes synchronously, the plugin loader rejects Promise-returning register() implementations instead of silently dropping late side effects, and the changelog records the fix as shipped in v2026.4.18 via merged PR #67941.

## Best Possible Solution

Close #67891 as implemented. Keep the shipped synchronous webhooks register implementation and loader-level synchronous contract enforcement from PR #67941; future work should only adjust this area through narrower regressions or documented Plugin SDK contract changes.
