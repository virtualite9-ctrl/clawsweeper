## Summary

This PR adds a provider-specific bundled hook, but current OpenClaw direction is to keep optional capability out of core and ship it as external plugins/hook packs. The repo already supports plugin hooks and hook-pack installation, and the PR itself points to an existing standalone implementation.

## Evidence
