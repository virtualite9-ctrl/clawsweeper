## Summary

Keep open. Current `main` still coerces omitted subagent `runTimeoutSeconds` to `0`, which conflicts with the documented/configured default-timeout behavior the PR is trying to restore.

## Evidence

- **Timeout resolver distinguishes omitted from explicit zero:** `resolveAgentTimeoutMs` uses the configured default only when `overrideSeconds` is `undefined`; an explicit `0` means "no timeout" and maps to the max-safe timer.
  - file: [src/agents/timeout.ts:15](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/timeout.ts#L15)
  - command: `nl -ba src/agents/timeout.ts | sed -n '1,220p'`
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
- **Registry wait-timeout wrapper still rewrites omitted timeout to zero:** Current `main` still calls `resolveAgentTimeoutMs({ overrideSeconds: runTimeoutSeconds ?? 0 })`, so an omitted subagent timeout cannot inherit the configured default.
  - file: [src/agents/subagent-registry.ts:702](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/subagent-registry.ts#L702)
  - command: `nl -ba src/agents/subagent-registry.ts | sed -n '696,715p'`
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
- **Run manager persists the same coercion on register and steer-replace:** `registerSubagentRun` stores `registerParams.runTimeoutSeconds ?? 0`, and `replaceSubagentRunAfterSteer` uses `replaceParams.runTimeoutSeconds ?? source.runTimeoutSeconds ?? 0`, so both paths turn an omitted timeout into explicit no-timeout.
  - file: [src/agents/subagent-registry-run-manager.ts:266](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/subagent-registry-run-manager.ts#L266)

- **Steer restart path also forces zero:** `subagent-control.ts` passes `params.entry.runTimeoutSeconds ?? 0` into `replaceSubagentRunAfterSteer`, so steer restarts also lose the configured default when the original run omitted the timeout.
  - file: [src/agents/subagent-control.ts:604](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/subagent-control.ts#L604)
