## Summary

Current `main` does not implement the requested internal hook events for subagent spawn/complete. The internal hook surface still exposes `agent:bootstrap` only, while subagent lifecycle is handled through separate plugin hooks (`subagent_spawning`, `subagent_spawned`, `subagent_ended`). That is adjacent functionality, but it is not the internal `agent:spawn` / `agent:complete` event capability requested in #52581.

## Evidence

- **Internal hook docs still omit spawn/complete:** The internal hooks event table lists `command:*`, `session:*`, `agent:bootstrap`, `gateway:startup`, and message events, but no `agent:spawn` or `agent:complete` entry.
  - file: [docs/automation/hooks.md:36](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/docs/automation/hooks.md#L36)

  - file: [src/hooks/internal-hooks.ts:32](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/hooks/internal-hooks.ts#L32)
