## Summary

Keep open. Current `main` still intentionally removes the Chrome extension relay path and does not ship a like-for-like cross-machine replacement for the old `driver: "extension"` flow. The only documented remote replacement is the node-host/browser-node path or raw remote CDP, which does not resolve the reporter's request for the removed extension-relay workflow.
