## Summary

Keep open. Current `main` appears to have fixed the SecretRef warning behavior, but the LaunchAgent cleanup-hint bug is still not closed out conservatively: macOS service detection is still based on a broad `openclaw` substring match, the cleanup hint still targets the canonical gateway label instead of the detected plist, and legacy active `com.openclaw.*` LaunchAgents would still be treated as extras.

## Evidence
