## Summary

Keep this PR open. Current main still injects and tests the blanket group-chat instruction that forbids same-group message-tool sends, while the current message tool and Telegram/Slack threading paths already support media sends with same-conversation thread inference. The PR is a focused implementation candidate, not obsolete, but it needs a rebase because current main now has existing group prompt tests and trigger fixtures that still assert the old wording.

## Best Possible Solution

Keep this PR open and review a rebased version.
