## Summary

Keep #16711 open. Current main still lacks the requested chat composer Expand button and large prompt modal; the existing Control UI composer remains an inline, height-capped textarea with Enter-to-send behavior, and the related reports in the provided context cover adjacent but different UI work.

## Best Possible Solution

Keep this open for first-class Control UI work. The likely implementation belongs around `ui/src/ui/views/chat.ts`, `ui/src/ui/chat/run-controls.ts`, and `ui/src/styles/chat/layout.css`: add an Expand control, modal draft state that preserves/cancels/applies/sends text, and modal keyboard behavior where Enter inserts a newline and Ctrl/Cmd+Enter sends. Coordinate with adjacent send-on-enter/session-composer issues, but do not close this as a duplicate unless a canonical issue specifically tracks the expanded long-prompt modal.
