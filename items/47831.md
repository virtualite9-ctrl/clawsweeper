## Summary

Keep open. Current `main` does contain explicit Slack `app_mention` handling for channel traffic, regression coverage for channel `app_mention` routing, and docs that require the expected Slack bot events, but I did not find strong evidence that the specific 2026.3.13 Socket Mode failure in #47831 was definitively fixed on `main` or shipped in a known release.
