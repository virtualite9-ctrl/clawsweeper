## Summary

Keep open. Current `main` does not contain this PR's `memory` CLI preload change, but the codebase now has capability-based fallback paths that may already resolve `github-copilot` memory embedding providers without that catalog entry. That makes the underlying bug status ambiguous, not strongly closable.
