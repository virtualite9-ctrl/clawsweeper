## Summary

Keep this PR open. Current main still includes Linux nvm/fnm/Volta/asdf/Bun paths in the daemon minimal service PATH, and the current regression tests still assert that behavior. The PR remains a focused implementation candidate for the #71578 Linux doctor/minimal-PATH contradiction rather than something implemented or superseded.

## Best Possible Solution

Keep this PR open for maintainer review or land an equivalent patch: make Linux minimal service PATH include stable user bins, PNPM/NPM prefix, Nix profiles, and system dirs while excluding Linux version-manager directories; preserve macOS version-manager behavior; update daemon service-env and service-audit regression coverage for the #71578 hardened-PATH expectation.
