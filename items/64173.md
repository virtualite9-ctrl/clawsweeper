## Summary

Current main already routes bare `/new` and `/reset` through an explicit Session Startup prompt, prepends runtime startup context for fresh sessions, and still injects workspace bootstrap files into `Project Context`; regression tests and the changelog show this behavior was shipped after the reporter's `2026.4.9` build.

## Evidence
