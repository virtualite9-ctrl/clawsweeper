## Summary

Keep open. Current `main` still lacks the requested persisted webchat toggle to expand tool output by default, so this is not implemented on main. The PR also no longer matches the current UI expansion architecture cleanly: `main` now has explicit tool expansion state plumbing, and the discussion already identified real UX/regression issues in the proposed patch.
