## Summary

Keep PR #71399 open. Current main still does not implement an `agent:turn:end` internal hook, and this open PR is the current focused implementation attempt for that ACP turn-completion seam. It is not superseded by the older closed PR #60734 or by the existing plugin `agent_end` hook, and maintainer discussion already identifies review follow-up rather than an obsolete item.

## Best Possible Solution

Keep the PR open for maintainer review. The best path is to finish this PR or a successor by emitting `agent:turn:end` from `AcpSessionManager.recordTurnCompletion()` on both success and failure, carrying `sessionKey`, `success`, `durationMs`, and optional `errorCode`, using bounded non-blocking dispatch or documenting why this hook intentionally differs, and covering the type guard plus manager-level success/failure emission tests. If accepted as an operator-facing internal hook, update the internal hooks docs as well.
