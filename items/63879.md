## Summary

Current `main` has explicit config-write validation, rejected-payload preservation, last-known-good recovery for invalid direct edits/hot reloads, and a narrow guard on agent-driven `gateway` config mutations. That means the reported bad config write path no longer needs to corrupt `openclaw.json` and take the Gateway down.
