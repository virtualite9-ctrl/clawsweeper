## Summary

Keep open. Current `main` still fails one-shot gateway calls on any pre-hello close and still reports connect failures generically, so this PR is not implemented on `main`. The PR also addressed the one substantive review concern with a structured close-error follow-up, so there is still useful unresolved work here rather than something ready for conservative cleanup.
