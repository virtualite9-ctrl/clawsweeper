## Summary

Current main still renders the Control UI Sessions table Updated column with relative time, not the requested full date/time. There is an open implementation PR, #53445, that uses closing syntax for this issue and adds the exact `YYYY-MM-DD HH:mm` display, so this issue should stay open until that PR is reviewed and either landed or closed.

## Best Possible Solution

Keep this issue open while #53445 remains open. Maintainers should review, refresh, and either land that narrow Control UI formatting change or close the PR with a reason; after it merges, GitHub can close this issue as implemented.
