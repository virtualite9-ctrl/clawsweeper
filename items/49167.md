## Summary

Current `main` already contains the Azure/OpenAI Responses replay fixes for the reported 400: Azure is routed through the OpenAI-compatible replay policy, orphaned `reasoning` blocks are stripped before outbound Responses calls, reasoning/function-call pairings are downgraded when replay metadata is incomplete, and the repo includes regression coverage for both cases.

## Evidence
