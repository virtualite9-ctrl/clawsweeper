## Summary

Current `main` already routes the small-model security audit through a generic plugin-backed web-search credential detector, and the bundled Gemini, Grok, Kimi, and Perplexity/OpenRouter providers all declare the env vars this PR was trying to add. The original under-reporting gap is therefore already covered on `main`, so this PR is superseded.
