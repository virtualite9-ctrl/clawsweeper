## Summary

Current main still appears to contain the reported reload-diff behavior: `diffConfigPaths` treats `undefined` versus `{}`/`[]` as a changed path, and `plugins.*` paths still require a gateway restart. The linked PR #72137 is an active fix candidate, but it is open and not merged into current main, so the issue should stay open until that or an equivalent fix lands.

## Best Possible Solution

Keep #72061 open and use #72137 as the active fix candidate. The best code path is a narrow `diffConfigPaths` fix that treats missing and empty config containers as equivalent while preserving real missing-to-populated diffs, plus regression tests in `src/gateway/config-reload.test.ts` for the exact skill-toggle/plugin-config phantom diff scenario.
