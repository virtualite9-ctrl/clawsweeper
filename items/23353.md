## Summary

Keep #23353 open. Current main still does not implement Anthropic native server-side web_search/web_fetch/code_execution, response block handling, or Programmatic Tool Calling allowed_callers. The related PRs #23469 and #23573 are closed and unmerged in the provided context, and the latest maintainer discussion already explicitly kept this tracker open.

## Best Possible Solution

Keep #23353 open as the broad Anthropic provider/runtime feature tracker. The best implementation path is an explicit Anthropic design covering config/schema/docs, request payload injection for web_search/web_fetch/code_execution server tools, managed tool suppression or coexistence rules, response block preservation/rendering, Programmatic Tool Calling allowed_callers behavior, and focused tests. If maintainers want smaller units, split web_search/web_fetch and code_execution/PTC into canonical follow-up issues rather than closing this tracker.
