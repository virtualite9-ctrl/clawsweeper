## Summary

Keep #23353 open. Current main still lacks Anthropic native server-side web_search/web_fetch/code_execution support, Programmatic Tool Calling allowed_callers, and response handling for Anthropic server-tool content blocks. The issue also has a same-day maintainer/Codex review comment explicitly keeping it open, while related implementation PRs #23469 and #23573 are closed unmerged.

## Best Possible Solution

Keep #23353 open as the broad Anthropic native server-tools tracker. The best path is a provider/runtime design covering config and schema, Anthropic request payload injection for web_search/web_fetch/code_execution server tools, suppression or coexistence with managed web tools, response block preservation/rendering, Programmatic Tool Calling allowed_callers semantics, and focused tests/docs. Closed PRs #23469 and #23573 can be used as reference, but they do not supersede the open issue.
