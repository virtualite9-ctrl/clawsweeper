## Summary

Current main already implements the reported fix. The Control UI model picker now builds fully qualified option values, preserves already-qualified refs instead of prepending the active provider, and submits the selected value verbatim to `sessions.patch`. The latest release snapshot `v2026.4.22` contains the same relevant code as current main.
