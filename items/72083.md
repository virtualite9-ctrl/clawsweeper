## Summary

Current main implements the requested maintainer workflow: bind-mounted bundled plugin source overlays under the legacy `extensions/<plugin>` path are discovered before packaged `dist/extensions/<plugin>` bundles, with warning diagnostics, regression coverage, and Docker/plugin docs. The fix is on main after the latest release, so no fixed release tag is known yet.

## Best Possible Solution

Keep the current source-overlay implementation and documentation: mounted bundled plugin sources should override packaged dist bundles for maintainer testing, copied source directories should remain inert for normal packaged installs, and the fix should ship with the next release.
