## Summary

Keep open. Current main still intentionally suppresses commentary-only assistant text in the TUI, so there is not strong evidence that the reported "no assistant reply until completion" behavior is fully fixed. The watchdog work improves stale streaming state, but it does not make commentary-only output render.
