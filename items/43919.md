## Summary

Keep open. Current main now covers the HTTP_PROXY/HTTPS_PROXY gateway and embedded-runner dispatcher bootstrap, but this PR’s distinct ALL_PROXY/SOCKS fallback is still not implemented and current tests explicitly preserve ALL_PROXY-only environments as non-proxied. The PR is stale and needs rebase/security cleanup, but it still contains potentially useful remaining work tied to the open #43821 umbrella.

## Best Possible Solution

Keep this PR open for maintainer judgment. Rebase and narrow it to the remaining ALL_PROXY/SOCKS/global-dispatcher gap, with explicit tests for lowercase precedence, malformed/authenticated proxy URL redaction, NO_PROXY and loopback preservation, and dispatcher replacement. If maintainers want OpenClaw to keep matching Undici’s HTTP_PROXY/HTTPS_PROXY-only EnvHttpProxyAgent semantics, close this only with that explicit product/security rationale and leave #43821 or a narrower follow-up tracking the rejected ALL_PROXY/SOCKS request.
