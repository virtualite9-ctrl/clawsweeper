## Summary

Keep open. Current `main` does not implement time-bound exec escalation: the built-in command surface still exposes only `/elevated` and `/exec`, exec overrides are persisted as ordinary session fields with no TTL/expiry state, and docs still describe `/exec security=full` as a session-level convenience that skips approvals until the session changes. The request is coherent, in-scope for core security/ergonomics work, and not old enough for `stale_insufficient_info`.
