## Summary

Keep open. Current `main` does not implement the requested no-op skip; it still treats matching installed/target package versions as a package-manager refresh path. There is also a real correctness concern in PR #69421 as proposed: its early return would bypass later `update.channel` persistence when a user changes channels but the resolved version is unchanged.
