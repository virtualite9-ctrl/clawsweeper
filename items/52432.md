## Summary

Current main already handles `null` token fields safely in usage normalization, so the crash described by this PR is not actionable anymore. The PR's remaining value is type/test cleanup, while it also bundles unrelated changes that are already present elsewhere on main.

## Evidence

  - file: [src/agents/usage.ts:89](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/agents/usage.ts#L89)

  - file: [src/agents/usage.ts:100](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/agents/usage.ts#L100)
