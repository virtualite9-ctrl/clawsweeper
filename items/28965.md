## Summary

Keep #28965 open. Current main still has partial model normalization and one-session override clearing, but it does not implement the requested `openclaw models reconcile` command or a bulk `openclaw sessions model-reset` workflow.

## Best Possible Solution

Keep this issue open until OpenClaw implements first-class core CLI support for model config reconciliation and bulk session override reset, or a maintainer explicitly marks a canonical issue for the same concrete work. The likely implementation belongs in the core CLI plus config/session-store code, with docs updates for `docs/cli/models.md` and `docs/cli/sessions.md`.
