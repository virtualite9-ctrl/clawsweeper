## Summary

Keep #28965 open. Current main still only has targeted model normalization and per-session override reset paths; it does not implement the requested `openclaw models reconcile` command or bulk `openclaw sessions model-reset` workflow. The prior ClawSweeper comment remains accurate on current main `67b9167b808f2ff065c91665d6f36256a5c23ef2`.

## Best Possible Solution

Keep this issue open until OpenClaw implements first-class core CLI support for model configuration reconciliation plus bulk session model override reset, or a maintainer explicitly designates a canonical issue for that exact work. The likely implementation belongs in the core CLI, config update helpers, session-store maintenance layer, and public docs for `openclaw models` and `openclaw sessions`.
