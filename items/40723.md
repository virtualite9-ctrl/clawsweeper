## Summary

Keep open. Current `main` still does not implement the PR's core behavior: the Agents overview has no separate global `Active Primary Model` control, and the existing overview model dropdown still writes per-agent `agents.list[].model` state rather than `agents.defaults.model`. The fallback-preservation helper/tests described in the PR are also not present on `main`.
