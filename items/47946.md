## Summary

Keep open. Current main has only a partial vLLM hardening: it avoids sending `tool_choice` unless explicitly requested, but it still sends `tools` on OpenAI-completions requests, and the vLLM docs still do not explain the tool-calling startup requirements implicated by the report. That is not strong enough evidence that issue #47946 is fixed or obsolete.
