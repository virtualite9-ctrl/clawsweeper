## Summary

Keep open. Current `main` already documents Telegram multi-account bindings and the relevant Telegram group gotchas, but it does not yet include the dedicated same-group multi-agent walkthrough this PR proposes. The PR also still has an unresolved review concern about scoping the example to a single Telegram group instead of implicitly opening every group.
