## Summary

Keep open. Current `main` still does not implement the core behaviors in PR #49145: Signal inbound quote ids are not threaded into `ReplyToId`, outbound Signal sends do not accept/pass quote metadata, and reaction-only Signal events enqueue a system event without an immediate heartbeat wake. The PR also remains active evidence-wise: the provided timeline shows review-driven fixes through the April 22, 2026 force-push, so this is not a stale or incoherent cleanup candidate.
