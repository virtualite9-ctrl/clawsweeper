## Summary

Keep open. Current `main` does not already implement this PR’s Signal work: reply/quote context is still not threaded through inbound or outbound Signal paths, reaction-only events still do not request an immediate heartbeat wake, and the lightweight quote helper/export surfaces from the PR are absent.
