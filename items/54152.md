## Summary

Keep open. Current `main` has Xiaomi only as a bundled model provider; there is no Xiaomi `web_search` provider contract, runtime registration, plugin config surface, or docs showing the requested MiMo web-search integration.

## Evidence

- **Xiaomi manifest has no web-search contract or config:** The bundled Xiaomi plugin manifest declares only `providers: ["xiaomi"]` and an empty `configSchema`; unlike plugins that ship search support, it has no `contracts.webSearchProviders` entry and no `webSearch` config block.
  - file: [extensions/xiaomi/openclaw.plugin.json:2](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/xiaomi/openclaw.plugin.json#L2)
  - command: `nl -ba extensions/xiaomi/openclaw.plugin.json | sed -n '1,120p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Xiaomi plugin entry registers only a model provider:** The Xiaomi plugin entry is built with `defineSingleProviderPluginEntry(...)` and only wires auth, catalog, usage auth, and usage snapshot for provider `xiaomi`; there is no web-search provider registration path here.
  - file: [extensions/xiaomi/index.ts:8](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/xiaomi/index.ts#L8)
  - command: `nl -ba extensions/xiaomi/index.ts | sed -n '1,200p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Xiaomi docs cover models only:** The Xiaomi provider doc describes the OpenAI-compatible model endpoint, onboarding, and built-in model catalog, but does not document any bundled `web_search` capability or Xiaomi search configuration.
  - file: [docs/providers/xiaomi.md:9](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/docs/providers/xiaomi.md#L9)
  - command: `nl -ba docs/providers/xiaomi.md | sed -n '1,170p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Supported web-search providers list excludes Xiaomi:** The web tools docs enumerate current bundled `web_search` providers (Brave, DuckDuckGo, Exa, Firecrawl, Gemini, Grok, Kimi, MiniMax, Ollama, Perplexity, SearXNG, Tavily); Xiaomi is not listed. For contrast, the MiniMax provider doc explicitly calls out bundled `web_search` support when a provider has it.
