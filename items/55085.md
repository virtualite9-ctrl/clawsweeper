## Summary

Keep open. Current `main` does not include the proposed `openclaw configure surface` export or an equivalent stable machine-readable setup-surface CLI. The only shipped surfaces I found are the interactive `configure` wizard, plugin inventory JSON, and onboarding JSON summaries, which do not replace this PR’s read-only provider/channel setup export.

## Evidence

- **`configure` still has no setup-surface subcommand:** Current `registerConfigureCommand` only wires `openclaw configure` with repeatable `--section` and runs the interactive wizard action. There is no `surface` subcommand registration or handler on `main`.
