## Summary

Keep open. Current `main` still routes Discord inbound replies through `deliverDiscordReply`, and that function sends directly to Discord without any `message_sending` hook invocation. The shared outbound path and Telegram inbound path do run the hook, so the inconsistency described in #54220 still appears real on `8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf`.
