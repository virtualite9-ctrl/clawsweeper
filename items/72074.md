## Summary

Keep #72074 open. Current main does not implement a `labelTemplate` or `{agentId}` / `{taskPreview}` fallback for `sessions_spawn`; it only forwards an explicitly provided `label`. The linked PR #72128 is an active, open implementation path, but it is unmerged, so the issue remains useful tracking until that PR or an equivalent fix lands.

## Best Possible Solution

Keep #72074 open and use PR #72128, or an equivalent maintainer-approved patch, as the implementation path. The fix should add `labelTemplate` to defaults and per-agent config schemas/types/generated metadata, apply it only when `sessions_spawn.label` is omitted, preserve explicit-label precedence, cover per-agent-over-default precedence plus `{agentId}` and sanitized/truncated `{taskPreview}` tests, and update public subagent/config docs.
