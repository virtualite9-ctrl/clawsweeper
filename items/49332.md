## Summary

Keep open. Current `main` at `f3c37a946cc49d26e5fe65a1043f7286a86aa4a4` does not contain an idempotent fix for duplicate `final` chat events, and this PR is not a conservative cleanup candidate: it bundles large unrelated changes and still has unresolved review concerns on the proposed UI dedupe logic.
