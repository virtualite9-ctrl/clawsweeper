## Summary

Keep open: current `main` does not implement the requested ReAct/local-model tool-calling fallback. OpenClaw still handles incompatible local backends by disabling tool schemas entirely via `compat.supportsTools: false`, and the docs continue to recommend that workaround rather than any fallback execution path.
