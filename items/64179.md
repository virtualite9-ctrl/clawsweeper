Keep open. Current main still has English-only, unthrottled inline tool summaries and no `agents.defaults.toolSummaries` config surface. This PR is still a relevant implementation candidate, but it should not be closed as obsolete or implemented; it still needs maintainer review, especially around ACP summary throttling and redaction-sensitive rendering.

## Best Possible Solution

Keep this PR open as the implementation candidate. The best path is to rebase it onto current main, resolve the ACP throttle timing concern, preserve current English default behavior and redaction, then land it with aligned config types, strict Zod schema, generated schema, public docs, and focused Pi/ACP coverage for locale and minimum-interval behavior.
