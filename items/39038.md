Keep #39038 open. The best maintainer path is either a focused Windows 11 24H2 regression/smoke proving `openclaw node run --host 127.0.0.1 --port 18789` reaches Gateway connection with Node 24 and a non-ASCII profile path, or an explicit docs/product decision that native Windows plus `npm install -g openclaw --ignore-scripts` is unsupported and should use the installer, normal npm install, or WSL2.
