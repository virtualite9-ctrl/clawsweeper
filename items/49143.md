## Summary

Current `main` already contains the plugin-route registry fallback that addresses the reported Google Chat `/googlechat` 404/state-drift failure, and the latest shipped release `v2026.4.22` already includes that code plus regression coverage for stale/empty route registries and webhook POST dispatch.
