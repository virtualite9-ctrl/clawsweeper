## Summary

Keep open. Current main still ignores per-agent `agents.list[].reasoningDefault` in the Telegram and Feishu reasoning-preview paths, while the config schema and docs already expose that setting. The PR is a recent focused bug fix with targeted tests and active review discussion, not obsolete maintainer cleanup.

## Best Possible Solution

Keep the PR open for maintainer/channel-owner review. The best path is to merge or refine the focused change so Telegram and Feishu fall back to `agents.list[].reasoningDefault` while preserving explicit session overrides and Feishu's stream-only preview behavior; broader global or per-model reasoning defaults should stay in their own canonical issues.
