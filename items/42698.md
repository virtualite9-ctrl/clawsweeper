## Summary

Keep this PR open as useful bug-fix evidence. Current main has moved Bonjour/Ciao handling into the bundled Bonjour plugin and does suppress some Ciao cancellation/assertion paths, but it still misses the exact upstream `IPv4 address changed from undefined to defined` wording reported here. The submitted diff should not merge as-is because it targets removed `src/infra/bonjour-ciao.*` files; the useful fix needs to be ported to the current plugin implementation.

## Best Possible Solution

Keep this PR open, but treat it as a reference patch rather than a directly mergeable branch. Port the exact missing `IPv4 address changed from undefined to defined` handling into `extensions/bonjour/src/ciao.ts`, add focused coverage in `extensions/bonjour/src/ciao.test.ts` and the advertiser handler test if needed, then run the Bonjour/plugin test lane before closing the linked crash work as implemented.
