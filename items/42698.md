## Summary

Keep this PR open as useful bug-fix evidence, but do not merge it as-is. Current main moved Bonjour/CIAO handling into the bundled Bonjour plugin and still misses the exact upstream `IPv4 address changed from undefined to defined` assertion variant described here. The fix needs to be ported from obsolete `src/infra/bonjour-ciao.*` files into the current `extensions/bonjour` implementation with focused tests.

## Best Possible Solution

Keep this PR open as a reference patch. Port the exact missing `IPv4 address changed from undefined to defined` handling into `extensions/bonjour/src/ciao.ts`, add focused coverage in `extensions/bonjour/src/ciao.test.ts` and the advertiser handler test if needed, then run the Bonjour/plugin test lane before closing the linked crash work as implemented.
