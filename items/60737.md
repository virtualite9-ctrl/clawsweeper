## Summary

Keep open. Current `main` has related Telegram ACP primitives, but it does not implement a per-DM/per-chat `defaultAcp` config or automatic ACP session creation on first message in a new topic.

## Evidence

  - file: [src/config/types.telegram.ts:247](https://github.com/openclaw/openclaw/blob/6bc0dc8fb6714b4e858fb65f9f8051e6b64c8440/src/config/types.telegram.ts#L247)

  - file: [src/config/types.telegram.ts:33](https://github.com/openclaw/openclaw/blob/6bc0dc8fb6714b4e858fb65f9f8051e6b64c8440/src/config/types.telegram.ts#L33)
