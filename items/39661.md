## Summary

Keep open. Current `main` still models Discord `requireMention` only at guild/channel scope, while sender-specific overrides exist for tools and allowlists, not mention policy. The issue is a coherent, still-current feature request and is not old enough for `stale_insufficient_info`.

## Evidence
