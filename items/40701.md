## Summary

Current `main` already implements this Feishu tool-routing fix, with a stricter guard than the PR: tool account resolution prefers the session/context account only when it matches a configured and enabled Feishu account, otherwise it falls back to `channels.feishu.defaultAccount`. The corresponding regression tests are also present on `main`, and the same implementation is already present in release `v2026.4.22`.

## Evidence
