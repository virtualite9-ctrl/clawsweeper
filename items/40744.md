## Summary

Current `main` does not already implement this PR's malformed `required` cleanup. The top-level early strip described in the PR is absent, and the Gemini cleaner on `main` only drops empty `required` arrays, not `required: null` or other non-array values. The current tests cover `required: []` and valid arrays, but not malformed non-array `required`, so there is no strong basis to close this as already implemented or obsolete.

## Evidence
