## Summary

Keep open. Current `main` still parses `offloadedRefs` for node/iOS-share `agent.request` attachments but never persists them into transcript media fields, while the agent transcript writer still records only plain user text. The equivalent `chat.send` path does persist offloaded media and has regression coverage, which reinforces that the node path gap described in #60339 is still real.

## Evidence
