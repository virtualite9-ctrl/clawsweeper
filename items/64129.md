## Summary

Keep open. Current `main` still has the core behavior described: Gemini API-key onboarding auto-selects the Google default model, and heartbeat still defaults to main-session turns with transcript reuse unless `isolatedSession` is explicitly enabled. I found no current-main evidence of a guardrail, confirmation step, or shipped fix for the paid-background-usage path described in #64129.
