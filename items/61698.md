## Summary

Keep open. Current `main` no longer has the exact stale menubar-only sentence, but `AGENTS.md` still does not include the daemon-install/onboarding guidance this PR proposes. The daemon path exists and is documented elsewhere on `main`, so this PR is still a real docs-alignment change rather than something already implemented in the contributor guide.
