## Summary

Keep this issue open. Current main implements global `agents.defaults.thinkingDefault`, but the issue body also requests global `agents.defaults.reasoningDefault`, and current main still only supports `reasoningDefault` on individual `agents.list[]` entries. The open #39115 PR is the best implementation candidate for the remaining global-default work; #71817 is related but only covers per-agent reasoning defaults for Telegram/Feishu.

## Best Possible Solution

Keep this issue open until maintainers either land a complete global `agents.defaults.reasoningDefault` implementation, likely by reviewing and landing #39115, or explicitly decide that reasoning visibility should remain per-agent/session-only. Treat #71817 as useful related per-agent channel work, not as the complete fix for this issue.
