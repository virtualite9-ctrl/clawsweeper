## Summary

Current main already handles the post-update plugin drift this PR targeted through supported update flows: `openclaw update` now performs post-update plugin sync/update, installer upgrades run `openclaw plugins update --all`, and the remaining LCM-specific checks are extension-specific rather than a core script fit.
