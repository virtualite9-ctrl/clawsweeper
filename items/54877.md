## Summary

Current main still has the reported doctor/cleanup mismatch: doctor suggests a missing-transcript cleanup preview without `--fix-missing`, while `sessions cleanup --dry-run` only counts missing transcript pruning when that flag is set. The prior linked fix PRs in the provided context are closed unmerged, so this issue should remain open.

## Best Possible Solution

Keep this issue open and land a narrow commands fix: update `src/commands/doctor-state-integrity.ts` so the missing-transcript `Preview cleanup impact` command includes `--dry-run --fix-missing`, then strengthen `src/commands/doctor-state-integrity.test.ts` to assert that exact preview hint. The existing explicit `sessions cleanup --fix-missing` behavior can remain unchanged unless maintainers want a broader CLI behavior change.
