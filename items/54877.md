## Summary

Keep open. Current main still has the reported mismatch: doctor detects missing recent session transcripts but suggests a dry-run cleanup command without `--fix-missing`, while `sessions cleanup --dry-run` only counts missing transcript pruning when `fixMissing` is true.

## Best Possible Solution

Land a narrow commands fix in `src/commands/doctor-state-integrity.ts` so the missing-transcript `Preview cleanup impact` command includes `--dry-run --fix-missing`, then update `src/commands/doctor-state-integrity.test.ts` to assert that exact hint. Keep `sessions cleanup --fix-missing` flag-gated unless maintainers intentionally choose a broader cleanup behavior change.
