## Summary

Keep open. Current `main` has partial live agent-event plumbing and chat tool streaming, but it does not ship the requested real-time activity dashboard (execution tree, timeline, metrics) in the Control UI. The issue is also actively tied to open draft PR #60814, so conservative maintainer cleanup should not close it.

## Evidence
