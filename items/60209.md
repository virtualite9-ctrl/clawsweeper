## Summary

Current `main` explicitly wires `tool_result_persist` and `after_tool_call` into embedded Pi runs, with regression tests added in commit `51c11cfd903033947f01f96a7169ebf92d194064`. The report appears valid for `2026.3.28`, but the behavior is now implemented on `main` and not yet in a tagged release.

## Evidence
