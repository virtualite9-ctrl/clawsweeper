## Summary

Keep open. Current main has related protections for commentary-phase assistant text and inter-session route preservation, but it does not implement this PR's untagged tool-use suppression or session-local `sessions_send` behavior. The PR is also not merge-ready as-is because the discussion includes credible route-persistence and sanitizer review blockers.

## Best Possible Solution

Keep #70344 open, but request rework or split. The best path is a narrow tool-use prelude suppression fix that does not rely on broad post-hoc monologue stripping, plus a `sessions_send` redesign that prevents the current inter-session turn from externally delivering while preserving persisted `lastChannel`/`lastTo` for later legitimate deliveries. Coordinate the narration work with #70372 and treat #62030 as the already-shipped commentary-phase baseline, not a full replacement for this PR.
