## Summary

Current `main` already implements the WhatsApp outbound-media path this report describes: media inputs are normalized across `mediaUrl`/`mediaUrls`, the primary media source is loaded before delivery, and the changelog records a shipped WhatsApp gateway-media fix in `v2026.4.12` for silently dropped attachments.
