## Summary

Keep open. Current `main` still hard-codes `cacheRead: 0` and `cacheWrite: 0` for both the Volcengine/Doubao and BytePlus catalogs, and OpenClaw's cost estimator charges cached tokens directly from those fields. On current `main`, cached usage would still be priced as free when those usage counters are present.

## Evidence
