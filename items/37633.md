## Summary

Current main already handles Feishu/Lark SecretRef-backed app credentials on read-only plugin/status paths without throwing. The Feishu plugin now uses inspect-mode resolution for status/account inspection, regression tests cover both env-backed and file-backed SecretRefs, and the changelog records the shipped fix in 2026.4.15.
