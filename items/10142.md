## Summary

Keep open. Current main still does not implement or document the requested internal `session:end` hook event for HOOK.md-style handlers. The repo has adjacent plugin lifecycle hooks (`session_end`, `agent_end`, `subagent_ended`) and an internal `session:patch` event, but no current-main `session:end` internal event or completion-path emitter.

## Best Possible Solution

Keep #10142 open until maintainers either implement the internal `session:end` contract or explicitly document that plugin hooks are the supported replacement. A complete fix should add the internal event context/type guard, emit `createInternalHookEvent("session", "end", ...)` from relevant completion paths, add regression tests, and update https://docs.openclaw.ai/automation/hooks.
