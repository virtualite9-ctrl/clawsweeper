## Summary

Keep #10142 open. Current main at 7e13f3f5140168e31027b39a9855ade3fadf24d5 still does not implement or document the requested internal `session:end` HOOK.md event. The repo has adjacent typed plugin `session_end` behavior, but docs distinguish plugin hooks from internal hooks, and exact searches found no internal `session:end` emitter, type guard, or docs table entry. Related implementation PR #37103 is closed/unmerged, while #56672 and #61675 remain open/adjacent rather than shipped replacements.

## Best Possible Solution

Keep #10142 open until maintainers either implement the internal HOOK.md `session:end` contract or explicitly document typed plugin `session_end` as the supported replacement. A complete core implementation should add a typed `SessionEndHookEvent`/guard, emit `createInternalHookEvent("session", "end", ...)` from the relevant gateway, cron, and subagent completion paths, add regression tests, and update https://docs.openclaw.ai/automation/hooks.
