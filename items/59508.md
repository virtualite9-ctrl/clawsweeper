## Summary

Current `main` implements the `--dangerously-force-unsafe-install` override for plugin installs, including local-link probes, and adds regression coverage for the exact bad fallback path reported here. The fix is documented and shipped in `v2026.4.23`.

## Evidence

- **Built-in scanner now honors forced-unsafe override:** `buildBlockedScanResult` returns `undefined` instead of blocking when `dangerouslyForceUnsafeInstall` is set for built-in `critical` findings, and the runtime logs the explicit forced-install warning.
