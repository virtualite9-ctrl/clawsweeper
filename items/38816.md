## Summary

Current main has partial image-model fallback work, but it does not clearly fix #38816. Unprefixed image model refs still resolve through a default provider, and the exact prior fix PR was closed unmerged. A newer maintainer draft PR (#71458) is related, but it is open and does not make this issue safe to close yet.
