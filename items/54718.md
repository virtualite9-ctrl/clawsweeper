## Summary

Keep open. Current `main` does not implement this PR’s same-host loopback trusted-proxy fallback; it explicitly rejects loopback trusted-proxy sources, tests that rejection, and documents token/password as the supported same-host alternative. That makes this a live behavior/product-security decision, not a safe auto-close.
