## Summary

Keep open. Current `main` still blocks sandbox Docker `network: "host"` in runtime validation and config schema, documents host mode as blocked, and has no shipped `dangerouslyAllowHostNetwork` config surface. The PR is proposing new behavior for a real gap, not duplicating something already implemented or obsolete.
