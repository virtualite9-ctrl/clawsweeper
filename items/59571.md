## Summary

Keep open. Current `main` clearly registers `feishu_doc`/`feishu_wiki` and ships Feishu skills, so the report is not a simple 'tool missing on main' close. But the issue's actual symptom is Feishu IM chat wiki-link recognition, and the repo evidence I found only proves tool registration plus document-comment link handling, not that normal Feishu chat messages now recognize and use wiki/doc links correctly.
