## Summary

Keep open. Current `main` still treats MiniMax M2.5/M2.7 chat models as text-only for image understanding, routes MiniMax image-tool pairing to `MiniMax-VL-01`, and only enables the MiniMax VLM fallback for `MiniMax-VL-01`. The issue body's claim that no MiniMax media-understanding provider is registered is outdated, but the underlying bug remains plausible and is supported by current code and tests.
