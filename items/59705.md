## Summary

Current `main` already picked up much of this draft's Windows smoke logging hardening, especially for the npm-update wrapper and the standalone onboard flow, but the PR's gateway-helper logging/retry work is still missing on `main`. That is not strong enough evidence for a conservative close-as-implemented decision.
