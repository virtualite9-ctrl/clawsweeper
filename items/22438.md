## Summary

Keep open. Current main no longer has the exact old binary split described in the issue because subagent/cron bootstrap now includes persona files, and lightweight context exists for subagents/cron/heartbeat. However, main still does not implement the requested configurable `agents.defaults.bootstrapTier` (`minimal`/`standard`/`full`) surface, and the linked implementation PR #22439 remains open/unmerged in the provided GitHub context.

## Best Possible Solution

Keep #22438 open until maintainers either land a schema-backed `agents.defaults.bootstrapTier` design or explicitly decide that the existing fixed subagent/cron allowlist plus `lightContext` is the intended product direction. If implemented, the work likely belongs across `src/agents/workspace.ts`, `src/agents/bootstrap-files.ts`, `src/config/*agent-defaults*`, generated schema/docs, and focused workspace/bootstrap tests.
