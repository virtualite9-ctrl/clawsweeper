## Summary

Keep #22438 open. Current main has partial related context controls, but it still does not implement the requested schema-backed `agents.defaults.bootstrapTier` option or a `minimal | standard | full` tier resolver. The linked successor PR #22439 remains the active implementation path rather than shipped behavior.

## Best Possible Solution

Keep #22438 open until #22439 or a successor merges, or until maintainers explicitly choose the current fixed allowlist plus per-run `lightContext` as the intended product direction. A complete fix should thread `agents.defaults.bootstrapTier` through `src/agents/workspace.ts`, `src/agents/bootstrap-files.ts`, config schema/types/help/labels/generated schema, public config docs, bootstrap hook behavior, and focused tests.
