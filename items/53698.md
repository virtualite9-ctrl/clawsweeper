## Summary

Keep open. Current `main` still hardcodes the WhatsApp web watchdog idle timeout to 30 minutes with a 60 second check interval, and there is still no config surface that lets operators set `messageTimeoutMs` or `watchdogCheckMs` for real gateway runs.
