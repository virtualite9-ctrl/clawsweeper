## Summary

Keep open. Current `main` does not implement this Slack history-retention change: handled room messages still clear pending history, the Slack test suite still asserts that prior replied channel messages are not injected into `Body`, and the docs describe group history as pending-only. The PR therefore represents a real behavior change that is not already on `main` and still needs maintainer product/behavior judgment rather than conservative auto-close.
