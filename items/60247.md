## Summary

Keep open. Current main’s Control UI path is a Gateway WebSocket text-frame JSON transport, so the issue’s suspected UTF-8 chunk-splitting root cause does not match the code/docs I inspected. But I did not find a targeted CJK/webchat regression test or a runnable repro in this read-only environment, so the evidence is not strong enough for a conservative close.
