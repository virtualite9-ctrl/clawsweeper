## Summary

Keep this issue open. Current main still routes Control UI chat and reasoning text through the shared sanitized markdown renderer without a MathJax/KaTeX/math layer, and the related open PR #67103 is only an implementation candidate. The earlier PR #61510 was closed unmerged, so it did not ship the requested behavior.

## Best Possible Solution

Keep this issue open as the canonical user request. Review and land #67103, or a similar focused Control UI change, by adding sanitizer-safe KaTeX or MathJax support in ui/src/ui/markdown.ts with regression coverage in ui/src/ui/markdown.test.ts, then close this issue after the implementation merges to main.
