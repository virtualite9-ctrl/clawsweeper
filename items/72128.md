## Summary

Keep this PR open. Current main still lacks configurable `sessions_spawn` label templates, and the linked request #72074 was closed after being superseded by this active implementation PR rather than shipped on main. The patch is relevant and already incorporated the Greptile `replaceAll` feedback, but it still needs maintainer/contributor revision for resolved target-agent semantics, requester-scoped config lookup, tests, docs, and generated schema follow-through.

## Best Possible Solution

Keep this PR open for maintainer review or contributor revision.
