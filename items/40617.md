## Summary

Current main already provides an official bundled `qwen` provider for Alibaba DashScope/Qwen Cloud, with documented China/global API-key onboarding and shared provider-auth resolution used by tool/runtime paths. The old `auth.profiles.dashscope` workaround described in the March 2026 report is obsolete on current main.
