## Summary

Keep open. Current main still resolves sandbox Docker pidsLimit only from agent/global config, with no fallback default, so the reported fork-bomb hardening gap remains. The open related PR #44098 tracks the same fix but has not landed, and the closed related PRs are not merged into main.
