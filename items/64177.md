## Summary

On current `main`, the built-in TTS tool already receives runtime config on every shipped `createOpenClawTools(...)` call path. The PR's `resolvedConfig` swap only changes the `__testing.setDepsForTest(...)` injected-deps seam, so I could not verify a user-facing bug to merge here.
