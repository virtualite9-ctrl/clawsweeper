## Summary

Current main already routes the Sessions view through the full gateway session list, and that list includes subagent rows plus parent/child metadata. The UI does not apply any subagent-hiding filter before rendering. The same implementation is present in release `v2026.4.22`, with tests covering subagent session rows, labels, and navigation.

## Evidence

- **Gateway sessions.list returns the combined session store through listSessionsFromStore:** `sessions.list` loads the combined gateway session store and responds with `listSessionsFromStore({ ..., opts: p })`, so the Sessions view is backed by the general session listing path rather than a main-session-only query.
  - file: [src/gateway/server-methods/sessions.ts:579](https://github.com/openclaw/openclaw/blob/cb2c36b04975e34ab7014a11575d6589aa96ae32/src/gateway/server-methods/sessions.ts#L579)
  - sha: [cb2c36b04975](https://github.com/openclaw/openclaw/commit/cb2c36b04975e34ab7014a11575d6589aa96ae32)
- **Session listing includes subagent rows by default:** `listSessionsFromStore` iterates all store entries, only excluding cron/global/unknown based on explicit flags; it does not exclude `agent:*:subagent:*` keys. It also supports `spawnedBy`/`parentSessionKey` relationships for child sessions.
  - file: [src/gateway/session-utils.ts:1461](https://github.com/openclaw/openclaw/blob/cb2c36b04975e34ab7014a11575d6589aa96ae32/src/gateway/session-utils.ts#L1461)
  - sha: [cb2c36b04975](https://github.com/openclaw/openclaw/commit/cb2c36b04975e34ab7014a11575d6589aa96ae32)
- **Gateway session rows preserve subagent metadata:** `buildGatewaySessionRow` returns `spawnedBy`, `parentSessionKey`, `childSessions`, `subagentRole`, and `subagentControlScope`, which is the metadata needed to surface subagent sessions in UI consumers.
  - file: [src/gateway/session-utils.ts:1381](https://github.com/openclaw/openclaw/blob/cb2c36b04975e34ab7014a11575d6589aa96ae32/src/gateway/session-utils.ts#L1381)

  - file: [ui/src/ui/controllers/sessions.ts:160](https://github.com/openclaw/openclaw/blob/cb2c36b04975e34ab7014a11575d6589aa96ae32/ui/src/ui/controllers/sessions.ts#L160)
