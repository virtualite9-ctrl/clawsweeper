## Summary

Keep open: current `main` still routes Z.AI image analysis through `glm-4.6v`, still hard-caps `image` tool requests at 30 seconds with no tool-level timeout override, and still surfaces aborted image requests as `Request was aborted`. I found no change in the implicated code paths between `v2026.4.23` and `HEAD`.
