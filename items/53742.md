## Summary

Keep open. Current `main` fixes the token-drift loop for `gateway.auth.token`, but the macOS LaunchAgent install path still reads durable `.env`/config env values into the gateway service environment and serializes that environment directly into the LaunchAgent plist, so the plaintext-projection part of #53742 is not implemented on `main`.

## Evidence
