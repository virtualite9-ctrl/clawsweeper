## Summary

Keep open. Current `main` still does not implement the PR’s core behavior: the reply path never auto-switches the main agent run to `agents.defaults.imageModel` when inbound images are present and the active reply model lacks vision support. The PR remains materially relevant, and the available evidence is not strong enough for any close reason.
