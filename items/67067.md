## Summary

Close as `clawhub`: the request is a useful reminder/scheduling feature, but current `main` scope keeps this kind of optional behavior in plugins/skills. Active Memory is intentionally bounded to pre-reply memory recall, while OpenClaw already provides plugin hooks plus cron/system-event scheduling and delivery primitives that a dedicated plugin can compose.
