## Summary

Keep #19680 open. Current main still has scope guidance in VISION.md, CONTRIBUTING.md, and the PR template, but it has not implemented the requested VISION.md restructure into a human/AI PR review decision framework. The provided issue context also includes a same-day steipete Codex review comment explicitly keeping this issue open, and I found no current-main change or canonical duplicate that supersedes it.

## Best Possible Solution

Keep the issue open until a maintainer either accepts and implements a docs-only VISION.md restructure or explicitly records that current guidance is sufficient. A good implementation would add a concise product thesis, three or four PR gut-check questions, and rationale-grounded examples for security, memory, lean core, plugins, and won't-merge categories so humans and AI review tools can reason from principles.
