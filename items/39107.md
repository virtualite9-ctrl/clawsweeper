## Summary

Current main already provides the requested completion access through supported surfaces: `openclaw infer model run` for scripts/CLI, a plugin SDK simple-completion runtime for Node/plugin code, and the Gateway OpenAI-compatible `/v1/chat/completions` endpoint for Node/Python/OpenAI-compatible clients. The prior related PR #39131 was useful context but did not merge; main now solves the same user problem through these broader maintained interfaces.
