## Summary

Keep #46810 open. Current main partially implements the Control UI exec approval modal, including pending-event routing, timeout text, security/ask metadata, and Allow once / Always allow / Deny buttons. The requested command syntax highlighting and requested-vs-effective command preview/diff are still missing from the Control UI path.

## Best Possible Solution

Keep #46810 open and narrow the implementation: preserve `commandPreview`/`allowedDecisions` in `ui/src/ui/controllers/exec-approval.ts`, render a highlighted shell command plus requested-vs-effective preview/diff in `ui/src/ui/views/exec-approval.ts`, and add focused Control UI tests for preview rendering, action visibility, and expiry/countdown behavior.
