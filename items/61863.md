## Summary

Current `main` already routes MCP/tool schemas through generic normalization paths instead of the old draft-07-style validation path, and it has regression coverage for object-ish, parameter-free, union, and `$defs`-based schemas. That makes the reported draft/2020-12 registration failure stale against current `main`.
