## Summary

Keep this PR open. Current main still expands sandbox tilde paths with `os.homedir()` directly, and `shortPath()` still uses the same direct home lookup. The PR is a narrow implementation candidate for a live sandbox/write-tool path bug, with useful reproduction detail and a follow-up commit that addressed the bot review concern.

## Best Possible Solution

Keep this PR open as a valid implementation candidate. The best maintainer path is to review it, ask for or add a focused regression test for sandbox tilde expansion and error abbreviation, and consider revising the implementation to reuse the existing OS-home helper. The broader `OPENCLAW_HOME` issue should remain separate because this PR targets OS-home tilde expansion in sandbox paths, not the full internal-state home override policy.
