## Summary

Current main still only partially satisfies #30389. WebChat now has a discoverable attachment button and session model picker, but the composer upload path remains image-only and the requested visible skill/capability picker is not implemented beyond slash-command and settings surfaces.

## Best Possible Solution

Keep #30389 open or split it into narrower maintainer-owned follow-ups. The shipped model picker and image attachment button should stay. Remaining work should add document/file upload end to end through the WebChat picker, previews/validation, `chat.send` serialization, Gateway parsing/extraction/offload, provider/tool handling, docs, and tests; separately define a composer-visible skill/capability picker that maps to skills, tools, agents, task profiles, or model profiles instead of relying only on typed slash commands.
