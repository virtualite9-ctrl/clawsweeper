## Summary

Keep PR #70589 open. Current `main` still has the reported local-agent JSON output bug: the `--local --json` delivery path serializes the result envelope through `runtime.log()`, and JSON mode routes console/log output to stderr. The PR’s proposed `writeRuntimeJson()` change is still the right code path, and the related PR #62453 was closed unmerged rather than superseding this work.

## Best Possible Solution

Keep #70589 open and land this fix, or an equivalent maintainer patch, by replacing the local JSON `runtime.log(JSON.stringify(...))` call in `src/agents/command/delivery.ts` with `writeRuntimeJson(runtime, buildOutboundResultEnvelope(...))` and adding a regression test that asserts `runtime.writeJson()` is used for `--local --json`.
