## Summary

Keep open. Current `main` still has bare `fetch()` calls in `src/agents/tools/pdf-native-providers.ts` with no local `ensureGlobalUndiciEnvProxyDispatcher()` bootstrap, and the latest release snapshot shows the same code. Similar direct-fetch codepaths in the repo explicitly add that bootstrap because `HTTP_PROXY`/`HTTPS_PROXY` would otherwise be ignored.
