## Summary

Current main already carries the WhatsApp outbound local-media-root fix: gateway send reconstructs outbound session context from `agentId`/`sessionKey`, shared outbound delivery resolves agent-scoped media access from that session, and the WhatsApp adapter forwards `mediaLocalRoots` into media loading. The changelog shows this shipped in `v2026.3.7`.
