## Summary

Current `main` does not include the basename fallback from PR #48021. `buildBootstrapInjectionStats` still returns `name: file.name` and `path: pathValue || file.name`, while downstream `/context` and bootstrap-size output render `f.name` directly. Hook-mutated bootstrap files remain a mutable array and are not sanitized for `name`, so the report this PR fixes is still plausible on `main`.
