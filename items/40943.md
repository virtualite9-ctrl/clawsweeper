## Summary

Current `main` already implements Feishu outbound image/file upload through the generic Feishu message action and reply delivery path. The work appears to have landed on `main` on 2026-04-24 and is documented as requiring OpenClaw 2026.4.24+, so it is likely newer than the latest tagged release `v2026.4.22`.

## Evidence

- **Feishu send action accepts media:** The Feishu channel action handler now accepts `send`/`thread-reply`, reads `params.media`, and routes media payloads into the outbound media sender instead of limiting sends to text/card content.
  - file: [extensions/feishu/src/channel.ts:685](https://github.com/openclaw/openclaw/blob/84fc00afdacbbb67e39fc1341242eedad09243df/extensions/feishu/src/channel.ts#L685)
  - sha: [84fc00afdacb](https://github.com/openclaw/openclaw/commit/84fc00afdacbbb67e39fc1341242eedad09243df)
- **Runtime uploads and sends files/images:** `sendMediaFeishu` uploads either an image or a file, derives the correct Feishu message type, and sends `image`, `file`, `audio`, or `media` messages as appropriate.
  - file: [extensions/feishu/src/media.ts:572](https://github.com/openclaw/openclaw/blob/84fc00afdacbbb67e39fc1341242eedad09243df/extensions/feishu/src/media.ts#L572)
  - sha: [84fc00afdacb](https://github.com/openclaw/openclaw/commit/84fc00afdacbbb67e39fc1341242eedad09243df)

  - file: [extensions/feishu/src/outbound.ts:230](https://github.com/openclaw/openclaw/blob/84fc00afdacbbb67e39fc1341242eedad09243df/extensions/feishu/src/outbound.ts#L230)

## Close Comment

Closing as implemented on `main` based on a Codex review. Feishu now accepts media on the shared message action path (`extensions/feishu/src/channel.ts:685`), routes it through outbound media delivery (`extensions/feishu/src/outbound.ts:230`), and the runtime uploads/sends native Feishu image/file/audio/video messages (`extensions/feishu/src/media.ts:572`). Coverage was added for media sends and reply attachment delivery (`extensions/feishu/src/channel.test.ts:432`, `extensions/feishu/src/reply-dispatcher.ts:346`). The Feishu docs also mark this support as requiring OpenClaw 2026.4.24+ (`docs/channels/feishu.md:19`), so this looks merged on `main` after the current `v2026.4.22` release.
