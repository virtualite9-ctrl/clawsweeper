## Summary

Keep open. Current `main` still documents unquoted bracket-notation config paths for `agents.list[...]`, and that syntax still trips shell globbing before OpenClaw runs in failglob/zsh-style shells. The CLI parser itself accepts bracket notation, so this remains a docs/usability bug rather than an implemented fix on `main`.
