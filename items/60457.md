## Summary

The reported scanner block appears obsolete on current `main`: the `env-harvesting` rule still requires same-file `process.env` access plus a network send, but current `@openclaw/nextcloud-talk` sources split those across different files and the plugin package version is now `2026.4.20`, not the reported `2026.3.13`.
