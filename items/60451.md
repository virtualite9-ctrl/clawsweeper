## Summary

Close as `clawhub`. The PR is a useful hook-based plugin idea, but current OpenClaw direction and docs say community plugins should live in their own repo and be published via ClawHub or npm rather than added to the core repository. The exact APIs this PR uses are already public on main, so no core change is needed to ship it externally.
