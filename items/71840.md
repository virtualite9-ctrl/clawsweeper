## Summary

Keep this PR open. Current main already has the intended runtime split between model-facing media hints and transcript-facing prompts, but it still lacks focused regression coverage for `REPLY_MEDIA_HINT` being excluded from `transcriptCommandBody` while staying in `prefixedCommandBody` and `queuedBody`. The PR is narrow, recent, test-only, and not superseded by merged #71229.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to merge the focused regression coverage, or request small test-shape adjustments, while keeping the shipped runtime behavior from #71229 intact.
