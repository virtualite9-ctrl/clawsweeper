## Summary

Keep open. Current `main` still emits the exact ambiguous doctor note in `src/commands/doctor-config-preflight.ts:95`, while unrecovered invalid config still blocks gateway startup via `src/gateway/server-startup-config.ts:198-213`. The latest release tag `v2026.4.22` also still contains the same doctor wording, so this has not been implemented on `main` or shipped in the latest release.
