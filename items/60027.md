## Summary

Keep PR #60027 open. Current main still has no `OPENCLAW_SKIP_MODEL_WARMUP` or equivalent model-warmup escape hatch: gateway sidecar startup still awaits `prewarmConfiguredPrimaryModel()` before channel startup unless the broader channel/provider skip flags are set. The PR remains useful, but it needs rebasing/adaptation because the current implementation lives in `src/gateway/server-startup-post-attach.ts` while `src/gateway/server-startup.ts` is now only a barrel export.

## Best Possible Solution

Keep the PR open and rebase it onto current main by adding the `OPENCLAW_SKIP_MODEL_WARMUP` check, operator log, and focused regression test around `prewarmConfiguredPrimaryModel` in `src/gateway/server-startup-post-attach.ts`; also register the env var in gateway test env snapshot helpers that isolate skip flags.
