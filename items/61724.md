## Summary

Keep open. Current `main` still exposes `streamTo` on the shared `sessions_spawn` schema, still rejects `runtime="subagent"` when that field is present, and still carries a test that locks that rejection in. The report's exact `implicitStreamToParent` root-cause theory is not confirmed from current TypeScript, but the user-visible failure mode has not been fixed on `main` or in the latest shipped release checked here.
