## Risks / Open Questions

- The provided PR context reports `mergeable: false`, so it likely needs a rebase or conflict resolution against current main before review/landing.
- The patch inserts the node-host check before provider and method selection, while `models auth login` can run OAuth, API-key, and custom methods. Maintainers should confirm the broader block is intentional or scope it to credential-writing methods that can corrupt gateway-owned auth state.
