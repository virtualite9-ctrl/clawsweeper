## Summary

Keep open. Current `main` does not implement this PR’s TUI feature set: there is no `/image` slash command, no `@path` image extraction module, and the TUI send path still forwards only plain text with no attachment payload. The lower layers do already support prompt image detection and Gateway chat attachments, so this remains useful additive TUI work rather than something already shipped or obsolete.
