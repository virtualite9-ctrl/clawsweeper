## Summary

Current `main` does not implement DeepInfra as a shipped provider, so there is no strong basis to close this PR. The feature area is still in scope for OpenClaw, but the PR targets an older architecture; current provider onboarding, auth/env wiring, and implicit discovery are plugin-owned under `extensions/*`, so this PR would need a substantive rewrite rather than a simple stale close.
