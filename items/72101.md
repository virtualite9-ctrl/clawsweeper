## Summary

Keep #72101 open. Current main at 631552c554095b0c520376db9131dac0f5427f94 still has the two behaviors the PR targets: `sessions_spawn` rejects `streamTo` for `runtime="subagent"`, and it forwards `model: "default"` as a literal model override. The linked issue #72078 being closed does not make this PR obsolete, and the overlapping #68397 is closed/unmerged and did not cover the `model="default"` normalization.

## Best Possible Solution

Keep #72101 open or fold its unique `model="default"` normalization into a consolidated `sessions_spawn` patch. The best maintainer path is one focused change that ignores ACP-only fields for native subagent spawns, preserves ACP behavior, normalizes empty/`default` model overrides to unset, and updates `sessions_spawn` regression tests for all covered parameters.
