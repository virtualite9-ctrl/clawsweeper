## Summary

Current `main` already handles non-monotonic assistant streaming by replacing earlier partial text with the corrected final text, and it now reliably emits terminal lifecycle/chat-final events. That covers both symptoms described in #61916: truncated streamed table rows and runs that failed to deliver the final UI refresh path.
