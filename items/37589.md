## Summary

Keep open. Current `main` still recreates every text part and the outer message object in `stripInlineDirectiveTagsFromMessageForDisplay`, so the identity-preserving improvement described in #37589 is not implemented on `55389f4be29359e2dd57ebaf548ba4a70331a445`.
