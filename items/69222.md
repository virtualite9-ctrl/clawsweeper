## Summary

Keep open. Current main only contains a narrower Ollama web-search endpoint change, not #69222's requested compatibility behavior: stable local endpoint, legacy experimental local fallback, env-key-only Ollama Cloud fallback, and source-scoped auth. The linked bug #69132 is still open in the provided context, and the related duplicate PR #69178 was closed unmerged rather than shipped.

## Best Possible Solution

Keep this PR or a successor open. The best maintainer path is to rebase onto current main, preserve main's `/api/web_search` and `https://ollama.com` baseUrl handling, add the missing legacy local endpoint and env-key-only cloud fallback, fix the mixed-auth review finding, and land regression tests/docs that cover #69132 before closing the bug.
