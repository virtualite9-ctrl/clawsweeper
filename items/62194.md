## Summary

PR #62194 is superseded by merged PR #62286, which fixed the same Windows bundled channel/Jiti raw drive-letter failure and shipped in v2026.4.7. Current main keeps the bundled entry loader on the shared plugin Jiti policy, disables native Jiti on Windows for bundled dist loads, and has focused regression coverage for the channel-entry sidecar seam. Linked issue #62083 is already closed as implemented.

## Best Possible Solution

Close #62194 as superseded by #62286. Keep the shipped shared loader policy and regression tests on main; track any future distinct Windows Jiti path failure as a new focused issue rather than reviving this PR.
