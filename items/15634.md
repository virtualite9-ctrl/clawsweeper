## Summary

Keep #15634 open. Current main has shipped substantial Mattermost parity work, including markdown-aware delivery, reactions, threading, media upload, typing indicators, and interactive buttons. The umbrella still includes read receipts / presence, and current Mattermost code/docs do not implement or document that remaining scope.

## Best Possible Solution

Keep #15634 open as the Mattermost parity umbrella for now, or split the unresolved read receipts / presence work into a narrower canonical Mattermost issue and close this umbrella only after that remaining scope is implemented, documented, or explicitly descoped by a maintainer. Keep the already-shipped #19957 rich-interaction/button work as canonical evidence for the completed portion.
