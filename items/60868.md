## Summary

Keep open. Current `main` already handles simple `Error.cause` chains through `formatErrorMessage`, so the PR's basic wrapped-cause example is partly obsolete, but the broader `.error`/deep-chain behavior described in the PR is still not on `main`, and the latest patch still appears to have an unresolved fallback-classification edge case.
