## Summary

Keep PR #72092 open. Current main still rejects Bedrock aws-sdk credential-chain auth in the media-understanding image, MiniMax fallback, and audio/video execution paths, while the docs and chat path already support no-static-key aws-sdk auth. The PR is a focused fix for open bug #72031 and its latest head also addresses the bot-reviewed empty-key rotation gap.

## Best Possible Solution

Keep PR #72092 open for maintainer review and CI. The best path is to validate or iterate on the focused implementation that carries aws-sdk missing-key allowance through image, MiniMax fallback, and audio/video execution, including the empty-key rotation contract, then merge it to close #72031.
