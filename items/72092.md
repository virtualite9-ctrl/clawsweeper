## Summary

Keep PR #72092 open. Current main still requires a static API key in the media-understanding image, MiniMax fallback, and audio/video paths, while Bedrock docs and chat/BTW paths already support aws-sdk credential-chain auth without a static key. The PR is a focused fix for linked bug #72031, and its latest head also addresses the reviewed empty-key rotation gap.

## Best Possible Solution

Keep PR #72092 open for maintainer review and CI. The best path is to land this PR or an equivalent focused fix that carries aws-sdk missing-key allowance through image, MiniMax fallback, and audio/video execution, including the empty-key rotation contract and regression tests, to resolve #72031.
