## Summary

Keep this PR open. Current main does not contain the reply-run watchdog or empty-final fallback proposed here, and the current auto-reply path can still return no payload when the run produces or filters down to no visible output. The PR is recent, contributor-authored, focused, and the provided discussion shows bot review feedback was followed by a second commit addressing the flagged watchdog/error-handling gaps.
