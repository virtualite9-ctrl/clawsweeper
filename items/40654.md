## Summary

Keep open. Current `main` does not implement this PR’s Discord `/acp close` archive behavior: the ACP close path still only closes the session and unbinds bindings, there is no current-main archive-after-reply outbound path, and the existing tests still assert only the old unbind behavior.

## Evidence
