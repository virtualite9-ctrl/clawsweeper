## Summary

Keep #12554 open. Current main has scoped exec/process supervision and individual background-process kill support, but it still does not provide the requested session-wide operator surface to list every running exec process for an OpenClaw session or terminate them as one unit. A maintainer/Codex review already kept this open on April 26, 2026, and the relevant files are unchanged between that reviewed SHA and current main.

## Best Possible Solution

Keep #12554 open and implement an additive Gateway/CLI/RPC operator surface: resolve the canonical session key, list managed exec runs for that session scope, terminate matching runs through process-supervisor cancellation plus process-tree fallback, return per-process status, document the contract, and cover same-session kill, unrelated-session isolation, background exec, HTTP/RPC/CLI validation, and subagent cancellation interactions in tests.
