## Summary

Current main does not implement WhatsApp message delete support. The shared message tool knows a generic `delete` action, but the WhatsApp plugin only advertises `react` and `poll`, only routes `react` through its action adapter, and throws for non-react WhatsApp actions. The docs likewise only list WhatsApp `react` plus `reactions`/`polls` gates, so the requested delete action remains meaningful work.
