## Summary

Keep open. Current `main` still uses glob-based skill watch targets with `chokidar`, so the reported nested-skill refresh bug is not implemented on `main`; the PR contains real fix work, but its latest revision still has an unresolved overlapping-watch-root correctness concern, so conservative cleanup should not auto-close it.

## Evidence

That means current `main` has not adopted the PR's root-watching approach.
