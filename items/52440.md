## Summary

Current main already defers gateway restarts while ACP background turns are active: ACP prompt turns are recorded in the shared task registry, both config-reload and scheduled SIGUSR1 restart paths count active tasks before restarting, and regression tests cover the ACP task-tracking and SIGUSR1 deferral behavior.
