## Summary

Keep open. Current `main` has optional backend-based sandboxing and exec approvals, but it does not implement this PR's macOS `sandbox-exec` wrapper or `sandboxProfile` exec parameter. The PR is also paired with the same author's open umbrella issue #58730, so it should not be auto-closed while that paired work remains unresolved.

## Best Possible Solution

Keep this PR open for maintainer review, but require a redesign before merge. If OpenClaw wants native macOS process isolation, it should be implemented as a proper sandbox backend or backend capability behind the existing `agents.defaults.sandbox` / `BashSandboxConfig.buildExecSpec` model, with docs and regression tests. Broader default-isolation and permission-model work should continue under #58730 or narrower maintainer-owned follow-ups.
