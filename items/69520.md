## Summary

Keep open. Current `main` still implements chat auto-follow using only the broad 450px `NEAR_BOTTOM_THRESHOLD` heuristic, with no explicit manual-scroll lock or wheel-intent handling, so the reported Control UI bug remains plausible and the PR is still actionable rather than closeable.
