## Summary

Keep open. Current main already has the directive-tag indentation and model-scan optional input pieces, but the remaining `tools.profile: "minimal"` regression from #42165 is still present, and the PR remains a useful implementation candidate after the author addressed the review feedback.

## Best Possible Solution

Keep the PR open, rebase it on current main, and narrow it to #42165.
