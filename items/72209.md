Keep this PR open. Current main still lacks the proposed passive recovery prompt discipline, Recovery Bundle / Task Ledger state, and checkpoint writes for update_plan, sessions_yield, and sessions_spawn. ## Best Possible Solution

Keep this PR open for maintainer review.
