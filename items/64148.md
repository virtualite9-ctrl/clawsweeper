## Summary

Keep this PR open. Current main still archives the whole state directory with no tar filter or extension-local node_modules exclusion, and the broader open backup exclude-pattern PR does not clearly supersede this default behavior. The PR remains a focused implementation candidate with prior review feedback addressed in its later commits.

## Best Possible Solution

Keep this PR open for maintainer review. Either land or recreate the narrow default exclusion with canonical state-path matching, Windows path normalization, and regression coverage, plus backup/restore docs if maintainers want this default; or explicitly fold the same `extensions/*/node_modules` behavior into #44288 before closing this PR.
