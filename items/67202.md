## Summary

Keep this PR open. Current main still lacks the proposed post-write verification on the sandbox write adapter, sandbox edit adapter, and host writes when `workspaceOnly` is false. The linked bug #67136 is still open, and this PR is a focused implementation candidate rather than code that has already shipped or been clearly superseded.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to refresh it against current main, preserve `expandTildeToOsHome` in host writes, extract shared sandbox write verification, add sandbox edit-path regression tests, coordinate any lower-level byte-count hardening with #44662, and then run the changed agents lanes before landing if the behavior remains correct.
