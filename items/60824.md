## Summary

Keep open. Current `main` still accepts only the renamed `acp.stream` keys and the active legacy-config migration pipeline has no `acp.stream` migration, so this PR addresses a real gap. The branch appears stale against a refactor because the migration code now lives under `src/commands/doctor/shared/*`, but the requested behavior is not implemented on `main` yet.
