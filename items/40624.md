## Summary

Keep open. Current main has newer Discord gateway lifecycle recovery and separately landed the PR's Windows secrets follow-ups, but it does not implement the PR's core safeguard to clear Discord resume state and force a fresh identify after repeated pre-READY 1005/1006 close loops.
