## Summary

Keep open. Current `main` intentionally restores persisted per-session model overrides ahead of the configured default model, and there is no config/schema support for a `resumeStrategy`-style setting. The request is therefore not implemented, remains coherent, and is in scope for core session behavior.

## Evidence
