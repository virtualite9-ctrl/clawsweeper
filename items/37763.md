## Summary

Keep open. Current `main` still has the two gaps behind this report: the Slack draft-preview path does not pass agent identity into `sendMessageSlack`, and bare `agents.list[].identity.emoji` aliases like `stitch-1` are still not normalized to Slack `:emoji_name:` syntax. Normal non-preview reply delivery does pass identity, so the bug is only partially addressed, not implemented on main.
