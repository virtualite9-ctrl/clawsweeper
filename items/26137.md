## Summary

Keep #26137 open. Current main has partial supply-chain hardening, but the central OpenSSF proposal is still not implemented: no Scorecard workflow, Best Practices badge, SECURITY-INSIGHTS file, Dependency Review Action, OSV Scanner, Allstar/SLSA workflow, or full GitHub Actions SHA pinning. The existing maintainer comment on the issue also reached the same keep-open conclusion.

## Best Possible Solution

Keep #26137 open as the umbrella supply-chain hardening request, or split it into maintainer-owned work items for Scorecard, SECURITY-INSIGHTS.yml, Dependency Review Action, OSV Scanner, Best Practices badge/application, remaining GitHub Actions SHA pinning, Allstar/org policy, branch protection/signed-commit decisions, and SLSA/release attestations. Preserve the shipped SECURITY.md, Dependabot, CodeQL, zizmor, dependency audit, and npm provenance work as partial progress.
