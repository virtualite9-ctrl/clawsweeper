## Summary

Keep this issue open. Current main still does not define, validate, document, or consume a per-model `reasoningDefault` under `agents.defaults.models[...]`. Existing support is per-agent plus a catalog-derived on/off fallback, which does not provide the requested per-model visibility default or `stream` default.

## Best Possible Solution

Keep this issue open and implement a first-class per-model reasoning visibility default, likely on `AgentModelEntryConfig` / `agents.defaults.models[modelRef]`, with schema/help/docs updates and runtime precedence that resolves directive, session override, per-agent default, then selected-model default before catalog fallback. Treat #71817 as useful partial channel work, not as completion of this request.
