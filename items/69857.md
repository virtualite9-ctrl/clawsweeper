## Summary

Keep open. Current `main` still synthesizes `ollama-local` for explicit remote Ollama configs, so the PR is not implemented already; and the proposed locality gate appears directionally correct but still narrower than existing OpenClaw local-host handling, so conservative cleanup should leave this for maintainer review rather than auto-close it.
