## Summary

Keep open. Current `main` still contains the broken `QMD + bridge mode` example in `docs/plugins/memory-wiki.md`, and the surrounding docs plus plugin code consistently show `memory-wiki` config belongs under `plugins.entries["memory-wiki"].config`. This PR addresses a real docs bug and is not already implemented on `main`.
