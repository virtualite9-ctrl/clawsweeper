## Summary

Keep open. Current `main` does not contain first-class DeepInfra support: there is no bundled `extensions/deepinfra` plugin, no DeepInfra provider doc, no DeepInfra onboarding/auth-choice wiring, and the docs still point users to the generic custom OpenAI-compatible provider path instead. This request is also in scope for core, not ClawHub, because `VISION.md` lists support for major model providers as a repo priority.
