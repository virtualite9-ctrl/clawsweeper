## Summary

Keep open. Current `main` does not implement the proposed `OPENCLAW_TUI_USER_BG` / `OPENCLAW_TUI_USER_TEXT` overrides, and the PR still appears to contain useful, unlanded work. The existing TUI theme code only supports `OPENCLAW_THEME`/`COLORFGBG`, while the docs and tests on `main` likewise have no user-message color override surface. There is also an unresolved readability risk in the PR design for background-only overrides because user messages always render both background and text colors together.
