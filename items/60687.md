## Summary

Current `main` no longer expects a `channelPlugin` export from bundled channel entrypoints. BlueBubbles, Discord, and Slack now ship bundled entry contracts, and the latest release `v2026.4.23` already contains those entry files, so this startup warning from `2026.4.3` is obsolete.

## Evidence

- **Loader now resolves bundled channel entry contracts:** `loadGeneratedBundledChannelEntry` resolves the module through `resolveChannelPluginModuleEntry(...)` and only warns if the bundled-channel-entry contract is missing; the old `missing channelPlugin export` path is gone.
