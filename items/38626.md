## Summary

Keep open. Current main has useful partial subagent/task observability, including active/recent listing, coarse lifecycle timestamps, completion announcements with transcript pointers, and kill/steer/cancel controls. It does not yet satisfy the issue’s broader acceptance criteria for a persisted per-subagent lifecycle timeline, tool start/end and warning/error event history, first-failure signature plus recommended next action, or idempotent/audit-logged cancel/steer/resume supervision controls. The related #62478 is complementary background-spawn work, not a canonical replacement for this observability/control issue.
