## Summary

Keep PR #39806 open. Current main still has the Debug Snapshots UX gap from #39777: disconnected refresh returns before setting any visible state, and the Snapshots card still only shows Status, Health, and Last heartbeat blocks while loading is limited to the Refresh button label. The PR targets a real remaining bug, though its patch still needs maintainer review because the supplied review context flags a Snapshots card layout regression risk.

## Best Possible Solution

Keep this PR open, or replace it with an equivalent focused fix that adds body-level loading and disconnected states to the Debug Snapshots card, passes connection state cleanly, preserves the existing Status/Health/Last heartbeat layout, and closes #39777 once landed.
