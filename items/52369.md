## Summary

Issue #52369 is best closed as `clawhub`. On current `main` at `8866544ffefb42ca234c66c337134fb60743d8d0`, the proposed ambient-attunement layer is not implemented in core, and the repo vision explicitly pushes optional coordination/memory-style capability out of core and into plugins/skills. The plugin SDK already exposes memory, context-engine, and lifecycle hooks that fit a prototype like this without adding a new core orchestration layer.
