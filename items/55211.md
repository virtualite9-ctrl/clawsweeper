## Summary

Keep open. Current `main` at `c070509b7f8fdbe479e5d829643782c272e30e83` still runs `triggerInternalHook` without any re-entrancy guard in `src/hooks/internal-hooks.ts:286`, so this PR is not implemented on `main`. The PR discussion is still relevant: review feedback caught the first global-Map approach as too broad for concurrent deliveries, and the author revised it to `AsyncLocalStorage`, which makes the item substantive rather than stale or incoherent.
