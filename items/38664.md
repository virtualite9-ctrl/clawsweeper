## Summary

Keep open. Current main does not contain this PR's proposed Trivy/KICS/TruffleHog workflow template or TruffleHog pre-commit guide, and the predecessor #8951 was closed unmerged because this PR replaced it. The PR still has substantive review concerns in the proposed templates, so it needs maintainer docs/security judgment rather than conservative cleanup.

## Best Possible Solution

Keep this PR open for maintainer docs/security judgment. If OpenClaw wants this scanner template, update the hook to stream staged paths without command substitution, pin every GitHub Action reference to an immutable SHA, add the new pages to docs navigation, and align the guidance with the existing security audit, detect-secrets, zizmor, and dependency-audit stack. If maintainers decide generic scanner templates are outside core docs scope, close manually with that product rationale.
