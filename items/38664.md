## Summary

Keep open. Current main still lacks this PR's proposed Trivy/KICS/TruffleHog workflow template and TruffleHog staged-content pre-commit guide, and no equivalent docs or workflow surface has shipped. The predecessor PR #8951 was closed unmerged because this PR replaced it. The PR also has unresolved template correctness/security review concerns, so there is no conservative cleanup close reason.

## Best Possible Solution

Keep this PR open for maintainer docs/security judgment. If OpenClaw wants this template, update the hook to stream NUL-delimited staged paths directly, pin every GitHub Action reference to immutable SHAs, add the new pages to docs navigation, and align the guidance with the existing detect-secrets, zizmor, dependency-audit, and openclaw security audit docs. If maintainers decide generic scanner templates are outside core docs scope, close manually with that product rationale.
