## Summary

Current `main` already supports the requested session reuse semantics: DMs share the main session by default, cross-channel sharing is configurable via `session.dmScope` and `session.identityLinks`, WebChat already points at the agent main session, and heartbeat/subagent session separation is intentional and documented.

## Evidence
