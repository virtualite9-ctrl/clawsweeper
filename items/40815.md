## Summary

Keep open. Current `main` still unconditionally appends restart-gateway guidance for local dispatcher-path failures in the browser client, and the attachOnly-specific timeout handling described by PR #40815 is not present. The relevant files are unchanged between current `main` (`b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503`) and the latest release `v2026.4.22`, so there is no evidence this was already implemented or shipped elsewhere on `main`.
