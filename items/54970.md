## Summary

Keep open. Current `main` still has the core bug on the surviving shell-execution path: `exec host=node` unconditionally invokes `system.run.prepare`, but the macOS companion node does not advertise or handle that command. The PR is partially stale because `nodes run` was removed from `main`, yet the remaining `exec host=node` fix is not implemented.
