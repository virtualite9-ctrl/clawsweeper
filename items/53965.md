## Summary

Keep open. Current `main` has partial hardening for config/JSON writes, but the active secrets write path this PR is trying to address still uses unguarded `chmodSync` calls, so the Windows-mounted Docker failure is not clearly fixed on `main` and this PR is not safely obsolete.
