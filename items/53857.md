## Summary

Keep open. Current `main` contains explicit env-fallback and bundled-provider resolution for `web_search`, and unit tests cover Brave/Perplexity env-key detection, but the issue has a recent follow-up report that it still persisted in `v2026.4.5`. I could not do a live repro in this read-only checkout because `pnpm` is unavailable here, so the evidence is not strong enough to close conservatively.
