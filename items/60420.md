## Summary

Keep open. Current `main` still only unbinds stale ACP thread bindings; it does not scrub persisted ACP session routing fields from the session store, so the cleanup proposed in PR #60420 is not implemented on `926068b14fc661e915a0247e9ecb5949fb95b4f`.
