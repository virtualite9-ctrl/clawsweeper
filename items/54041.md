## Summary

Keep open. Current `main` has partial Anthropic beta-header support (`anthropicBeta` plus `context1m`), but the requested Anthropic context-management support is still not implemented: `context_management` injection/config is wired only through the OpenAI Responses policy, and Anthropic tool serialization still hardcodes normal custom tools rather than allowing provider-defined tool types like `memory_20250818`.

## Evidence
