## Summary

Current main still does not implement the central Copilot replay sanitizer from this PR. The PR discussion is consistent with the source: #19459 previously added Copilot thinking-block stripping, #58916 later made the shared helper preserve the latest assistant turn for Anthropic-style replay, and main now sends GitHub Copilot Claude replay through that generic latest-turn-preserving helper.

## Best Possible Solution

Keep this PR open and ask the author to refresh it against current main: keep the Copilot-specific drop-all-plain-thinking replay sanitizer and regression tests, select it from a normalized provider/policy strategy, align context-pruning estimates with the same strategy, and split or justify the ACP/guard-script changes against the newer generic conversation-resolution path.
