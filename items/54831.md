## Summary

Keep open. Current `main` still has the original runtime-only failure path, and this PR's proposed remediation text points users to a non-existent CLI command. The underlying bug is not implemented on `main`, and the PR as filed still needs correction and tests before it would be a conservative maintainer close candidate.
