## Summary

Keep #69030 open. Current main partially mitigates the reported context leak by running `infer model run` with `modelRun: true`, `promptMode: "none"`, and tools disabled, but it still has no `--raw` flag or raw capability path and still routes through the agent runtime/session machinery. That leaves the requested stateless raw/vanilla inference mode unresolved.

## Best Possible Solution

Keep the issue open and implement a clearly named raw/stateless model inference path, either `--raw` on `openclaw infer model run` or a separate `model.raw` capability. It should define and test no agent prompt stack, no tools, no session transcript/write-lock dependency, provider/model attribution, local and gateway behavior, and docs at `docs.openclaw.ai/cli/infer`. The existing `promptMode: "none"` path should remain as the backwards-compatible one-shot agent-runtime mode.
