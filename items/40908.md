## Summary

Keep open. Current `main` does not implement Feishu Drive file upload, large-file multipart upload, or a Drive-root upload path; the only upload support in the Feishu plugin is still chat media upload capped at 30 MB and doc-scoped `feishu_doc.upload_file`.

## Evidence

- **`feishu_drive` has no upload action:** `FeishuDriveSchema` only defines `list`, `info`, `create_folder`, `move`, `delete`, `list_comments`, `list_comment_replies`, `add_comment`, and `reply_comment`.
  - file: [extensions/feishu/src/drive-schema.ts:22](https://github.com/openclaw/openclaw/blob/b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503/extensions/feishu/src/drive-schema.ts#L22)
  - command: `nl -ba extensions/feishu/src/drive-schema.ts | sed -n '1,220p'`
  - sha: [b0d2003fe65b](https://github.com/openclaw/openclaw/commit/b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503)
- **Registered drive tool still exposes no upload path:** The `feishu_drive` tool description and action switch on current `main` match the schema above and do not include `upload_file` or any multipart flow.
  - file: [extensions/feishu/src/drive.ts:758](https://github.com/openclaw/openclaw/blob/b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503/extensions/feishu/src/drive.ts#L758)
  - command: `nl -ba extensions/feishu/src/drive.ts | sed -n '720,840p'`
  - sha: [b0d2003fe65b](https://github.com/openclaw/openclaw/commit/b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503)
- **Existing file upload is document-scoped, not Drive-root:** `feishu_doc.upload_file` requires `doc_token`, and its implementation routes through `document_id`/`docToken` plus `parent_type: "docx_file"`, so it is for attaching a file to a document context rather than uploading to Drive root or an arbitrary Drive folder.
  - file: [extensions/feishu/src/doc-schema.ts:161](https://github.com/openclaw/openclaw/blob/b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503/extensions/feishu/src/doc-schema.ts#L161)
  - command: `nl -ba extensions/feishu/src/doc-schema.ts | sed -n '140,190p'; nl -ba extensions/feishu/src/docx.ts | sed -n '780,860p'`
  - sha: [b0d2003fe65b](https://github.com/openclaw/openclaw/commit/b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503)
- **Current Feishu media upload path is still 30 MB chat media:** `uploadFileFeishu` is documented in code as `Max file size: 30MB`, uses `client.im.file.create(...)`, and the Feishu docs page still documents `channels.feishu.mediaMaxMb` with a default of 30 MB.
  - file: [extensions/feishu/src/media.ts:363](https://github.com/openclaw/openclaw/blob/b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503/extensions/feishu/src/media.ts#L363)
