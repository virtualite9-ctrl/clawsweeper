## Summary

Keep #12601 open. Current main still does not expose Slack Canvas content reading: Slack action discovery, dispatch, runtime, setup scopes, docs, config gates, and tests only cover existing message/file/pin/member/emoji actions. The linked PR #18752 was closed unmerged, so it does not supersede the issue.

## Best Possible Solution

Keep #12601 open for the bundled Slack plugin. A complete fix should add a Canvas read action or dedicated Canvas tool under `extensions/slack`, expose it through Slack message action discovery and dispatch, add `canvases:read` to setup/docs/config as needed, call Slack's Canvas API through the existing account/token model, return bounded markdown or section data, and cover discovery, dispatch, runtime behavior, docs, and live Slack validation.
