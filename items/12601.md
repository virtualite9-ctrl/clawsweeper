## Summary

Keep #12601 open. Current main still does not expose Slack Canvas content reading in the bundled Slack plugin: action discovery, dispatch, runtime handling, file download behavior, setup scopes, docs, and tests all cover the existing Slack message/file/pin/reaction/member/emoji actions only. The related PR #18752 was closed unmerged, and a maintainer review comment already kept this open as Slack plugin work.

## Best Possible Solution

Keep #12601 open for the bundled Slack plugin. A complete fix should add a Slack Canvas read action or dedicated Canvas tool under extensions/slack, extend the shared message action contract if the message tool gets a new action, add canvases:read to setup/docs/config guidance, call Slack Canvas APIs through the existing account/token model, return bounded markdown or section data, and cover discovery, dispatch, runtime behavior, docs, tests, and live Slack validation. PR #18752 remains useful prior art but was closed unmerged.
