## Summary

Current main already implements the requested skill-first behavior: OpenClaw now injects a mandatory skills-selection section into the system prompt, provides the model with each eligible skill's description and location, and for `claude-cli` also exposes the same filtered skills through a temporary native plugin. This behavior is present in the shipped `v2026.4.23` release.

## Evidence
