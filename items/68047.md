## Summary

Keep this PR open. Current main still only applies the Claude-name image-support fallback to `claude-cli`, not direct `anthropic`; current tests do not cover the direct Anthropic stale-catalog or catalog-miss cases this PR adds. The PR is a narrow implementation candidate for the linked direct Anthropic image-support bug, with no visible security-sensitive changes in the provided diff.
