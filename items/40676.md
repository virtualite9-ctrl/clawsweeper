## Summary

Keep open. Current `main` at `b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503` does not contain the PR's two core fixes: voice auto-join is still only wired to `READY`, and the join path still does not clean up any pre-existing global `@discordjs/voice` connection before calling `joinVoiceChannel()`. The PR also picked up extra session-key behavior beyond the title, and the provided review history shows unresolved regression concerns, so there is no strong basis to close this as already implemented or obsolete.
