## Summary

Current `main` does not appear to contain this PR’s fix. Anthropic `refusal`/`sensitive` stop reasons are still collapsed to `error` in the transport, runner recovery still only normalizes unhandled-stop-reason text, and there is no Anthropic-only transcript rewind path for refused turns. The PR remains a plausible, scoped fix and should stay open for maintainer review rather than cleanup-closing.
