## Summary

Current `main` does not include this optimization. The vector fallback path in `extensions/memory-core/src/memory/manager-search.ts` still scores all candidates, sorts the full scored array, then slices, and the current test file only covers the sqlite-vec KNN path rather than the proposed fallback top-k behavior. This PR still represents unmerged work rather than something already implemented or obsolete.
