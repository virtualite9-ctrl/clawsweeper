## Summary

Current main already routes WhatsApp group gating through the configured `groupPolicy`, and the present implementation explicitly allows group traffic when `groupPolicy: "open"` unless a separate `groups` allowlist is configured. The same implementation is present in release `v2026.4.22`, with regression tests covering the relevant WhatsApp group-gating paths.
