## Summary

Keep #17213 open. Current main still has lifecycle/finalization plumbing and GPT-5 strict-agentic safeguards, but it does not implement the requested atomic run.begin -> run.exec -> mandatory run.commit/run.abort contract with structured abort blockers across agent run surfaces. The related #40710 work was closed unmerged per provided GitHub context, and current main does not contain its execution-intent-gate files or config surface.

## Best Possible Solution

Keep #17213 open as a core reliability/product-contract item. The best path is a maintainer-approved, backward-compatible run lifecycle contract that adds begin intent, execution evidence, mandatory commit or abort terminal state, and structured abort/blocker details across shared event types, agent emitters, gateway finalization, clients, docs, and regression tests.
