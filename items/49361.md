## Summary

Keep open. Current `main` still falls back from Google Chat account name to `agent.name` and then hardcoded `"OpenClaw"`; it does not check `agent.identity?.name`, so the PR’s requested behavior is not implemented on `main` or in the latest release snapshot inspected.
