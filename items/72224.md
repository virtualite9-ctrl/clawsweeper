## Summary

Keep open. Current main still lacks the PR's central non-systemd gateway restart fixes: Linux listener discovery depends on `lsof`, bare `openclaw-gateway` argv is not accepted, and unmanaged restart health checks do not reuse config-resolved probe auth. The PR is a focused, recent bug-fix for closed issue #72223 and still appears useful rather than obsolete or superseded.

## Best Possible Solution

Keep #72224 open for maintainer review. The best path is to land or adapt the PR's fixes for Unix listener discovery, bare `openclaw-gateway` argv verification, and config-resolved probe auth, with the included regression tests and a final non-systemd/container restart verification before merge.
