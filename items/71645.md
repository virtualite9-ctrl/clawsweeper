## Summary

Current main still has the Mattermost streaming/tool-status overwrite path described by PR #71645. The PR is a focused channel bug fix that has not been implemented, merged, superseded, or moved out of core/plugin ownership.

## Best Possible Solution

Keep PR #71645 open for maintainer review. The best path is to land a Mattermost-owned fix that preserves streamed assistant text before tool-start status updates, add a focused regression test for partial text followed by `onToolStart`, and update the Mattermost channel docs if the accepted UX is to move tool status into a separate post at that boundary.
