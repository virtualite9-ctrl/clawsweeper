## Summary

Current main has related Feishu per-sender session scoping and subagent delivery hooks, but it does not implement the requested automatic Feishu group-chat per-user subAgents with isolated skills/configuration/memory and admin hot configuration. The only dynamic agent creation path is explicitly DM-only, so a meaningful requested capability remains missing.
