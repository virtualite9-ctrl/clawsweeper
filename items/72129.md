## Summary

Keep this PR open. Current main still has Hugging Face as a chat/model-provider plugin only, while this PR is an active, focused implementation of Hugging Face text-to-image on the existing image-generation plugin seam. It is not obsolete, not implemented on main, and not a clean ClawHub auto-close because it targets an existing bundled model-provider plugin and needs normal maintainer product/code review.

## Best Possible Solution

Keep this PR open for maintainer review. If accepted, the best path is to land the focused Hugging Face text-to-image provider in the existing bundled Hugging Face plugin with the manifest contract, docs, tests, and changelog already proposed. If maintainers decide this capability should live outside the repository, close it only after an explicit product-scope decision and ideally point the contributor toward a ClawHub/plugin path.
