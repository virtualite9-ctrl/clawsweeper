## Summary

Keep #33329 open. Current main has provider-level discovery toggles for GitHub Copilot, Hugging Face, Ollama, and Bedrock, with tests for the non-Bedrock toggles. The issue is still not fully resolved because public docs still do not document the non-Bedrock toggle paths, and the later issue discussion requests a global disable-all model discovery control that current config/runtime do not provide.

## Best Possible Solution

Keep the issue open until public docs document the non-Bedrock provider toggles: `plugins.entries.github-copilot.config.discovery.enabled`, `plugins.entries.huggingface.config.discovery.enabled`, and `plugins.entries.ollama.config.discovery.enabled`. Then either add a typed global model-discovery disable control checked before `resolveImplicitProviders`, or split that global switch into a focused follow-up that defines its config path and interaction with `models.mode`.
