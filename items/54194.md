## Summary

Current `main` already ships MCP client support for community server integrations: OpenClaw stores outbound MCP server definitions, supports stdio plus remote HTTP transports, and materializes discovered MCP tools into normal agent tool profiles. The latest release `v2026.4.22` already contains this implementation.
