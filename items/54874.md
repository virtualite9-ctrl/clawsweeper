## Summary

Keep open. Current `main` still uses reactive `chatMessage` state and still rebuilds chat items during `renderChat`, while the linked March fixes were closed unmerged and a later rebased PR for the same bug is still open. I do not have strong evidence that `main` already fixed or obsoleted the reported keystroke-lag regression.
