## Summary

Keep open. Current `main` still sends a `model` field in the Azure image-generation request body even when the deployment name is already encoded into the Azure `/openai/deployments/{deployment}/images/...` URL, and I found no follow-up fix after `v2026.4.23` that would resolve the reporter's specific failure mode.
