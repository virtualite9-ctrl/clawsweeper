## Summary

Current main only partially addresses #47913. Moonshot/Kimi now has per-model `params.thinking` control, but MiniMax on the Anthropic-compatible path is still handled by forcing `thinking: { type: "disabled" }` as a safety workaround, and the remaining text sanitizer only strips tagged reasoning. That is not strong evidence that the reported MiniMax leak and requested per-model control are fully implemented on `main`.
