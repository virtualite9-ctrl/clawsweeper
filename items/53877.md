## Summary

Keep open. Current `main` does not contain a shipped fix for this report: the gateway lock path is still a direct TCP bind, and the same bind-based logic was already present in the reporter's `v2026.3.22` build. The report's proposed WebSocket self-probe root cause does not match the code, but without a WSL2 mirrored-networking repro environment there is not enough evidence to close this as obsolete or unreproducible.
