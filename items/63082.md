## Summary

Keep this PR open. Current main still hardcodes shell completion profile paths and lacks regression coverage for ZDOTDIR/XDG_CONFIG_HOME, so the PR addresses a real remaining bug and is paired with the same author's open bug report #63069.

## Best Possible Solution

Keep the PR open as the implementation candidate for #63069. The maintainer path is to review or update it against current main by moving the unified profile resolver into src/cli/completion-runtime.ts, respecting ZDOTDIR and XDG_CONFIG_HOME, using the same resolver for install/check/doctor messages, and adding focused tests for zsh, fish, and bash fallback behavior.
