## Summary

Keep open. Current `main` already sends Brave API keys via `X-Subscription-Token` for both Brave web and Brave llm-context requests, but `main` still does not contain the regression tests this PR adds. Because the proposed test coverage is not already implemented on `main`, this PR remains useful and there is not a strong basis to auto-close it.
