## Summary

Keep PR #38789 open. Current main still does not rebuild or remap the skills prompt for non-rw sandbox workspaces when a persisted skills snapshot exists, so the PR targets a real remaining agents bug. The PR should be updated before landing because review comments identify correctness gaps around empty sandbox skill sets, preserved skill filters, remote eligibility, and env override alignment.

## Best Possible Solution

Keep this PR open, or replace it with a focused agents fix that rebuilds sandbox prompts from the same sandbox-aware skill set used for runtime setup in both attempt and compaction paths. The fix should preserve the saved skill filter, remote eligibility note, and env overrides, return no host snapshot prompt when sandbox entries are empty, and add regression coverage for snapshot-present non-rw sandbox runs plus attempt/compaction parity.
