## Summary

Keep open. Current `main` has adjacent runtime restrictions for per-group and per-sender tool policy, but it does not implement the RFC's requested bridge from gateway operator scopes to agent tool access. I found no `toolsByScope` config surface, and the effective tool-policy resolver does not accept or merge gateway scopes.
