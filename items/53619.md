## Summary

Keep open. Current `main` has a persisted Control UI toggle for showing or hiding tool calls/results, plus per-message manual collapse, but it does not implement the requested tri-state Control UI setting (`collapsed` / `expanded` / `hidden`) or a persisted auto-expand preference.
