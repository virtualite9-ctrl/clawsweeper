## Summary

Keep #11655 open. Current main still lacks Slack App Home support: the bundled Slack plugin only enables the App Home messages tab, does not subscribe to `app_home_opened`, does not register a Home-tab monitor handler, and exposes no `views.publish` action or Home-tab config path. Related implementation PRs were closed unmerged, and duplicate #52020 was closed back to #11655 as the canonical tracker.

## Best Possible Solution

Keep #11655 open as the canonical Slack App Home tracker. Implement it in the bundled Slack plugin by adding `app_home_opened` to setup/docs, registering a monitor handler under `extensions/slack/src/monitor/events`, publishing a safe default Home view with Slack `views.publish`, and adding focused tests for event filtering, payload shape, error handling, and the chosen customization surface.
