## Summary

Keep #11655 open. Current main still lacks Slack App Home support: the bundled Slack plugin enables the App Home messages tab, but there is no `app_home_opened` event subscription, no monitor handler registration, no `views.publish` call path, and no Home-tab config/tool customization surface. Related implementation PRs were closed unmerged, while duplicate #52020 was closed back to #11655 as the canonical tracker.

## Best Possible Solution

Keep #11655 open as the canonical Slack App Home tracker. Implement it in `extensions/slack` by adding `app_home_opened` to setup/docs, registering a monitor handler under `extensions/slack/src/monitor/events`, publishing a safe default Home view through Slack `views.publish`, deciding the supported customization path, and adding focused tests for event filtering, payload shape, error handling, and config/tool behavior.
