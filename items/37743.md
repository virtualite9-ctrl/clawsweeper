## Summary

Issue #37743 targets the legacy Chrome extension relay path from OpenClaw 2026.2.25. Current `main` has removed that path, migrated browser attach flows to Chrome MCP `existing-session` / `user`, and documents the removal as shipped in v2026.3.22.
