## Summary

Current `main` contains a targeted agents/exec fix for late stdout/stderr `tool_execution_update` events after the agent run has ended or been aborted, plus regression tests and a shipped changelog entry in `v2026.4.23` that matches this issue's crash signature.

## Evidence

- **Issue symptom matches shipped fix:** The report describes `Agent listener invoked outside active run` triggered by exec stdout arriving after the turn ended, and the April 8 follow-up comment confirms the same stale-listener/Telegram wedge pattern outside the original reproduction.
