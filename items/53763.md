## Summary

Keep open. Current `main` already has a first-class bundled browser plugin and managed browser-control path, so part of the request is effectively present. But the exact ask to ship a built-in headless Chromium without extra browser installation or user Chrome is not fully implemented: current docs still rely on an installed Chromium-based browser or an explicit Playwright Chromium install, and the default Docker image explicitly does not include bundled browsers.
