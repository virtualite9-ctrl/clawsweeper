## Summary

Keep open. Current `main` at `f3c37a946cc49d26e5fe65a1043f7286a86aa4a4` does not include the `local-gguf` feature surface described in PR #49330. The repo still routes local open-weight inference through server-backed providers like LM Studio/Ollama, while `node-llama-cpp` GGUF support on `main` is limited to memory embeddings rather than a selectable chat/inference provider.
