## Summary

Keep open. Current `main` already has generated `zh-TW` key parity, but it does not clearly implement the Taiwan-standard terminology this PR is proposing, and the discussion points to overlap with another still-open PR rather than a merged replacement. That is not strong enough evidence for a conservative close under the allowed reasons.
