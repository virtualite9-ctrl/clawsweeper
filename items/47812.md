## Summary

Current `main` does not implement smart session-switch reminders or automatic topic-change-based session rollover. OpenClaw today supports manual `/new` or `/reset`, plus daily and idle expiry, and the web UI only shows a passive high-context warning banner.

## Evidence

- **Session lifecycle docs list only daily, idle, and manual reset:** The current session docs describe reuse until expiry, with only daily reset, idle reset, and manual `/new` or `/reset`. There is no documented topic-change trigger or automatic context-threshold session switch.
