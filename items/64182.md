## Summary

Keep open. Current `main` does not implement this PR's authorized-group tool-update behavior, and the discussion identifies real integration details rather than an obsolete or incoherent change.

Authorized non-forum group command sessions are still excluded.

There is no current-main coverage for authorized non-forum group command sessions receiving tool summaries/start statuses.
