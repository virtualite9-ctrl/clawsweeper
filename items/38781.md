## Summary

Keep open. Current `main` still lacks macOS `system.run.prepare` support in the companion app while the node exec path on `main` still unconditionally invokes `system.run.prepare`, so the PR’s core bug is not implemented on `main` and there is not strong evidence for any close reason.
