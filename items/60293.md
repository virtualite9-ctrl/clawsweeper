## Summary

Issue #60293 is implemented on current main: the exec-approval dashboard modal now constrains dialog height, makes long command text scroll inside the card, and keeps the action buttons reachable. Current OpenClaw also exposes a supported no-prompt path via exec approval ask policy (`ask: off`) in both UI/docs.
