## Summary

Keep open. Current `main` still matches the reported bug: agent-side node exec always sends `system.run.prepare`, while the macOS companion node still advertises only `system.run`/`system.which`/`system.notify`, so gateway command-policy would reject the preflight before it reaches the node.
