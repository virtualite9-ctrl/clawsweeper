## Summary

Keep open. Current `main` does not contain the PR's replay-side invalid-signature stripping, Bedrock Claude still bypasses the boundary-aware Anthropic transport shaper that would coerce empty-signature thinking blocks into text, and the only built-in recovery path matches a different Anthropic error. The PR also appears to have addressed the review feedback already raised in discussion.
