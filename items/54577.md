## Summary

Keep open. Current main still lacks the requested `openclaw timeline` command, `/timeline` slash command, TUI/Control UI timeline surface, and JSON-exportable agent activity timeline. Existing task, subagent, and trajectory surfaces are adjacent but do not provide the proposed cross-agent activity timeline. Related #60814 is an open draft for a web Control UI activity dashboard and timeline, but it only explicitly closes #60810 and does not cover this issue’s CLI/chat/TUI/export requirements.

## Best Possible Solution

Keep this issue open until the remaining surfaces are explicitly owned. Best maintainer path: either broaden #60814 to cover the CLI `openclaw timeline`, `/timeline`, TUI panel, and JSON export requirements, then close this as a duplicate after that scope is explicit, or implement a dedicated timeline surface over existing task, session, subagent, and trajectory data.
