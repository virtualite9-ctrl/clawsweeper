## Summary

Current `main` already supports daemon-safe proxy configuration for Gateway mode on macOS: managed service installs preserve standard proxy env vars, and the supported persistent path for daemon env is `~/.openclaw/.env`. The reported manual LaunchAgent plist edit is no longer needed as an OpenClaw core change request.

## Risks / Open Questions

- launchd still does not inherit an interactive shell environment automatically; users who only export proxy vars in their shell will still need to persist them via `~/.openclaw/.env` or reinstall the managed gateway from an environment that already has those vars.
