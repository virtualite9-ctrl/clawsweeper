## Summary

Current `main` already contains the exec/PTY late-output guard this issue requested, and the changelog records the crash fix as shipped in `v2026.4.22`. The gateway now drops late `onUpdate` emissions after run settlement or abort instead of letting them reach a disposed agent run.

## Evidence

- **Late exec/PTY updates are now dropped on current main:** `emitUpdate()` returns immediately once the session is backgrounded, exited, or updates have been disabled, which directly addresses late listener events after run end.
  - file: [src/agents/bash-tools.exec-runtime.ts:572](https://github.com/openclaw/openclaw/blob/e6d04682d33c9576edc4ad77b031949f84407bec/src/agents/bash-tools.exec-runtime.ts#L572)

  - file: [src/agents/bash-tools.exec-runtime.ts:776](https://github.com/openclaw/openclaw/blob/e6d04682d33c9576edc4ad77b031949f84407bec/src/agents/bash-tools.exec-runtime.ts#L776)
