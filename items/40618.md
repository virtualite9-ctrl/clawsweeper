## Summary

Current `main` at `b0d2003fe65bd7a9812f7c7f5ad9f608a7fd1503` does not implement configurable `session-memory` output directories. The hook still hardcodes `<workspace>/memory`, the bundled hook docs still expose only `messages` config, and the public hooks docs still describe a fixed memory path. This PR should stay open unless superseded by a different implementation.
