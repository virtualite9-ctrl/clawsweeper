## Summary

Current `main` does not already contain this docs change, and the PR's proposed wording does not match current runtime behavior closely enough to justify closing it. Heartbeat runs still go through the normal reply/fallback path, and for cross-provider overrides the configured fallback list is skipped but the configured primary is still appended as a final fallback candidate.
