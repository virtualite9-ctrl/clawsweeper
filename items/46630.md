## Summary

Keep open. Current main has slash-command autocomplete, a markdown/tool side panel, and an agent skills management view, but it does not implement the requested WebChat shortcut/keyword sidebar, `webchat.shortcuts` config, hover descriptions, categories, or click-to-insert prompt behavior.

## Best Possible Solution

Keep the issue open for maintainer/product judgment. The likely path is a small Control UI/WebChat feature: add a validated shortcut source, render a responsive collapsible sidebar or drawer in `ui/src/ui/views/chat.ts`, insert the selected prompt into the draft without auto-sending, and decide whether skill/plugin-owned trigger metadata needs a new explicit API before external plugins can participate.
