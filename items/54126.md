## Summary

Keep open. Current `main` does not implement this PR’s first-class profile management surface: there is no `profile` top-level CLI command, docs still describe the legacy `~/.openclaw-<name>` profile layout, and runtime path resolution still uses the older path-heuristic model instead of managed profile manifests.
