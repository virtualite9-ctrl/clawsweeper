## Summary

Current `main` already implements the requested behavior: gateway YOLO exec (`tools.exec.security: "full"` plus `tools.exec.ask: "off"`) skips the script-preflight rejection path, docs now say there is no separate heuristic obfuscation gate in that mode, and regression tests cover the reported heredoc/direct-interpreter patterns.

## Evidence

- **Gateway YOLO skips exec script preflight:** `shouldSkipExecScriptPreflight()` returns true for `host="gateway" && security="full" && ask="off"`, so the extra Python/Node preflight rejection layer is bypassed in the exact mode reported.
