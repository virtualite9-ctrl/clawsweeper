## Summary

Keep #11955 open. Current main has shipped adjacent memory search, opt-in session transcript indexing, usage/cost/session aggregation, startupContext for /new and /reset, and an opt-in BOOT.md startup hook, but the umbrella still has meaningful remaining product/API work: an agent-facing self-evaluation metrics API, automatic conversation chaining or compact-summary handoff, and default gateway-restart preload before first user input. The provided issue discussion also includes a recent maintainer Codex review explicitly keeping #11955 open as the canonical umbrella while #11956 is closed.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory_search, extraPaths, opt-in sessionMemory, usage/cost/session aggregation, startupContext, and BOOT.md behavior, then create narrower product/API work for an agent-facing self-evaluation metrics API, documented conversation continuation or compact-summary handoff, and explicit gateway startup preload/prewarm of core workspace context before first user input.
