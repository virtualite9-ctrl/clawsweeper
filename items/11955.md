## Summary

Keep #11955 open. Current main implements several adjacent pieces, but the umbrella still has meaningful remaining core work: an agent-facing self-evaluation/metrics API beyond usage accounting, automatic conversation chaining or compact-summary handoff, and default gateway-restart preload/prewarm of core context before the first user message. The provided issue discussion also includes a recent maintainer Codex review explicitly keeping this canonical umbrella open.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory_search, extraPaths, opt-in sessionMemory, usage/cost/session aggregation, startupContext, and BOOT.md behavior, then create narrower product/API work for an agent-facing self-evaluation metrics API, documented conversation continuation or compact-summary handoff, and explicit gateway startup preload/prewarm of core workspace context before first user input.
