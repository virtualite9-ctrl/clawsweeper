## Summary

Keep #11955 open. Current main has substantial adjacent work for memory search, transcript indexing, active memory, usage/cost summaries, startupContext, and opt-in BOOT.md hooks, but the umbrella still has meaningful missing product/API pieces: an agent-facing self-evaluation metrics API, documented conversation chaining or compact-summary handoff, and gateway-restart preload/prewarm before first user input.

## Best Possible Solution

Keep #11955 open as the canonical umbrella unless a maintainer splits it. Preserve the shipped memory and usage surfaces, then implement or track the remaining product/API work: an agent-facing self-evaluation metrics API, documented conversation continuation or compact-summary handoff, and explicit gateway-startup preload/prewarm semantics before first user input.
