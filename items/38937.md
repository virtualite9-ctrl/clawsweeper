## Summary

Keep #38937 open. Current main appears to fix the prompt/parser mismatch part of the report, but the cross-block streaming/display problem is still a meaningful remaining issue: block payloads can still be delivered before whole-reply TTS parsing/cleanup, and the final accumulated TTS path does not retroactively clean already-delivered streaming text.
