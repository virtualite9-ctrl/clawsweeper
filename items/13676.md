## Summary

Keep #13676 open. Current main still lacks the requested first-class Project object, Projects dashboard workflow, project-home-session binding, project workspace list, and project-scoped tool/skill/runtime constraints; existing agents, sessions, workspaces, skills, and sandbox settings only cover adjacent pieces.

## Best Possible Solution

Continue tracking #13676 as the canonical product/core dashboard request. A complete solution should add an explicit Project storage/protocol model, expose Projects in Control UI, bind projects to a home session plus optional workstreams, associate one or more workspace paths, and surface project-scoped tool/skill/runtime constraints while reusing existing agents, sessions, workspace, skill policy, and sandbox primitives where they fit.
