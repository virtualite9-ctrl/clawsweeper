## Summary

Current main already fixes the Bonjour/CIAO cancellation crash reported against v2026.4.24. The bundled Bonjour plugin now passes scoped process handlers into the advertiser, the advertiser classifies CIAO announcement/probing cancellations as handled, the global unhandled-rejection path returns before fatal exit when a registered handler handles the reason, and tests cover the handler and stuck-advertiser self-disable behavior. The issue comment's macOS/Homebrew confirmation is useful extra reproduction evidence and explicitly points to the same fixed #72089 path.

## Best Possible Solution

Close this issue as implemented on current main, keep the shipped Bonjour scoped handler/recovery implementation, and let the normal release process deliver it. If the same crash is reported after a release containing this main-line fix, it should be opened as a fresh regression with new logs and environment details.
