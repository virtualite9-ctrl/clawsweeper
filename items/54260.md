## Summary

Keep open. Current `main` still treats `gateway` as one tool for policy purposes, so `tools.deny` can block `gateway` but not a specific action like `config.get`. The gateway tool still exposes `config.get` and `config.schema.lookup` behind one `action` parameter, and `config.get` still returns the full redacted snapshot object including multiple config representations, so the requested action-level deny is not implemented on `main`.
