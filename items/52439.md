## Summary

Current `main` already contains the approval-flow hardening that matches this report: gateway approval IDs are registered synchronously before the request is surfaced, resolving during the initial broadcast is regression-tested, and Control UI now preserves pending approvals across reconnects while resolving by full approval id.
