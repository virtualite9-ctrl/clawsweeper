## Summary

Keep open. Current main still does not support using the bundled Perplexity web-search provider against a custom LiteLLM Perplexity-compatible search endpoint. Any configured `plugins.entries.perplexity.config.webSearch.baseUrl` forces the provider onto the chat-completions path, while the native Search API path is hardcoded to `https://api.perplexity.ai/search`, so the issue's requested LiteLLM `/search/<tool>` flow is not implemented.

## Evidence

- **Custom `baseUrl` still forces chat-completions transport:** `resolvePerplexityRuntimeTransport()` returns `chat_completions` whenever `configuredBaseUrl` or `configuredModel` is set, or when the resolved base URL is not the direct Perplexity host.
  - file: [extensions/perplexity/src/perplexity-web-search-provider.shared.ts:97](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/perplexity/src/perplexity-web-search-provider.shared.ts#L97)
  - command: `nl -ba extensions/perplexity/src/perplexity-web-search-provider.shared.ts | sed -n '97,128p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **Runtime transport logic still rejects custom base URLs for native search:** `resolvePerplexityTransport()` computes `transport` as `chat_completions` when `baseUrl` is overridden or the hostname is not `api.perplexity.ai`. A LiteLLM search endpoint therefore cannot use the structured Search API branch.
  - file: [extensions/perplexity/src/perplexity-web-search-provider.runtime.ts:142](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/perplexity/src/perplexity-web-search-provider.runtime.ts#L142)

- **Native Search API endpoint is hardcoded to Perplexity:** `runPerplexitySearchApi()` posts only to constant `PERPLEXITY_SEARCH_ENDPOINT = "https://api.perplexity.ai/search"`; it does not use configured `baseUrl`, so a LiteLLM `/search/web` endpoint cannot be targeted on the native-search path.
  - file: [extensions/perplexity/src/perplexity-web-search-provider.runtime.ts:34](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/perplexity/src/perplexity-web-search-provider.runtime.ts#L34)

  - file: [extensions/perplexity/src/perplexity-web-search-provider.runtime.ts:261](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/perplexity/src/perplexity-web-search-provider.runtime.ts#L261)
