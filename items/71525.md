## Summary

Keep open. Current `main` partially addresses the report by auto-mapping legacy `streamMode`, but it does not implement the requested safety change: Telegram still defaults to `partial` preview streaming, and the finalizer path still intentionally retains an incomplete preview on ambiguous/recoverable final-edit failures instead of guaranteeing a full final send.
