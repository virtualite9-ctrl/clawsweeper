## Summary

Keep open. Current `main` still enforces the reported 3-minute resumed CLI no-output cap by default, and it still does not support `null`/disabled watchdog maxima. The issue body slightly overstates scope because per-backend watchdog overrides already exist, but the requested default-behavior change is not implemented on `main` or in the latest release tag provided.
