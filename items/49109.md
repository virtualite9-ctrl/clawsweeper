## Summary

Keep open. Current `main` does not implement the PR's two core CLI fixes: `warnIfCronSchedulerDisabled` still performs an unconditional `cron.status` helper call with no `--json` skip or quiet/probe options, and `callGatewayFromCliRuntime` still forwards a single CLI-mode gateway call with no bounded transport retry path.
