## Summary

Keep PR #71796 open. Current main still has the Docker VM runtime docs bug the PR targets, and the PR contains useful Docker URL and wacli Linux-container corrections. It is not ready to close or merge as-is because its AGENTS.md replacement tells Mac users to append `--deep` to `openclaw gateway restart`, but `restart` only accepts `--json`; `--deep` is valid for `gateway status` only. The PR body also still claims a Node baseline update that is not present in the diff.

## Best Possible Solution

Keep #71796 open and request a small revision: preserve the Docker VM runtime URL fixes and wacli Linux-container clarification, change AGENTS.md to recommend `openclaw gateway restart` for restarts and `openclaw gateway status --deep` only for diagnostics, and either update the AGENTS runtime line to Node 22.14+/Node 24 recommended or remove the Node claim from the PR. Coordinate with #72154 so only one Docker docs fix lands.
