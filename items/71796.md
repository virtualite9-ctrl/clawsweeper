## Summary

Keep PR #71796 open. Current main still has the Docker VM runtime docs gap the PR targets, and external release checks confirm the existing sample URLs are stale. The PR is not ready to close or merge as-is because the AGENTS.md gateway wording still needs revision: `gateway restart` does not accept `--deep`, while `gateway status` does. The Node baseline claim also needs cleanup because package engines already require Node >=22.14.0 but the PR diff does not update the AGENTS runtime line.

## Best Possible Solution

Keep the PR open and ask for a small revision: preserve the Docker VM runtime URL fixes, change the gateway guidance to recommend `openclaw gateway restart` for restarts and `openclaw gateway status --deep` only for diagnostics/extra service scans, and either update AGENTS.md's runtime line to `Node >=22.14.0` or remove that claim from the PR. After that, docs-only validation such as the changed docs lane is enough.
