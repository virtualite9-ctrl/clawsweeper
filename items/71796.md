## Summary

Keep PR #71796 open. Current main still has the Docker VM runtime URL bug and stale AGENTS guidance, and this PR contains useful Docker fixes that are not fully duplicated by #72154, especially removing the Linux `wacli` example and selectively extracting only the needed binaries. It should not be closed as implemented or superseded, but it needs revision because its AGENTS change would incorrectly tell Mac users to append `--deep` to `openclaw gateway restart`, which current CLI code and docs do not support.

## Best Possible Solution

Keep #71796 open for maintainer review or author revision. The best path is to preserve its useful Docker corrections where they are stronger than #72154, especially selective binary extraction and removing or qualifying the stale Linux `wacli` example, while fixing or dropping the AGENTS `restart --deep` wording. If maintainers prefer #72154 as the canonical Docker URL fix, fold the nonduplicated `wacli` and extraction details into that PR, then close #71796 only after its remaining AGENTS/Node docs value is either corrected elsewhere or explicitly rejected.
