## Summary

Keep PR #71796 open. Current main still has the broken Docker VM runtime URLs and stale gateway/Node AGENTS guidance, so the user-visible docs problem is real. However the PR is not clean to close as implemented or superseded: its AGENTS change would document an invalid `openclaw gateway restart --deep` invocation, while related PR #72154 only covers the Docker `gog`/`goplaces` URL slice and leaves the AGENTS and `wacli` parts separate.

## Best Possible Solution

Keep #71796 open for maintainer or author revision. The best path is to either narrow it after #72154 lands, or revise it to use the focused Docker URL approach from #72154, handle `wacli` with verified release evidence, correct AGENTS.md so `--deep` applies only to `openclaw gateway status`, and update the AGENTS Node line to match the shipped `>=22.14.0`/Node 24 recommendation.
