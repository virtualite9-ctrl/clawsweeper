## Summary

Keep open. Current `main` still has a TUI-specific leak path: when assistant output arrives as plain text or text blocks containing inline `<think>`/`<thinking>` tags, the TUI formatter keeps that text visible instead of stripping internal scaffolding. The `showThinking` toggle only suppresses explicit `thinking` blocks, so the reported behavior remains plausible on current `main` and the latest release.

## Evidence
