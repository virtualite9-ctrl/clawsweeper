## Summary

Keep #47558 open. Current main still reproduces several of the reported sub-agent documentation gaps/inaccuracies, and the linked fix PRs in the provided GitHub context are either still open or closed unmerged rather than shipped.

## Best Possible Solution

Keep #47558 as the canonical tracker until a focused docs PR lands. The best fix is a small update to `docs/tools/subagents.md` that documents `announceTimeoutMs`, adds the ACP-only `streamTo: "parent"` parameter, generalizes the depth and auto-archive wording for max depths up to 5, and replaces vague tool-policy phrasing with the concrete deny/allow behavior from `src/agents/pi-tools.policy.ts`.
