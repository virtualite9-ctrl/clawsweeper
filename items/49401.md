## Summary

Keep open. Current `main` already classifies this generic provider wording as a transient timeout, but it still does not rewrite the plain-text `An error occurred while processing your request ... request ID ...` shape into the friendlier user-facing copy described in the PR. The PR therefore still addresses a real gap, even though part of its failover-classification change is now redundant on `main`.
