## Summary

Keep #8582 open. Current main implements the ElevenLabs realtime STT/provider-generic Voice Call streaming part, but the issue bundled three asks and the duplicate-warning control plus focused schema-extension docs remain unresolved. Existing issue discussion reached the same partial-complete conclusion, and related PR #8922 was closed unmerged and scoped to ElevenLabs streaming TTS, not these remaining DX asks.

## Best Possible Solution

Keep #8582 open unless a maintainer splits it. Best path: add a tested per-id duplicate override/suppression acknowledgement in plugin config and manifest-registry diagnostics, document the override behavior, and add focused plugin docs explaining how to update configSchema when extending or forking a bundled plugin. Alternatively, create canonical follow-up issues for those two remaining DX asks before closing this umbrella item.
