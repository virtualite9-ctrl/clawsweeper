## Summary

Keep #8582 open. Current main implements the ElevenLabs realtime STT/provider-generic Voice Call portion, but the issue bundled three asks and the duplicate-plugin warning control plus focused custom-plugin schema-extension docs remain unresolved. The existing issue comments reached the same partial-complete conclusion, and the linked #8922 was a closed, unmerged ElevenLabs TTS proposal rather than a canonical fix for the remaining DX work.

## Best Possible Solution

Keep #8582 open unless a maintainer splits it. The best path is to add a tested per-id duplicate override/suppression acknowledgement in plugin config and manifest-registry diagnostics, document the override behavior, and add focused plugin docs explaining how to update `configSchema` when extending or forking a bundled plugin. Alternatively, create canonical follow-up issues for those two remaining DX asks before closing this umbrella item.
