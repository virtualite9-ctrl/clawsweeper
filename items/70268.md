## Summary

Keep open. Current `main` still routes wake hooks to the default main session, does not accept wake `sessionKey` input, and its own tests assert that behavior, so the PR’s requested wake-session routing is not implemented on `main` yet. The PR discussion also shows a maintainer-requested policy fix that the contributor says is addressed on PR head, which argues for review rather than cleanup closure.

## Evidence
