## Summary

Keep PR #72154 open. Current main still contains the reported broken Docker VM runtime URLs, GitHub release metadata supports the PR's `gogcli` and `goplaces` replacement asset names, and the overlapping #71796 is not a clean superseder because it includes broader AGENTS/gateway guidance concerns. The PR needs maintainer review or revision, not conservative cleanup closure.

## Best Possible Solution

Keep #72154 open for maintainer review. The best docs outcome is a focused `docs/install/docker-vm-runtime.md` fix that uses the working `gogcli` and `goplaces` versioned release assets, extracts binaries without leaving tarball side files in the image layer, and either corrects or removes the stale Linux `wacli` example. If maintainers prefer #71796's broader Docker changes, split or revise that PR's Docker-only portion rather than closing this narrower fix as obsolete.
