## Summary

Keep open. Current `main` does not implement the requested bootstrap-side CA bundle detection for non-`nvm` Node version managers, so this is not `implemented_on_main` or `cannot_reproduce`. The PR rationale matches a real gap, although the review discussion also surfaced an unresolved `nvs` path edge that means the patch likely needs revision before merge.
