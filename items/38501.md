## Summary

Keep #38501 open. Current main still does not add an active-run hint to TUI Ctrl+C status text, while Esc and `/abort` remain the documented abort paths. The linked fix PR #38502 is still open/unmerged in the provided GitHub context, #38540 was closed unmerged, and #38661 is also open, so the requested user-facing clarification has not landed on main.
