## Summary

Keep this PR open. Current main still has no Feishu thread-specific mention override, and the PR remains a coherent implementation candidate whose review findings were addressed on the branch. It overlaps with #48945, but that PR is also open and not an established canonical replacement yet.

## Best Possible Solution

Keep the PR open for Feishu maintainer review. The best path is to choose between this scoped `requireMentionInThread` option and #48945's `threadFollowUp` design, then land one rebased implementation with schema/docs/config metadata updates and tests for topic groups, `replyInThread` groups, bidirectional overrides, normal quote replies, and top-level mention gating. If #48945 becomes the selected merged implementation, this PR can then close as superseded.
