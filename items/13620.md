## Summary

Keep #13620 open. Current main still lacks outbound Telegram media-group/album sending: Telegram reply delivery, direct send, and outbound payload paths fan multiple media URLs out as individual media sends, and there is no `sendMediaGroup`/`InputMedia` path or documented album parameter. The related in-repo attempts are closed and unmerged, so they do not supersede this issue.

## Best Possible Solution

Keep #13620 open and implement outbound album support in the bundled Telegram plugin. The likely path is to add a `sendMediaGroup` flow for 2-10 groupable photos/videos across reply delivery, direct Telegram send/action, and outbound payload delivery, with focused Telegram tests and docs updates for any new `mediaUrls`/`mediaGroup`/album user-facing parameter.
