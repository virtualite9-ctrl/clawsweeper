## Summary

Current `main` already implements Control UI drag-and-drop image attachments end to end: dropped files are captured in the chat UI, serialized into `chat.send.attachments`, normalized by Gateway, parsed into image/media inputs for the agent path, and covered by tests that assert both transcript persistence and agent delivery.
