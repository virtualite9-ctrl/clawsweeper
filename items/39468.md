## Summary

Current main no longer routes Feishu group messages through the dynamic-agent config-write path, and the current config schema only permits subagent settings under `agents.defaults.subagents`, not `agents.subagents`. The reported group-message crash path from v2026.3.2 is therefore already addressed on main.
