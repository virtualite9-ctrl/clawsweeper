## Summary

Current `main` already supports pasting and dragging images into the Control UI chat composer, previews those images locally, and sends them through `chat.send` as image attachments. The same implementation is present in the shipped `v2026.4.23` release.

## Evidence

- **Control UI paste and drag handlers exist on current main:** `handlePaste` reads clipboard image items into chat attachments, and `handleDrop` reads dragged image files into the same attachment list. The chat root binds `@drop`/`@dragover`, and the composer textarea binds `@paste`.
