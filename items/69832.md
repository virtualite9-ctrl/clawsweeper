## Summary

Keep open. Current `main` does not implement `openclaw diagnose`, and the repo already has adjacent diagnostics surfaces with a stricter privacy model (`gateway diagnostics export`, `gateway stability`). This PR is a fresh net-new feature proposal with substantive review feedback still in play, so it does not fit any conservative auto-close reason.
