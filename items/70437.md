## Summary

Keep open. Current `main` still sanitizes signed Anthropic thinking bytes during replay, still ignores raw `signature` fallback in transport message replay, and does not contain the helper/tests described in this PR, so this is not implemented on `main` and should not be conservatively auto-closed.
