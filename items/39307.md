## Summary

Keep #39307 open. Current main has partial compaction surfacing, but it still does not implement the requested default hard-to-miss user signal plus assistant-side disclosure/re-grounding contract after mid-thread compaction.

## Best Possible Solution

Keep this open for maintainer/product work. A complete fix should define the default compaction/reset visibility contract across supported chat surfaces, decide which surfaces get durable or hard-to-miss UI/system events, add safe assistant-side disclosure and re-grounding behavior after compaction, and update docs/tests around `agents.defaults.compaction.notifyUser`, post-compaction recovery semantics, and any customer-facing bot suppression knob such as the one discussed in #72043.
