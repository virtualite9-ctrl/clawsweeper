## Summary

Keep open. Current `main` does not implement the PR’s exec-approval denylist feature: core types still only support `allowlist`/`full` and `off`/`on-miss`/`always`, policy evaluation is still allowlist-only, and the Control UI/docs still expose allowlists only.

## Evidence
