## Summary

Keep open. Current `main` still lacks the media-only/text-aware suppression logic, so the reported behavior is not implemented there. The PR is also not ready to close out conservatively because the latest review thread identifies a remaining correctness bug in the proposed `sentText` logic for mixed same-turn sends to different targets.

## Evidence
