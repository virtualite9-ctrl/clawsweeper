## Summary

Keep open. Current `main` does not implement this fix, and the Google Vertex ADC path still appears to flow through a synthetic API-key marker that would be treated as a literal Google API key rather than OAuth/ADC. The PR body's exact `<authenticated>` sentinel story may be stale against today’s tree, but the underlying cleanup should not be closed as already fixed or obsolete.

## Evidence
