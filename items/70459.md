## Summary

Keep PR #70459 open. Current main still lacks the proposed Step 0 prerequisite checker in Getting Started, and the PR is a focused docs improvement, but it should not be merged as-is because review comments identify real shell snippet bugs and an overbroad installer auto-install claim. The best maintainer path is to revise the PR against the actual installer behavior rather than close it as obsolete.

## Best Possible Solution

Keep the PR open and request a small revision: fix the bash and PowerShell check snippets, scope the auto-install language to the actual install.sh/install.ps1 behavior, and keep the Getting Started page concise by linking deeper Node/PATH troubleshooting to https://docs.openclaw.ai/install/node and installer details to https://docs.openclaw.ai/install/installer.
