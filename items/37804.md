## Summary

Keep open. Current `main` has incremental forward-compat fixes for runtime and `models list`, but the RFC's broader invariant is not fully implemented: model list/status, runtime resolution, and capability/thinking gates still use separate data shapes and logic rather than one shared canonical resolved-model result.

## Evidence
