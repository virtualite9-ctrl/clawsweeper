## Summary

Keep #35203 open. Current main has useful building blocks for isolated subagents, plugin hooks/runtime helpers, memory search, session visibility, and usage reporting, but it does not implement the RFC's central capabilities: learned capability profiling/routing, shared blackboard storage/query/pub-sub, layered private/team/global memory, or global/per-task token/cost budget enforcement. The Octo/teleconference and KinthAI discussion is useful prior art, but it is not a merged complete implementation in this checkout.

## Best Possible Solution

Keep #35203 open as an umbrella RFC until maintainers split it into focused follow-ups: capability metrics/routing, blackboard or artifact search/query, memory visibility and promotion gates, and token/cost budget aggregation/enforcement. Encourage plugin-first experiments through runtime.subagent, taskFlow, before_tool_call, model/LLM hooks, subagent lifecycle hooks, session visibility, and memory plugin seams where sufficient; file narrow core/plugin API issues where those seams cannot express pre-spawn task-aware routing, visibility, telemetry, or enforcement.
