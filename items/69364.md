## Summary

Keep #69364 open. Current main has per-agent file, tools, and skills editors plus flat session/task surfaces, but it still lacks the requested cross-agent Control UI view that relates delegation, dataflow, task/flow sequence, and live session status. The related #52803 is broader orchestration-board work, not a clean duplicate of this config/workspace relationship view.

## Best Possible Solution

Keep the issue open as a focused first-party Control UI product/API request. The best path is a read-only MVP that assembles supported current data first: agents.list and subagents.allowAgents for delegation, session rows for parent/child/live status, and the task/Task Flow ledger for sequence. Reuse the existing per-agent Files, Tools, and Skills panels for drill-in editing, and coordinate scope with open #52803 without treating it as an exact duplicate.
