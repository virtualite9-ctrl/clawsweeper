## Summary

Keep #25295 open. Current main still does not expose Mattermost `message action=edit` or `message action=delete`; it only has internal post update/delete helpers used by preview streaming and button cleanup. The related block-streaming PRs were closed unmerged, and the maintainer Codex comment from April 26, 2026 remains accurate against current main.

## Best Possible Solution

Keep #25295 open. The best path is a focused refreshed Mattermost plugin change that wires `edit` and `delete` through `describeMattermostMessageTool()`, `supportsAction()`, and `handleAction()` in `extensions/mattermost/src/channel.ts`, reuses the existing `updateMattermostPost()` / `deleteMattermostPost()` helpers, adds targeted Mattermost action tests, updates `docs/cli/message.md` and `docs/channels/mattermost.md`, and drops unrelated gate-fix churn from the branch.
