## Summary

Keep #25295 open. Current main has Mattermost post update/delete helpers for draft preview streaming and interaction cleanup, but Mattermost still does not advertise or handle shared `message action=edit` or `message action=delete`. The related block-streaming PRs are closed and unmerged, and the existing maintainer/Codex comment already identified the same remaining gap.

## Best Possible Solution

Keep #25295 open, or replace it with a focused Mattermost plugin PR that wires `edit` and `delete` through `describeMattermostMessageTool()`, `supportsAction()`, and `handleAction()` in `extensions/mattermost/src/channel.ts`, reuses the existing Mattermost post update/delete helpers where appropriate, adds targeted tests and docs, validates required edit content, and drops unrelated branch cleanup.
