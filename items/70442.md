## Summary

Keep open. Current `main` still routes `dmScope="main"` direct chats onto the agent main session key, and sandbox mode `non-main` still exempts that key. The reported routing/sandbox problem is therefore not implemented on `main`, and the open PR also carries unrelated changes plus at least one concrete review concern.
