## Summary

Keep open. Current `main` still has no model-specific fix for `ollama/orieg/gemma3-tools:12b-ft`, and the docs now explicitly acknowledge that some Gemma/local backends can fail on full OpenClaw agent turns when tool schemas are present. That reads as an acknowledged compatibility limitation with mitigations, not a confirmed fix or a strong enough basis to close.
