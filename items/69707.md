Keep open. Current `main` still sends `dimensions` in the upstream embeddings request, and the current test suite explicitly expects that behavior, so this PR is neither implemented on `main` nor obsolete. The PR discussion also shows the only concrete review concern was addressed by follow-up commit `4ce5eb168d0320f831817848607f8b3e1e069803`, so there is no conservative auto-close basis.
