## Summary

Keep this PR open. Current main still lacks the mcporter empty-result direct-qmd fallback, while the PR is a focused implementation candidate with regression coverage and follow-up fixes for the prior review comments.

## Best Possible Solution

Keep this PR open and review the updated branch.
