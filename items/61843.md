## Summary

Current main already includes the Kimi web_search auth/base-url fix for Moonshot `.cn` keys, plus matching setup prompts, tests, and docs. The report's 401 symptom is consistent with the now-fixed host/key mismatch or with using a non-Moonshot key type, both of which are clarified in current docs/UI.
