## Summary

Keep #68982 open. Current main still does not implement the PR's central behavior: live chat deltas/finals are text-only, the Control UI state and item model collapse streaming payloads to strings, and the streaming renderer explicitly disables reasoning. The PR remains useful and is not superseded by the related #70815 resync fix, which explicitly says it does not reimplement reasoning streaming.

## Best Possible Solution

Keep #68982 open and review/land it or an equivalent end-to-end fix that carries structured assistant `thinking` plus `text` through gateway chat delta/final payloads, stores structured stream payloads in Control UI state/items, threads `showReasoning` into streaming rendering, preserves reading-indicator behavior when reasoning is hidden, and keeps the existing silent/heartbeat/control-token suppression contracts covered by focused gateway and UI tests.
