## Summary

Keep #68982 open. Current main still does not implement live-chat reasoning streaming: gateway chat deltas/finals are text-only, Control UI stream state is string-only, build/render paths only accept visible text, and the streaming renderer explicitly disables reasoning. The PR is a focused, recent fix attempt with useful review discussion; the related items do not clearly supersede it.

## Best Possible Solution

Keep #68982 open and have maintainers either land a refreshed focused fix or deliberately fold it into a chosen canonical reasoning-streaming PR. The implementation should carry structured assistant stream payloads with thinking and text through gateway chat delta/final events, preserve that structure in Control UI state/items, render reasoning-only live bubbles only when the Show Thinking/session level allows it, and cover the heartbeat, control-token, reconnect, abort/orphan cleanup, and message-less final resync contracts with regression tests.
