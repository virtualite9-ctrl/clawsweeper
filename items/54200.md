## Summary

Keep open. Current `main` has improved startup messaging for missing `gateway.mode`, but the root report about `~/.openclaw/moltbot.json` being ignored is not implemented away: config discovery and doctor migration only recognize `openclaw.json` plus `clawdbot.json`, not `moltbot.json`, and the breaking-change note still only calls out `~/.moltbot`.
