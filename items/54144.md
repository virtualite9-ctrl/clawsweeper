## Summary

Keep open. Current `main` does not contain this change, and the proposed 30s `AbortSignal.timeout(...)` on `/api/pull` would attach to the same request whose body is then streamed via `reader.read()`, so it risks aborting legitimate long Ollama downloads rather than only bounding initial connection setup.

## Evidence

- **Current main still streams `/api/pull` over one long-lived fetch with no per-request abort signal:** `pullOllamaModelCore` opens `/api/pull` with `fetchWithSsrFGuard(...)`, then keeps the same `response.body.getReader()` alive in a loop until the stream ends. There is no existing `signal` on this call in `main`.
  - file: [extensions/ollama/src/setup.ts:167](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/extensions/ollama/src/setup.ts#L167)
  - command: `nl -ba extensions/ollama/src/setup.ts | sed -n '159,248p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **A request signal would govern the whole guarded fetch lifecycle:** `fetchWithSsrFGuard` builds one abort signal for the request and passes it into the underlying fetch init before returning the streaming `Response`, so a 30s abort on that request would also cut off later body reads on the same response.
  - file: [src/infra/net/fetch-guard.ts:313](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/infra/net/fetch-guard.ts#L313)
  - command: `nl -ba src/infra/net/fetch-guard.ts | sed -n '301,407p'`
  - sha: [8f64cd3e4d83](https://github.com/openclaw/openclaw/commit/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf)
- **OpenClaw already uses long stream timeouts for guarded fetches:** The guarded-fetch/Undici timeout plumbing is designed around long-lived streams: the global default stream timeout is 30 minutes, and dispatcher timeouts are wired to `bodyTimeout` and `headersTimeout`, not a short 30s total request deadline.
  - file: [src/infra/net/undici-global-dispatcher.ts:6](https://github.com/openclaw/openclaw/blob/8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf/src/infra/net/undici-global-dispatcher.ts#L6)
