## Summary

Keep open. Current `main` no longer uses raw `params.card` in the Feishu `send`/`thread-reply` path, so the PR's original reproduction is partly stale there, but equivalent empty-card validation is still not implemented in the exported Feishu send/edit helpers or the `edit` action. That means the PR still contains useful bug-fix work and there is not strong evidence for an auto-close reason.

## Evidence
