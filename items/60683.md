## Summary

Keep open. Current `main` does not implement the PR’s requested `memorySearch.local` llama tuning knobs (`gpu`, `gpuLayers`, `flashAttention`), and the existing review discussion identifies real follow-up work the PR still needs before it would be safe to land.

## Evidence

There is no support for `gpu`, `gpuLayers`, or `flashAttention` in the resolved agent config.
