## Summary

Current `main` already repairs orphaned assistant tool calls during replay by inserting synthetic error `toolResult` entries, then applies that repair in the session-history sanitization path used before provider requests. Anthropic replay policy explicitly enables this path, and tests cover both the synthetic insertion and the Anthropic policy wiring.
