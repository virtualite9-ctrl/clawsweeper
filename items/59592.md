## Summary

Current `main` still does not suppress aborted embedded assistant partials in payload assembly, so this PR is not redundant. The code only treats `stopReason === "error"` as an error surface, still emits reasoning text for aborted turns, and still forwards `assistantTexts`/fallback answer text unless the deterministic-approval suppression path is active.
