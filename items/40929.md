## Summary

Current `main` already addresses the concrete failure modes from this report: plugin installs now use the manifest id as the config key, plugin install/enable flows update allowlists, async `register()` is rejected as a plugin load error instead of being silently half-applied, the global hook runner is preserved/reinitialized across reload and cache paths, and the macOS LaunchAgent restart path was fixed in the shipped 2026.3.11 release notes.

If this still reproduces on a current build, please open a fresh issue with the exact plugin package/version, current OpenClaw version, and gateway logs from the failing restart.
