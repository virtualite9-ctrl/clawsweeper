## Summary

Keep #23896 open. Current main still lacks Discord per-sender mention gating: the typed config, strict schema, runtime resolver, preflight path, tests, and docs expose only guild/channel-level `requireMention` plus sender allowlists/bot modes, not `requireMentionFromIds` or an equivalent.

## Best Possible Solution

Keep #23896 open. Best path is an additive Discord plugin/config change: add typed guild/channel `requireMentionFromIds`-style config, extend strict zod schema plus generated metadata/docs, make the mention-requirement resolver sender-aware, and add Discord preflight tests proving listed human senders must mention while unlisted allowed bridge bots can trigger without mentions.
