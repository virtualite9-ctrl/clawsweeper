## Summary

Current `main` already implements the PR's behavior: the overview language dropdown prefers the persisted `settings.locale` over `i18n.getLocale()` and has a regression test covering the refresh/hydration race.

## Evidence

- **Overview uses persisted settings locale first:** `currentLocale` now resolves from `props.settings.locale` when it is a supported locale, and only falls back to `i18n.getLocale()`. This is the same fix the PR proposes, with an added validity check.
  - file: [ui/src/ui/views/overview.ts:252](https://github.com/openclaw/openclaw/blob/ed6b487e2033d87a85b5713a964cc677c70353f8/ui/src/ui/views/overview.ts#L252)
  - command: `nl -ba ui/src/ui/views/overview.ts | sed -n '248,366p'`
  - sha: [ed6b487e2033](https://github.com/openclaw/openclaw/commit/ed6b487e2033d87a85b5713a964cc677c70353f8)
- **Persisted locale is loaded synchronously from UI settings:** `loadSettings()` reads `parsed.locale` from stored UI settings and returns it on initial app state, matching the PR rationale that the saved locale is available before i18n hydration finishes.
  - file: [ui/src/ui/storage.ts:266](https://github.com/openclaw/openclaw/blob/d38ed0831d7372c8505dc944aea69b5b83da1435/ui/src/ui/storage.ts#L266)
  - command: `nl -ba ui/src/ui/storage.ts | sed -n '232,276p'`
- **Startup wiring applies stored locale:** `OpenClawApp` initializes `settings` from `loadSettings()` and immediately calls `i18n.setLocale(this.settings.locale)` when the stored locale is supported.
  - file: [ui/src/ui/app.ts:144](https://github.com/openclaw/openclaw/blob/d38ed0831d7372c8505dc944aea69b5b83da1435/ui/src/ui/app.ts#L144)
  - command: `nl -ba ui/src/ui/app.ts | sed -n '138,148p'`
- **Regression test covers the reported race:** The render test asserts that with `settings.locale = "zh-CN"` and `i18n.getLocale() === "en"`, the dropdown still renders `zh-CN` selected before i18n hydration completes.
  - file: [ui/src/ui/views/overview.render.test.ts:64](https://github.com/openclaw/openclaw/blob/d38ed0831d7372c8505dc944aea69b5b83da1435/ui/src/ui/views/overview.render.test.ts#L64)
