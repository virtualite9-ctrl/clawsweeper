## Summary

Keep open. Current `main` does not implement the requested general egress firewall for web tools. The repo has low-level SSRF hostname allowlist support, but `tools.web.fetch` only exposes `ssrfPolicy.allowRfc2544BenchmarkRange`, and the only `allowedDomains` config in the codebase is the narrow `tools.web.search.openaiCodex.allowedDomains` path for native Codex web search. The security docs still describe external `web_fetch` exfiltration as a residual risk and list URL allowlisting as a pending recommendation.
