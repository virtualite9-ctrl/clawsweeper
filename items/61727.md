## Summary

Current main already contains a shipped fix for the reported hybrid-search failure mode: when strict hybrid scoring plus `minScore` would drop keyword-backed FTS hits, builtin memory search now falls back to a text-weight floor instead of returning `No matches`. That matches the issue’s config pattern (`minScore: 0.32` with hybrid search enabled) and was documented as shipped in v2026.2.27.
