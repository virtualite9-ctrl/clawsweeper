## Summary

Current `main` already ships the supported Railway path: the latest release documents Railway deployment, the gateway honors `OPENCLAW_GATEWAY_PORT`, defaults container startups to non-loopback bind behavior, and seeds Control UI origins on startup. The remaining PR delta is fork-specific and includes unresolved security regressions, so this PR is no longer the right change to merge.
