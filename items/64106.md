## Summary

Keep open. Current `main` at `5b8bd6371c666a5de43c82305e0315771e8e49d5` still lacks per-server MCP `tools.allow` / `tools.deny` support in the bundle MCP runtime, config types/schema, and user docs, so this PR is not implemented on main and is not obsolete. The PR discussion also points to independent issue `#65612` describing the same problem, which makes this look like still-relevant product work rather than stale churn.
