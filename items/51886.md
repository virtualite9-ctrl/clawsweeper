## Summary

Current main no longer matches the reported 2026.3.13 crash surface. The reported release pinned the older ElevenLabsKit revision c8679fbd, while current main and releases starting with v2026.4.5 pin ElevenLabsKit 7e3c948, which contains upstream fix 518e542 for the StreamingAudioPlayback teardown/semaphore waiter race plus regression coverage. Current main also adds a gateway talk.speak retry for local macOS ElevenLabs playback failures before falling back to system voice.
