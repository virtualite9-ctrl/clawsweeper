## Summary

Keep open. Current `main` still implements compaction only under `agents.defaults.compaction`, not per-agent `agents.list[].compaction`, so this PR's requested behavior is not already on `main`. The PR also appears to mix the compaction feature with unrelated changes, so there is not strong evidence for a conservative close.
