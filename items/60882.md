## Summary

Keep open. Current `main` only partially overlaps with this PR: it has a small config-path smoke test and some later utility tests elsewhere, but it does not already contain the dedicated `byte-size`/`config-paths` test files described in the PR. The discussion also surfaced multiple deterministic false-negative tests in the posted branch, so this is not a strong `implemented_on_main` close and not a candidate for the other close reasons either.
