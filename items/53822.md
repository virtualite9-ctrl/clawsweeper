## Summary

Keep open. Current `main` at `8f64cd3e4d8339a6aab3ca230d5e14944c91cfaf` still implements sandbox Docker mounts as bind strings only; the declarative `sandbox.docker.volumes` feature from PR #53822 is not present on `main`.
