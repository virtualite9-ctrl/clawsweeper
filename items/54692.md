## Summary

Keep open. Current `main` still uses the generic `gateway url override requires explicit credentials` message and the generic `Fix: pass --token or --password (or gatewayToken in tools).` hint, while docs now describe the stricter CLI-vs-env override rules. That means the PR's source-specific UX improvement is not implemented on `main`, and the item is still a plausible cleanup rather than something safe to auto-close.
