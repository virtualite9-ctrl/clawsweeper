## Summary

Keep open. I did not find a fix on `main` or in the latest release for the reported WhatsApp group-mention path. The current code still funnels group mention gating through a narrow `extractMentionedJids()` implementation, and there is no test coverage proving the reporter's payload shape is handled.

## Evidence

- **Mention extraction is still a fixed shortlist:** `extractMentionedJids()` only reads `contextInfo.mentionedJid` from a small hard-coded set of message types and returns `undefined` otherwise. It does not use the broader context walker in the same file.
