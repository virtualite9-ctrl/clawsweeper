## Summary

Current `main` already avoids the reported empty/default-heartbeat spam by skipping heartbeat model calls when `HEARTBEAT.md` is effectively empty, including the scaffolded default template. The PR's template rewrite is therefore obsolete for the reported symptom and would also change the still-documented missing-file proactive behavior.
