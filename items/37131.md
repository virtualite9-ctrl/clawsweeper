## Summary

Keep #37131 open. Current main has useful no-LLM direct invocation slices, including HTTP `/tools/invoke`, plugin slash commands, and skill `command-dispatch: tool`, but the umbrella request is still not implemented across the requested generic coding-tool, cron, CLI, and slash-command surfaces. Related #63919 and #18160 remain active narrower slices rather than a complete superseding tracker.

## Best Possible Solution

Keep #37131 open as the umbrella product/API request until maintainers either ship a single security-reviewed direct invocation contract across HTTP, cron, CLI, and slash-command surfaces, or split and link canonical narrower trackers for every remaining surface. Keep related #63919 and #18160 referenced as active implementation slices, not replacements for the umbrella.
