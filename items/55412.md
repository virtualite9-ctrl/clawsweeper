## Summary

Current `main` no longer falls through to the generic `Agent failed before reply: Gateway is draining...` message, but it still surfaces a user-visible restart message and does not auto-retry `GatewayDrainingError`. That means the exact reported string is obsolete, yet the requested behavior in #55412 is still not implemented.
