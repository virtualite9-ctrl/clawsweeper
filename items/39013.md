## Summary

Keep open. Current main still appears to render bootstrap injection stats from `file.name` directly, while `/context list` prints that `name` field; a hook-injected file with only `path` can still surface as `undefined`. The linked fix PRs were closed unmerged in the provided GitHub context, and I did not find an equivalent landed change on main.
