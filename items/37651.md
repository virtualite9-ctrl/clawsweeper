## Summary

Keep open. Current `main` does not implement a lightweight cron shell/exec payload type. The cron type definitions, gateway schema, tool schema/help, and user docs all still support only `payload.kind: "systemEvent"` and `payload.kind: "agentTurn"`. There is evidence of an unmerged PR branch for this work (`refs/pull/51276/*`), but nothing on current `main` proves the feature shipped.
