## Summary

Current `main` already contains the Anthropic transcript-replay/compaction hardening needed for this report, and that stack shipped in `v2026.4.23` (`a9797214338ba31c52c796adbb75afb16e0684a9`).

## Evidence

- **Anthropic replay policy preserves immutable thinking for Claude 4.x:** `shouldAllowProviderOwnedThinkingReplay` enables the immutable-signature path for Anthropic-compatible APIs, and the unowned Anthropic fallback only sets `dropThinkingBlocks` for older Claude models. For `claude-opus-4-6`, thinking is preserved rather than rewritten.
  - file: [src/agents/transcript-policy.ts:32](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/transcript-policy.ts#L32)
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
- **Session history is sanitized before replay/compaction:** `sanitizeSessionHistory` applies the transcript policy before replayed requests, including image sanitation, conditional thinking dropping, tool-call repair, and provider-specific replay sanitation. This is the path used before follow-up turns such as requester-session announce delivery.
  - file: [src/agents/pi-embedded-runner/replay-history.ts:455](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/pi-embedded-runner/replay-history.ts#L455)
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
- **Unsafe signed-thinking turns are dropped instead of mutated:** When provider-owned thinking replay is active, assistant turns that mix signed thinking with non-replay-safe sibling tool calls are removed wholesale rather than normalized in-place, specifically to avoid mutating provider-owned thinking content.
  - file: [src/agents/session-transcript-repair.ts:289](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/session-transcript-repair.ts#L289)
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
- **Regression tests cover Claude Opus 4.6 policy:** The test suite explicitly asserts that `claude-opus-4-6` does not drop thinking blocks under the Anthropic replay policy, while legacy Sonnet 3.7 still does.
  - file: [src/agents/transcript-policy.test.ts:288](https://github.com/openclaw/openclaw/blob/c070509b7f8fdbe479e5d829643782c272e30e83/src/agents/transcript-policy.test.ts#L288)
  - sha: [c070509b7f8f](https://github.com/openclaw/openclaw/commit/c070509b7f8fdbe479e5d829643782c272e30e83)
