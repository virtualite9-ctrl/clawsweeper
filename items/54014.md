## Summary

Keep open. Current main honors OPENCLAW_HOME for shared state/config/profile paths, but the reported OPENCLAW_GIT_DIR/default checkout and installer onboarding/workspace gaps remain on main. The related open #65536 PR is about sandbox tilde expansion and does not supersede this broader OPENCLAW_HOME contract issue.

## Best Possible Solution

Keep this issue open and narrow the remaining work to the OPENCLAW_HOME path contract. If maintainers decide OPENCLAW_HOME should seed the git checkout and installer/onboarding defaults, update src/cli/update-cli/shared.ts, scripts/install.sh, docs/cli/update.md, docs/help/environment.md, and focused tests for OPENCLAW_GIT_DIR/default checkout plus installer config/workspace behavior. Review #65536 separately as a sandbox tilde-expansion fix rather than the canonical fix for this issue.
