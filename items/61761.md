## Summary

Current `main` already resolves the four reported TypeScript errors directly in source: the control runtime exports `listControlledSubagentRuns`, `ChatMessage` is imported/exported in the shared subagents module, and legacy `x_search.apiKey` access is narrowed through `Record<string, unknown>` before use.
