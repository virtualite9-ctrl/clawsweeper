## Summary

Keep open. Current main has partial image containment for main chat markdown and attachment images, but the linked regression is not fully resolved: sidebar markdown still allows images to expand to the full sidebar width, and the directly related fix PR #46947 remains open/unmerged.

## Best Possible Solution

Keep #45846 open until #46947 or an equivalent maintainer-reviewed fix lands. The likely implementation is to align `.sidebar-markdown .markdown-inline-image` with the main chat limits, add a small CSS/style regression assertion if feasible, and verify Web UI chats or markdown sidebars with image-heavy content no longer expand images after rerenders.
