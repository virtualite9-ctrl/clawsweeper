## Summary

Keep #65691 open. Current main still lacks Slack message search support: there is no `searchSlackMessages`/`searchMessages` implementation, Slack action runtime handler, Slack message-action dispatch branch, or Slack discovery entry for `search`. The PR remains a focused implementation attempt rebuilt from closed-unmerged #58143, not something already shipped or superseded. It still needs maintainer review because the provided review context flags unresolved correctness/UX questions around Discord `--guild-id`, Slack channel scoping, and user-token-only account discovery.

## Best Possible Solution

Keep the PR or a successor open and finish it in the bundled Slack plugin: add a tested `search.messages` action/runtime path, expose `search` only for runnable account/token configurations, preserve or replace Discord `--guild-id` validation with a clear provider-specific error, and resolve Slack channel scoping by accepting channel names or resolving IDs before appending search syntax.
