## Summary

Keep open. Current `main` still builds Feishu markdown cards with the same `schema: "2.0"` plus `body.elements` shape reported in #53771, and the auto-card paths for code blocks/tables still route through that builder. The latest release tag provided (`v2026.4.22`, `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`) contains the same implementation, so there is no evidence this was fixed on `main` or shipped in a release.
