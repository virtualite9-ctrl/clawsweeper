## Summary

Keep open. Current main already includes the related `autoThreadName: "generated"` feature from PR #43366, but the requested `threadPromptGuideline` override is not implemented: Discord thread title generation still uses a fixed hardcoded prompt, and the Discord channel config schema/types expose no field for customizing that prompt.
