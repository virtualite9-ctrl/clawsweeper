## Summary

Current main already implements the issue's fallback request: `tools.exec.notifyOnExit` disables the `Exec finished` / `Exec denied` system events that WebUI surfaces, and the same behavior is present in release `v2026.4.23`.

## Evidence

- **Gateway suppresses exec lifecycle system events when configured off:** `handleNodeEvent` checks `cfg.tools?.exec?.notifyOnExit !== false` before enqueueing `exec.started`, `exec.finished`, or `exec.denied` system events, so setting it to `false` stops those WebUI-visible messages.
  - file: [src/gateway/server-node-events.ts:632](https://github.com/openclaw/openclaw/blob/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46/src/gateway/server-node-events.ts#L632)
  - sha: [ec3dbd22a4de](https://github.com/openclaw/openclaw/commit/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46)
- **Config surface exposes the user-facing toggle:** The tool config type includes `notifyOnExit?: boolean` with a comment describing it as the switch for exit system events/heartbeats.
  - file: [src/config/types.tools.ts:272](https://github.com/openclaw/openclaw/blob/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46/src/config/types.tools.ts#L272)
  - sha: [ec3dbd22a4de](https://github.com/openclaw/openclaw/commit/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46)
- **Docs describe the option:** The exec docs document `tools.exec.notifyOnExit` as the setting that controls whether backgrounded exec sessions enqueue a system event on exit.
  - file: [docs/tools/exec.md:95](https://github.com/openclaw/openclaw/blob/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46/docs/tools/exec.md#L95)
  - sha: [ec3dbd22a4de](https://github.com/openclaw/openclaw/commit/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46)
- **Regression tests cover suppression:** Gateway tests explicitly assert that `exec.started`, `exec.finished`, and `exec.denied` do not enqueue system events when `notifyOnExit` is `false`.
  - file: [src/gateway/server-node-events.test.ts:341](https://github.com/openclaw/openclaw/blob/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46/src/gateway/server-node-events.test.ts#L341)
  - sha: [ec3dbd22a4de](https://github.com/openclaw/openclaw/commit/ec3dbd22a4dea8da4bcc0d0832da48a83ae27d46)
- **Latest release contains the same implementation:** The `v2026.4.23` tag includes the same `notifyOnExit` guard in `src/gateway/server-node-events.ts` and the corresponding docs entry in `docs/tools/exec.md`.
  - file: [src/gateway/server-node-events.ts:632](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/gateway/server-node-events.ts#L632)
