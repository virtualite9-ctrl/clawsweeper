## Summary

Keep this PR open. Current main still contains the reported ClawHub parser bug, the PR is paired with open bug #56579 by the same author, and the PR remains the direct implementation candidate even though it now needs a rebase because the parser moved from src/infra/clawhub.ts to src/infra/clawhub-spec.ts.

## Best Possible Solution

Keep this PR open and ask for a focused rebase that applies the trailing-at rejection in src/infra/clawhub-spec.ts, preserves valid scoped package parsing including clawhub:@scope/pkg and clawhub:@scope/pkg@1.2.3, adds the regression tests in src/infra/clawhub.test.ts, and then lands it as the implementation for #56579 once the relevant changed gate passes.
