## Summary

Keep this PR open. Current main still lacks the Android/Termux PREFIX fallback for home-directory resolution, while this PR is a focused implementation candidate for a linked crash report and includes regression coverage for the normal fallback, precedence, generic-chroot rejection, and tilde-expansion edge case.

## Best Possible Solution

Keep this PR open, refresh it against current main, and review or land the focused Termux home-resolution fix with its regression tests. If maintainers want stricter safety, tighten `resolveTermuxHome` to recognize the canonical Termux prefix shape while preserving `OPENCLAW_HOME`, `HOME`, `USERPROFILE`, `OPENCLAW_STATE_DIR`, and `OPENCLAW_CONFIG_PATH` precedence.
