## Summary

Keep this issue open. Current main now fixes the thinking-default display path in gateway/session rows and UI tests, but the requested reasoning-default behavior is still not implemented: `chat.history` does not expose an effective reasoning level, the chat view still falls back to persisted session `reasoningLevel` or `off`, and `agents.defaults.reasoningDefault` is still not a supported global config key in the schema/docs.

## Best Possible Solution

Keep the issue open. Review #66098 or a successor against the newer thinking-default fixes, then finish the missing reasoning path by exposing an effective fresh-session reasoning level through the gateway/UI contract, rendering it in Control UI, adding regression tests, and explicitly deciding whether to support `agents.defaults.reasoningDefault` globally or document that only `agents.list[].reasoningDefault` is valid.
