## Summary

Keep open. Current main has the thinking-default display path partly covered, but the requested reasoning-default behavior is still missing: `chat.history` does not return an effective `reasoningLevel`, the Control UI still falls back to the persisted session row or `off`, and `agents.defaults.reasoningDefault` is not a supported global config key. Related #66098 is an open partial implementation candidate, not a merged/superseding fix.

## Best Possible Solution

Keep this issue open. Review and either land #66098 or a rebased successor for the effective chat-default plumbing, then finish the missing reasoning path by returning an effective `reasoningLevel` through `chat.history`, storing/rendering it in Control UI, adding gateway and UI regression tests, and explicitly deciding whether `agents.defaults.reasoningDefault` should be supported globally or documented as per-agent only.
