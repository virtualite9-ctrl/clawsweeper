## Summary

Current `main` already uses the corrected `openclaw/plugin-sdk/*` imports in `extensions/xai`, exports the reported SDK subpaths from the root package, and includes a compatibility test that keeps scoped and unscoped plugin-sdk aliases aligned. The shipped `v2026.4.23` tag already contains those corrected xai imports, so the reported build break appears fixed on main and in a release.
