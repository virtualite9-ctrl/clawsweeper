## Summary

Keep #62099 open. Current main still has the reported failure path: successful gateway runs await auth-profile bookkeeping, `markAuthProfileGood` / `markAuthProfileUsed` / `markAuthProfileFailure` still can propagate persistence failures through fallback saves, and the linked fix PR #67077 remains open/unmerged.
