## Summary

Keep open. Current `main` still overwrites explicit model `input` with the implicit catalog value, and the existing test suite on `main` asserts that overwrite behavior. The request is not implemented on `main`. At the same time, the proposed one-line PR change appears unsafe on call paths that pass a defaulted `loadConfig()` object without a runtime source snapshot, so this PR should be revised rather than closed.
