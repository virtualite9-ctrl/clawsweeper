## Summary

Keep open. Current `main` still has the exact early-return path the report calls out: when `message.is_from_me` is true and `isSelfChat` is false, iMessage inbound processing returns `{ kind: "drop", reason: "from me" }` before any echo-cache fallback runs. The later `selfChatCache` check only applies after that branch, so it cannot rescue a misclassified self-chat message. Current regression tests also preserve immediate-drop behavior for ambiguous/non-self detections, and `main` does not differ from the latest release on these files.
