## Summary

Current `main` already implements the substance of this request: enabled bundled plugins auto-repair missing runtime dependencies before import, `openclaw doctor` scans and repairs missing bundled plugin runtime deps, gateway startup can rebuild missing Control UI assets in source installs, and release verification now fails closed on missing Control UI payloads or bundled runtime sidecars.
