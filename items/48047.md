## Summary

Current `main` no longer has a Browser Relay Server on port 18792. The legacy Chrome extension relay path was removed, configs are migrated away from `driver: "extension"`/`browser.relayBindHost`, and the supported browser modes are the loopback control service plus managed local CDP or Chrome MCP `existing-session` attach. The report's expected behavior is obsolete against current `main`.
