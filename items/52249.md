## Summary

Keep this issue open. The original direct-resume root-cause text is stale, but the current issue thread describes an unresolved main-branch gap: ACP parent completion still relies on a session-scoped system event plus heartbeat wake, and that path can be skipped when no heartbeat handler/agent is active. The issue is also paired with open same-author PR #60402, which is an implementation candidate for the heartbeat-disabled ACP wake path, and the latest comment broadens the symptom to native `sessions_spawn` yield completion on `2026.4.23`.

## Best Possible Solution

Keep this issue open. The best maintainer path is to review, rebase, and either land or close PR #60402, while adding regression coverage that proves child completion wakes a yielded parent when periodic heartbeat is disabled. Given the latest native-spawn report, maintainers should also consider whether the durable fix belongs in a shared completion/yield wake primitive rather than only the ACP stream relay.
