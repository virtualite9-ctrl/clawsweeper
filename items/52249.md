## Summary

The current issue should stay open. Current `main` still routes ACP parent completion through session-scoped system events plus heartbeat wake delivery, and the issue has a same-author open PR candidate (#60402) plus fresh reproduction discussion for heartbeat-disabled/native-spawn yield behavior. The original direct-resume file/path is stale, but the remaining yielded-parent wake gap is still actionable in this repository.

## Best Possible Solution

Keep this issue open. Review, rebase, and either land or close PR #60402, or replace it with a shared yielded-parent completion wake primitive. The accepted fix should add regression coverage proving ACP child completion wakes a yielded parent when heartbeat is disabled or absent, and should evaluate the newer native-spawn yield report so the solution is not narrowly ACP-only if the bug is shared.
