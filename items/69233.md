## Summary

Current `main` does not implement the proposed WhatsApp group `admin` privacy control, and the request remains a plausible bundled-WhatsApp feature gap rather than cleanup-ready noise. Existing controls cover `groupPolicy`, `groupAllowFrom`, `requireMention`, and `toolsBySender`, but there is no per-group admin field or sender-aware admin bypass for mention or command gating. The issue also has a linked open PR (#69297) in the provided timeline, which is additional evidence that useful work may still exist here.
