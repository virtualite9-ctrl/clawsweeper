## Summary

Keep open. Current `main` still has a plausible live bug in the macOS `Launch at login` flow: the UI state is derived from whether the launchd job is currently loaded, not from persisted autostart configuration, and startup can actively remove the LaunchAgent plist after a false status read. That matches the report’s symptom of the toggle appearing off again after update/reload drift.

## Evidence

- **Status check uses live launchd state, not persisted autostart intent:** `LaunchAgentManager.status()` returns `false` unless both the plist exists and `launchctl print gui/$UID/ai.openclaw.mac` succeeds. If an update or relaunch leaves the plist present but the job temporarily unloaded, the UI will read as off even though autostart configuration still exists.
  - file: [apps/macos/Sources/OpenClaw/LaunchAgentManager.swift:9](https://github.com/openclaw/openclaw/blob/22f23fa5ab6f625835bfc3dbaba73c200775027c/apps/macos/Sources/OpenClaw/LaunchAgentManager.swift#L9)
  - command: `nl -ba apps/macos/Sources/OpenClaw/LaunchAgentManager.swift | sed -n '1,120p'`
  - sha: [22f23fa5ab6f](https://github.com/openclaw/openclaw/commit/22f23fa5ab6f625835bfc3dbaba73c200775027c)
- **Startup reload path can reapply a false read:** `AppState` initializes `launchAtLogin` to `false`, then asynchronously assigns `LaunchAgentManager.status()` back into the property. After `isInitializing` flips to `false`, the property `didSet` calls `AppStateStore.updateLaunchAtLogin(...)`, so a late `false` assignment on startup can immediately run the disable path.
  - file: [apps/macos/Sources/OpenClaw/AppState.swift:57](https://github.com/openclaw/openclaw/blob/22f23fa5ab6f625835bfc3dbaba73c200775027c/apps/macos/Sources/OpenClaw/AppState.swift#L57)

- **Disable path deletes the LaunchAgent plist:** `LaunchAgentManager.set(enabled: false, ...)` removes `~/Library/LaunchAgents/ai.openclaw.mac.plist`. Combined with the startup flow above, a false status read does not just affect the toggle UI; it can clear future launch-at-login persistence.
