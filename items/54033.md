## Summary

Current `main` already implements the requested continuity path: OpenClaw has pre-compaction hooks, a default-on silent pre-compaction memory flush that writes durable notes, persisted compaction summaries/checkpoints, and post-compaction context reinjection. The shipped design uses transcript/session-store persistence plus memory files rather than a separate `session-cache.json`, but it covers the issue’s requested behavior well enough.
