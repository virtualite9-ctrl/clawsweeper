## Summary

Keep open. Current main has built-in heartbeat skip mechanisms via `HEARTBEAT.md` emptiness and due-task checks, but it does not implement a first-class `preHook`/pre-run command gate for heartbeat or cron, and the existing hooks system does not expose cron/heartbeat trigger-phase events to cover this request.

## Evidence

- **Heartbeat config has no preHook surface:** The heartbeat config type and runtime schema list fields such as `every`, `model`, `prompt`, `timeoutSeconds`, `lightContext`, and `isolatedSession`; there is no `preHook` field.
  - file: [src/config/types.agent-defaults.ts:344](https://github.com/openclaw/openclaw/blob/f3c37a946cc49d26e5fe65a1043f7286a86aa4a4/src/config/types.agent-defaults.ts#L344)
  - command: `nl -ba src/config/types.agent-defaults.ts | sed -n '340,385p'`
  - sha: [f3c37a946cc4](https://github.com/openclaw/openclaw/commit/f3c37a946cc49d26e5fe65a1043f7286a86aa4a4)
- **Heartbeat runtime only supports built-in file/task gating:** `runHeartbeatOnce` preflights for `HEARTBEAT.md` and skips only for built-in reasons like `empty-heartbeat-file` and `no-tasks-due`; there is no command execution or exit-code mapping step before the run.
  - file: [src/infra/heartbeat-runner.ts:550](https://github.com/openclaw/openclaw/blob/f3c37a946cc49d26e5fe65a1043f7286a86aa4a4/src/infra/heartbeat-runner.ts#L550)

- **Heartbeat skip behavior is limited to HEARTBEAT.md task mode:** Current docs describe skipping only when `HEARTBEAT.md` is empty or when its `tasks:` block has no due items; this is a different feature from a configurable pre-run script gate.
  - file: [docs/gateway/heartbeat.md:345](https://github.com/openclaw/openclaw/blob/f3c37a946cc49d26e5fe65a1043f7286a86aa4a4/docs/gateway/heartbeat.md#L345)

- **Cron protocol/schema has no preHook field:** The cron protocol schema accepts only `systemEvent` payloads or agent-turn payloads with fields like `message`, `model`, `thinking`, `timeoutSeconds`, `lightContext`, and `toolsAllow`; there is no `preHook` object or exit-code policy in the public job shape.
