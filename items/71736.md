## Summary

Keep #71736 open. Current main still has adjacent plugin slash-command discovery and generic plugin approval rendering, but not the requested Control UI contribution descriptor/registry surface for chat modes, input guards, event classifiers, status surfaces, or plugin-owned UI actions. The linked #72082 implementation PR is open and not present on current main, so the RFC remains an active core SDK/UI design item rather than cleanup material.

## Best Possible Solution

Keep #71736 open as the focused Control UI contribution RFC. The best path is to review or refine #72082 as the candidate implementation, ensuring it adds a narrow additive descriptor contract, Gateway projection, safe host-owned renderers, plugin enable/disable semantics, sanitization, and fixture coverage for chat modes, approval/status surfaces, input guards, event classification, and plugin-owned actions.
