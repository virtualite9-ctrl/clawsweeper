## Summary

Keep open. Current `main` still exits `0` for supervised restarts, so this is not already implemented, and the PR remains useful; the main caveat is that its rationale overstates the default Linux service policy because OpenClaw’s canonical systemd unit uses `Restart=always`, not `Restart=on-failure`.
