## Summary

Keep open. Current `main` still has the reported behavior in code: the configure-wizard remove flow explicitly leaves credentials on disk, WhatsApp persisted auth still counts as configured, and gateway startup auto-enables and writes that config back out.

## Evidence

- **Configure wizard still leaves credentials behind:** The interactive remove flow advertises `Deletes tokens + settings from config (credentials stay on disk)` and then notes `credentials/sessions on disk are unchanged` after deletion.
  - file: [src/commands/configure.channels.ts:91](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/commands/configure.channels.ts#L91)
  - command: `nl -ba src/commands/configure.channels.ts | sed -n '80,130p'`
  - sha: [8866544ffefb](https://github.com/openclaw/openclaw/commit/8866544ffefb42ca234c66c337134fb60743d8d0)
- **CLI delete path only removes config + optional lifecycle hook:** `channelsRemoveCommand(... --delete)` calls `plugin.config.deleteAccount(...)`, then optional `plugin.lifecycle.onAccountRemoved`, then writes the new config. There is no core credential purge here.
  - file: [src/commands/channels/remove.ts:146](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/src/commands/channels/remove.ts#L146)

- **WhatsApp plugin has no removal cleanup hook:** The WhatsApp plugin lifecycle block only exposes `detectLegacyStateMigrations`; there is no `onAccountRemoved` cleanup to delete persisted auth on channel removal.
  - file: [extensions/whatsapp/src/channel.ts:172](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/extensions/whatsapp/src/channel.ts#L172)

- **Persisted auth probe still treats leftover creds as configured WhatsApp state:** Bundled metadata declares WhatsApp `persistedAuthState` via `./auth-presence#hasAnyWhatsAppAuth`, and that checker scans WhatsApp auth dirs and returns true when any `creds.json` is present.
  - file: [extensions/whatsapp/package.json:42](https://github.com/openclaw/openclaw/blob/8866544ffefb42ca234c66c337134fb60743d8d0/extensions/whatsapp/package.json#L42)
