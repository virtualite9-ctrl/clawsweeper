## Summary

Keep PR #69982 open. Current main still has the reported listener cleanup gap: Cancel and Delete remove the popover without removing the document-level capture listener, while only the outside-click path removes it. The PR is a focused fix for an observable leak and is not superseded by the related delete-confirmation cleanup already shipped.

## Best Possible Solution

Keep the PR open for maintainer review. The best path is to land the shared dismissPopover cleanup, address the minor wrapper-function nit if desired, and add a focused regression test around document listener cleanup in ui/src/ui/chat/grouped-render.test.ts.
