## Summary

Keep #27526 open. Current main has useful partial building blocks for sessions, session switching, labels/search, model/thinking overrides, checkpoints, multi-agent routing, and sub-agent background runs, but it still does not ship the requested WebChat conversation directory with folders/tags/archive/pinning/recommendations or the multi-AI comparison, roles, consensus, agenda/minutes, and committee UX. The issue also has explicit maintainer discussion marking it as the canonical tracker after duplicate #27603 was consolidated here.

## Best Possible Solution

Keep #27526 as the canonical product tracker. The best path is to split implementation into explicit milestones: first add session metadata/protocol and storage support for folders, tags, archive state, pinning, richer search/recommendations, and WebChat/Control UI directory navigation; then design and ship a deliberate multi-model collaboration UX with role definitions, side-by-side comparison, consensus/conflict handling, agenda/minutes, and task extraction. Existing session listing, checkpoint, multi-agent, and sub-agent work can be referenced as building blocks but should not close the umbrella until the central user-facing workflow ships.
