## Summary

Keep #27526 open. Current main still has useful foundations for sessions, session search/listing, per-session overrides, multi-agent isolation, and sub-agent orchestration, but it does not implement the requested WebChat conversation directory or multi-AI collaboration product surface. The issue also has a maintainer comment explicitly keeping it as the canonical tracker after duplicate #27603 was consolidated here.

## Best Possible Solution

Keep #27526 open as the canonical product tracker. The best path is to split it into explicit milestones: extend session protocol/storage with folder, tag, archive, persistent pin, richer search, and recommendation metadata; add WebChat/Control UI directory navigation; then design the multi-model collaboration UX with roles, side-by-side comparison, consensus/conflict handling, agendas/minutes, committee templates, and task extraction/project tracking. Existing session labels/search, checkpoints, multi-agent routing, and sub-agents should be treated as foundations, not completion.
