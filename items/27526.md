## Summary

Keep #27526 open. Current main has useful session primitives, session switching, labels/search, model/thinking overrides, checkpoint branching, and subagent lineage, but it still does not implement the requested WebChat conversation directory with folders/tags/archive/pinning/recommendations or the multi-AI comparison, roles, consensus, agenda/minutes, committee, and project-tracking UX. The discussion also marks this as the canonical tracker after duplicate #27603 was consolidated here.

## Best Possible Solution

Keep #27526 as the canonical product tracker. The best path is to split implementation into explicit milestones: add protocol/storage/session metadata for folders, tags, archive state, pinning, richer search and recommendations; ship WebChat/Control UI directory navigation; then design multi-model collaboration UX with role definitions, side-by-side comparison, consensus/conflict handling, agendas/minutes, and task extraction. Existing session and subagent work should be treated as building blocks, not completion.
