## Summary

Current main still lacks the requested legacy Word/OLE binary guard in the media-understanding prompt embedding path, and #54380 remains open as an unmerged implementation candidate for this issue. The issue should stay open until that PR or an equivalent scoped fix lands.

## Best Possible Solution

Keep this issue open and review/land #54380, or an equivalent scoped patch, that adds application/x-cfb and application/msword to the media-understanding binary MIME guard in src/media-understanding/apply.ts with focused regression coverage in src/media-understanding/apply.test.ts. If #54380 is no longer the preferred patch, close or replace that PR first, then close this issue only after the fix is on main.
