## Summary

Keep open. Current main does not implement the generic plugin host-hook SDK contracts, and PR #72082 is an active, unmerged implementation with concrete review blockers rather than a cleanup-close candidate.

## Best Possible Solution

Keep #72082 open for maintainer review or a follow-up patch.
