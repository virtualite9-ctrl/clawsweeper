## Summary

Current `main` does not implement prompt replacement for `before_agent_start` or `before_prompt_build`. The hook contract and both runner paths still only support `prependContext` plus system-prompt fields, so #8176 remains a valid open enhancement request.

## Evidence

- **Hook types still lack prompt replacement:** `PluginHookBeforePromptBuildResult` only defines `systemPrompt`, `prependContext`, `prependSystemContext`, and `appendSystemContext`, and `PluginHookBeforeAgentStartResult` is just that prompt-build shape plus model/provider overrides. There is no hook-level `promptOverride` field.
