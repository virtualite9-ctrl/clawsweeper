## Summary

Current `main` already disambiguates multi-agent session dropdown labels in the web UI, so PR #47822 is obsolete. The session label logic now lives in `ui/src/ui/chat/session-controls.ts`, where duplicate labels across agent groups are prefixed with the agent group label and further deduplicated as needed. Node tests on `main` cover the exact reported case of duplicate `main` sessions for multiple agents and assert labels like `Deep Chat (alpha) / main` and `Coding (beta) / main`, which satisfies the user-facing goal of distinguishing sessions in multi-agent setups.
