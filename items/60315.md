## Summary

Current `main` now stages agent-reply `MEDIA:` paths through the shared outbound attachment pipeline and the WhatsApp auto-reply path explicitly loads and sends media attachments, with targeted tests covering relative `MEDIA:./...` replies and WhatsApp delivery. The reported gap between CLI `message send --media` and normal agent replies appears implemented on `main` after the report.
