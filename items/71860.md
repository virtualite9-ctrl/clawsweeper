## Summary

Keep PR #71860 open. Current main still relays successful structured background exec completions when a chat target is available, and existing tests explicitly cover that visible-delivery behavior. The PR is a focused, recent fix for a real remaining heartbeat policy gap; the supplied discussion shows the mixed-batch leak review finding was addressed on the PR branch.

## Best Possible Solution

Keep #71860 open for maintainer review. The best path is either to land this focused heartbeat fix with its regression coverage after normal validation, or explicitly supersede it with a broader typed event-audience implementation under #69492. Current main does not justify closing the PR as implemented, obsolete, or duplicate.
