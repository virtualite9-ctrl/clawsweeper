## Summary

Keep open. Current `main` still intentionally warns on version-managed Node runtimes, still installs the gateway using the active `node` executable when supported, and still exposes no user-facing CLI option to pin a different Node path or suppress the warning. The report matches current behavior, and the follow-up `asdf` comment is consistent with the same code path.
