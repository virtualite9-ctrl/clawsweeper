## Summary

Current main already ships official GUI surfaces for the reported need: a browser Control UI for chat/config/skills/admin tasks and an in-tree macOS companion app with onboarding, gateway control, config editing, skills, and exec approvals. Native Windows/Linux companion apps are still planned, but the core request for an official GUI/companion app is already implemented well enough to close.
