## Summary

Current `main` already contains part of this PR's Talk Mode work, but not the feature set described in PR #53553. The double-playback/system-voice fallback split and the language-aware system-TTS watchdog are present, yet the CJK silence clamp, phase-transition sound toggles, Right Option interrupt monitor/settings, and Push-to-Talk/Talk Mode coordination UI are not on `main`. Because substantial user-facing work remains unlanded, this PR should stay open rather than be closed as implemented or obsolete.
