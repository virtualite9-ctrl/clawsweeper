## Summary

Keep #11146 open. Current main still only has the existing Cmd/Ctrl+K command palette shortcut plus lower-level agent creation APIs; it does not implement the requested dashboard-wide keyboard navigation or a Control UI New Agent wizard.

## Best Possible Solution

Keep #11146 open, or split it into two maintainer-tracked dashboard issues. Keyboard work should add a dashboard-level shortcut dispatcher with focus guards and focused tests for chat/navigation actions. Agent creation work should add a Control UI New Agent wizard that reuses the existing agents.create, agents.files.*, config, and session APIs, supports workspace/bootstrap customization such as SOUL.md and USER.md, and includes UI tests plus user-facing docs.
