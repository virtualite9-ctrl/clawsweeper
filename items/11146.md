## Summary

Keep #11146 open. Current main still lacks the requested dashboard-level keyboard navigation and dashboard New Agent creation wizard; it only has the existing Cmd/Ctrl+K command palette plus lower-level CLI/RPC agent creation and file-editing APIs.

## Best Possible Solution

Keep the issue open or split it into two maintainer-tracked tasks. The keyboard task should add a dashboard-level shortcut dispatcher with focus guards and tests for chat/navigation actions. The agent task should add an Agents dashboard creation wizard that reuses agents.create, agents.files.*, config/session APIs, supports workspace/bootstrap customization such as SOUL.md and USER.md, and includes focused Control UI tests plus user-facing docs.
