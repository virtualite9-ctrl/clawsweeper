## Summary

Keep #11146 open. Current main still lacks the requested dashboard shortcut set and the dashboard New Agent creation wizard. The repo has Cmd/Ctrl+K command palette support, lower-level `agents.create`/agent file APIs, and CLI/docs workflows, but not the user-facing dashboard workflows described in the issue.

## Best Possible Solution

Keep the issue open, or split it into two focused maintainer-tracked issues. The keyboard task should add a dashboard-level shortcut dispatcher with focus guards and tests for chat/navigation actions. The agent task should add a Control UI New Agent wizard that reuses `agents.create`, `agents.files.*`, config/session APIs, supports workspace/bootstrap customization such as `SOUL.md` and `USER.md`, and includes focused UI tests plus user-facing docs.
