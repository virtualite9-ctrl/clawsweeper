## Summary

Keep open. Current `main` still strips inline `/queue` directives when they accompany prompt text, the latest shipped release still contains that logic, docs advertise `/queue` as an inline directive, and existing tests do not cover the mixed inline `/queue` + prompt path this PR targets.
