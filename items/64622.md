## Summary

Keep this PR open. Current main still lacks the central A2A timeout/provenance, webchat-backed main-session metadata cleanup, and outbound continuity changes, but the submitted diff is not merge-ready because two review blockers are source-backed and the branch mixes unrelated docs-template policy changes into the runtime fix.

## Best Possible Solution

Keep this PR open for revision or replacement. A mergeable path should split out the docs-template commit, scope the timeout floor to true cross-agent sends with focused zero/short/long tests, add explicit webchat-backed main-session metadata cleanup in gateway persistence with regression coverage, and preserve outbound session continuity only when it does not overwrite the resolver-selected cross-target route metadata.
