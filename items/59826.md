## Summary

Current main already routes OpenAI-compatible reasoning deltas into non-user-visible thinking events, consumes those events in the embedded reply path, and retains tag-stripping safeguards. The issue discussion also narrowed the remaining leak to an older vendored OpenClaw copy inside `lossless-claw-enhanced`, with a follow-up plugin PR requiring `openclaw >= 2026.4.2`.
