## Summary

Current `main` has partial mitigations for busy-session races, but it does not implement the issue's requested behavior well enough to close. Heartbeats are explicitly deferred when the main lane or target session lane is busy, and the visible queued-message UX is limited to Control UI chat rather than same-channel heartbeat/prompt conflicts.
