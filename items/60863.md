## Summary

Keep open. Current `main` already covers much of the user need through the bundled `qwen` provider for Alibaba DashScope/Qwen endpoints, but it does not contain a dedicated `bailian` provider. The PR also still has a substantive cache-boundary concern against current `main`, so there is not strong evidence for any close reason.
