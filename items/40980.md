## Summary

Current `main` no longer ships or exercises the legacy Browser Relay extension path that this report depends on. The codebase has migrated host-local browser attach to Chrome MCP `existing-session`, removed extension-relay config, and treats the old relay hook as an intentional no-op, so the reported dark-to-light attach regression is obsolete against current `main`.
