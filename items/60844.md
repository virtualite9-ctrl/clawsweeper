## Summary

Keep open. Current `main` still implements heartbeat session isolation as an explicit opt-in, and the surrounding tests/docs/type comments all encode that default-false behavior. This PR is therefore neither already implemented nor stale cleanup material; it remains an unmerged behavior change that still needs normal maintainer review.
