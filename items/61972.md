## Summary

Keep open. Current `main` does not implement the PR's requested `baseUrl` overrides for Gemini web search, xAI web search, or `x_search`, and the config/schema surfaces for those settings are still absent. The PR also appears to need a rebase onto current runtime-file splits, but the underlying feature request remains valid.
