## Summary

Keep open. Current `main` only partially covers the PR’s goal: shared assistant-text extraction now strips reasoning tags, but the specific last-mile paths this PR targets are still not implemented on `main`. `sanitizeUserFacingText` still strips only `<final>` and internal runtime context, `normalizeReplyPayload` and streaming reply normalization still rely on that sanitizer, and Gateway `emitChatDelta` still forwards deltas without any reasoning-tag/stateful streaming filter.
