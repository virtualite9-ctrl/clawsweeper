## Summary

Keep this issue open. Current main still injects bootstrap/project context as path/content text without inline source, injectedAt/injected_at, volatile, or ctx:provenance metadata. The same author also has open PR #54830 as the Phase 1 implementation candidate, so the issue should not be closed before that PR is reviewed, merged, or explicitly rejected.

## Best Possible Solution

Keep this issue open and review PR #54830 as the immediate Phase 1 candidate. If maintainers accept the direction, land a focused provenance-tagging implementation with type, bootstrap assembly, prompt rendering, and tests; then split memory freshness, prompt decision rules, and orchestration-level auto-refresh into narrower follow-ups. If maintainers reject inline tags, close or redirect PR #54830 with the preferred design before resolving this tracker.
