## Summary

Keep this issue open. Current main still does not auto-propagate gateway NODE_OPTIONS or proxy env into MCP stdio children, and the linked implementation candidate #65574 is open and unmerged with closing syntax for this issue.

## Best Possible Solution

Keep the issue open and review #65574 as the implementation candidate. A conservative fix would likely propagate only safe proxy-related env as lowest-priority defaults, keep NODE_OPTIONS blocked unless maintainers explicitly approve a safer startup-hook path, and update docs.openclaw.ai/cli/mcp to state exactly what MCP stdio children inherit.
