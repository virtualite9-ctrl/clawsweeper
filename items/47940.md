## Summary

Keep open. Current main still runs heartbeat in the main session by default, and the context filter only removes no-op heartbeat prompt + HEARTBEAT_OK pairs. Meaningful heartbeat outputs remain in history unless `heartbeat.isolatedSession` is explicitly enabled, so there is not strong evidence that the reported alternating `sent`/`ok-token` behavior on the default path is fixed on main.
