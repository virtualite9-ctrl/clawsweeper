## Summary

Keep open. Current `main` does not implement `tools.exec.shell`, and the PR as submitted is not conservative-close material because it is both unmerged and incomplete: the current config/types/docs path has no shell override knob, while the proposed change would still miss PTY and gateway-approval execution paths.
