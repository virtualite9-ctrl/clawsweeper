## Summary

Keep open. Current `main` has plugin and node infrastructure adjacent to this request, but I did not find a shipped native desktop/computer-use tool surface comparable to the issue description. The repo vision also explicitly lists better computer-use as a next priority, so this is still in-scope rather than clearly a ClawHub-only request.
