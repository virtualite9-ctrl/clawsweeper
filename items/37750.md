## Summary

Keep open. Current `main` still throws on missing explicit API keys in both the PDF media helper and the image media runtime, and this PR only patches the PDF helper. The proposed change does not fully implement the request described in the PR body and may not match the repo’s current aws-sdk runtime-auth pattern.
