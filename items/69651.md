## Summary

Keep open. Current `main` still writes file-log JSON lines and terminal console output without sink-level redaction, so this PR is not already implemented on `main`. The public logging docs also still describe redaction as tools-only, and the PR discussion includes an unresolved console leak concern for split key/value arguments.
