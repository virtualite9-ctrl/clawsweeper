Keep this PR open. Main only implements per-agent `agents.list[].reasoningDefault`, while the PR discussion shows explicit maintainer evaluation plus recent author follow-up rather than obsolete work.

If accepted, the implementation should align schema, exported types, generated schema/help/labels, docs, reply resolution, current-level/status/direct-status behavior, and tests covering directive/session/per-agent/global/model-default precedence plus sender authorization. If rejected, close with an explicit maintainer product decision and point users to the existing per-agent `agents.list[].reasoningDefault` surface.
