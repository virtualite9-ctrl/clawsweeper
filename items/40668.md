## Summary

Keep open. Current `main` has partial media-delivery improvements, but the issue's core agent-awareness gap is not clearly fixed across browser/canvas/nodes. `tts` still has the only explicit auto-delivery/`NO_REPLY` guidance, browser/canvas/nodes descriptions still omit it, and the system prompt documents only assistant-authored `MEDIA:` lines rather than tool-result auto-delivery.
