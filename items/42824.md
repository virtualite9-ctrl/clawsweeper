## Summary

Keep this PR open. Current main at 66f4b52d still lets IRC inherit the global block-streaming default when `channels.irc.blockStreaming` is unset, which is exactly the behavior this PR changes for the linked IRC cut-off report #42121. The proposed diff remains narrow and still useful; the security pass found no supply-chain-sensitive changes.
