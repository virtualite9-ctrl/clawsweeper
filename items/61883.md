## Summary

Current `main` no longer pulls `cli-highlight` into the TUI bootstrap path. The reported regression was against a release build that still eagerly loaded that parser in `src/tui/theme/theme.ts`, so the startup-path fix is already present on `main` even though I could not tie it to a shipped release from this checkout.

## Evidence
