## Summary

Current `main` does not already implement the PR’s core fix set. The code still validates session thinking against raw stored overrides in `sessions-patch`, still persists provider-prefixed model overrides without canonicalization in `sessions/model-overrides`, and still writes fallback thinking downgrades back to the session without checking whether the value was only default-derived. This PR is also brand new, so conservative cleanup should not auto-close it.
