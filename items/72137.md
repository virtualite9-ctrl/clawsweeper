## Summary

Keep PR #72137 open. Current main still has the missing-vs-empty-object behavior in `diffConfigPaths`, and `plugins.entries.*.config` paths still fall through to the restart rule. The PR is a focused, current bug fix for open issue #72061 with targeted regression tests, and the earlier array-semantics review concern was addressed in the latest PR context.

## Best Possible Solution

Keep the PR open for maintainer review and normal landing. The best path is to merge a narrow comparator fix with regression coverage that collapses missing vs empty plain-object phantom diffs while preserving explicit empty-array semantics, then close #72061 through the PR.
