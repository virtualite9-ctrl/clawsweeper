## Summary

Keep PR #72137 open. Current main still has the reported gateway reload-diff behavior: `diffConfigPaths` reports asymmetric missing-vs-empty container branches, and `plugins.*` paths are restart-classified. The PR is a focused, plausible fix for open bug #72061, but the unresolved review concern is also source-backed: `undefined` -> `[]` can be meaningful for plugin subagent `allowedModels`, so the patch likely needs a narrower/path-aware adjustment rather than closure.

## Best Possible Solution

Keep the PR open and ask for a revision that fixes #72061 without globally collapsing `undefined` and `[]`. A good direction is a narrower comparator rule for missing vs empty plain object on normalization-only config branches, or a path/schema-aware no-value helper, plus regression tests for the skill-toggle phantom `plugins.entries.<id>.config` case and a guard that `plugins.entries.<id>.subagent.allowedModels: []` is still reported as a real change.
