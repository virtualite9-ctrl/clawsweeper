## Summary

Keep open. Current `main` still suppresses WhatsApp metadata mentions whenever self-chat mode is active, and self-chat mode is still inferred from `allowFrom` containing the linked self number. That means the reported `normalizedMentionedJids`/`selfE164` match can still coexist with `wasMentioned=false` in the self-chat path, so this is not implemented-on-main or obsolete.
