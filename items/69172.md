## Summary

Keep open. Current main does not implement the PR’s stated Google/Gemini override behavior, and the related closed PR #61714 is not a merged canonical fix. The PR is not merge-ready as-is because it is broad/dirty and has unresolved review findings, but the central bug/feature request remains actionable in this repository.

## Best Possible Solution

Keep the PR open only as a useful signal/reproduction for the remaining Google provider gap. The maintainable fix should be a narrow Google plugin change that adds documented baseUrl/apiType resolution for the intended surfaces, preserves native Gemini defaults and Google host normalization, uses existing provider HTTP/SSRF/auth helpers, and adds focused tests for web search, embeddings, image generation/editing, and media understanding. Unrelated script, memory SDK, BytePlus/Volcengine, and accidental root-file changes should be dropped or split into separate maintainer-reviewed PRs.
