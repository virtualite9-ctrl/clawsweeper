## Summary

Keep this issue open. Current main has internal ClawHub owner/profile links and partial repository metadata rendering, but it does not add GitHub profile links beside usernames or make the profile page name/username link out to GitHub. ClawHub policy also says issues should stay open rather than be auto-closed.

## Best Possible Solution

Keep the issue open and implement a small core UI improvement: expose a verified GitHub profile URL for users, add an external GitHub icon/link next to owner badges and on the profile header name/handle, and make repository/source links more visible on skill detail/list surfaces when metadata provides them. Handle card layouts carefully because the card itself is already a link.
