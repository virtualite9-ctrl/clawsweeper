Keep open. Repository policy for openclaw/clawhub only permits auto-closing pull requests that are certainly implemented on main, and this item is an issue. Substantively, current main still supports the reported behavior: VirusTotal Code Insight can set a suspicious status before AV engine stats are considered, that scanner status becomes suspicious moderation state, and the skill header still renders a generic suspicious warning. ## Best Possible Solution

Keep this issue open for maintainer moderation and product review.
