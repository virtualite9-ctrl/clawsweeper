## Summary

Keep this PR open. Current main does not implement the PR’s intended deduplication: the shared `convex/lib/math.ts` and `convex/lib/math.test.ts` files are absent, and the duplicate `clampInt` helpers remain in the Convex modules. The PR also has a real behavior-review point because it would change `skills.ts` and `souls.ts` pagination limits from `Math.round` to `Math.trunc`.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to either land the deduplication with an explicit decision on the `Math.round` to `Math.trunc` behavior change, or update the PR so `skills.ts` and `souls.ts` preserve their previous rounding behavior while shared code covers the other modules.
