## Summary

Current main does not already implement this PR’s intended stats-semantics cleanup. The skill header, shared skill stats, and soul stats components still use the older Package icon and star-first ordering, and main has no installsTooltip utility or companion tests. Because this is an open ClawHub PR and the change is not shipped on main, it should stay open for maintainer review rather than be auto-closed.

## Best Possible Solution

Keep this PR open for maintainer review. The best path is to rebase/update it against current main, preserve visible accessible current/all-time install information, apply the Download/Users icon and metric-order cleanup consistently across SkillHeader, SkillStats, and SoulStats, add the focused installs formatting/regression tests, and keep unrelated formatting or backend churn out of the diff.
