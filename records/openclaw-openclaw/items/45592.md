## Summary

Keep this PR open. Current main has not implemented the proposed Anthropic 1M GA migration: the active bundled Anthropic provider still injects `context-1m-2025-08-07` for `context1m: true`, still strips that beta for OAuth tokens, and docs/tests still describe or assert the beta-gated behavior. The PR also closes the same-author open #45550, so it should remain as an implementation candidate until it is reviewed, rebased onto the current plugin path, merged, or explicitly rejected.

## Best Possible Solution

Keep this PR open as the paired implementation candidate for #45550. The useful path is to rebase the intent onto `extensions/anthropic/stream-wrappers.ts`, update the provider docs and relevant tests, and explicitly preserve or retire the legacy OAuth beta-filtering compatibility behavior based on current Anthropic API proof.
