## Summary

Keep this PR open. Current main still does not implement the proposed tailnet IPv6 normalization: `isLocalGatewayAddress` normalizes the candidate IP, but compares the tailnet IPv6 address through lowercase string equality instead of canonical IP normalization, so compressed and expanded forms can still differ. The PR is a focused implementation candidate, though it needs a rebase and maintainer review because the helper currently appears unused on main.

## Best Possible Solution

Keep the PR open as a plausible focused gateway correctness fix. Best maintainer path: rebase it on current main, decide whether `isLocalGatewayAddress` should remain part of the gateway net helper surface, and if so land the normalized tailnet comparison plus reliable regression tests. If the helper is intentionally obsolete, close only after replacing that with a cleanup PR that removes the unused helper.
