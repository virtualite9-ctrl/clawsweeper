## Summary

Keep this PR open. Current main still does not implement skill `references/` auto-load support, and the PR is not obsolete. The latest PR diff is focused and includes path and size guards, but it still needs maintainer review because references are only appended during sandbox skill sync, while normal/non-sandbox skill reads still point agents at the original `SKILL.md`.

## Best Possible Solution

Keep this PR open and ask for a small revision: make `references/` content available wherever the runtime points agents at a skill file, not only after sandbox sync, while preserving the latest path traversal and aggregate-size protections. Add regression coverage for default alphabetical loading, selective `references.autoLoad`, normal/non-sandbox reads, sandbox sync reads, path escape rejection, and combined-size limits.
