## Summary

Current main still matches the reported Telegram failure. Mention-gated Telegram group messages with media are downloaded into `allMedia`, but the no-mention skip path records only text history, the shared history/context types have no media fields, and later Telegram context construction only forwards current-turn plus reply media. The linked fixes #40470 and #40490 are closed unmerged, so there is no shipped implementation to close this issue against.

## Best Possible Solution

Keep this issue open. Implement the focused Telegram/shared-history fix: add optional media references to reply history and inbound history, persist or refresh media for skipped mention-gated Telegram group messages under a documented lifetime policy, merge replayed history media into later Telegram `MediaPaths`/`MediaTypes`, update prompt metadata if useful, and add a regression test for image-without-mention followed by an @mention asking about that image.
