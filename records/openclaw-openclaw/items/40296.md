## Summary

Keep this PR open. Current main has added a system-prompt cache boundary and now places Group Chat Context plus dynamic project context below it, but the PR’s central objective is still not implemented: runtime/channel-sensitive Messaging and Reactions guidance is still assembled before stable Project Context and before the cache boundary. ## Best Possible Solution

Keep this PR or its linked tracker #40256 open, then refresh the patch against current main.
