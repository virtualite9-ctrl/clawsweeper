## Summary

Keep this PR open. Current main still rejects empty `aws-sdk` auth in the media image and audio/video paths, while this PR is the active focused implementation candidate for the Bedrock credential-chain regression tracked by #72031. The earlier #72066 PR was closed unmerged and does not supersede this broader PR.

## Best Possible Solution

Keep this PR open for maintainer review and validation. The best path is to review or iterate on the focused implementation that carries the `aws-sdk` missing-key allowance through media image, MiniMax fallback, and audio/video execution, including the empty-key rotation contract, then merge it to close #72031 if checks pass.
