## Best Possible Solution

Keep this PR open. Adjust the raw precheck so it only rejects payloads that cannot fit by base64 expansion alone, or replace the estimate with measurement of the actual serialized node.invoke.result frame. Add/keep a near-limit valid snapshot regression test, then rerun the focused macOS runtime tests and shared OpenClawKit bridge/session tests before maintainer review and merge. After merge, #68181 can close as implemented.
