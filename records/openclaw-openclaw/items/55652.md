## Summary

Keep this PR open. Current main still has no Discord cross-instance channel ownership registry, and the relevant Discord message, autocomplete, and native slash-command paths still make local allowlist decisions that match the bug this PR is trying to address.

## Best Possible Solution

Keep this PR open for maintainer review or replacement. The best path is a Discord-plugin-owned fix under extensions/discord/src/monitor that performs robust per-instance and per-bot ownership checks before local allowlist denial in message preflight, autocomplete, and native slash-command dispatch, with regression coverage for same-token multi-instance routing and the authorization edge cases already raised in review.
