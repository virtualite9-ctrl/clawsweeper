## Summary

Keep open. Current main fixes or mitigates several reported sub-symptoms, but this is still a recent omnibus crash/performance/downstream-plugin report. The remaining QClaw/Windows Crashpad upload, 1.5GB memory-growth, WeChat delivery, and skill-interceptor update behavior are not proven fixed on main and are not fully covered by the closed related #68810.

## Best Possible Solution

Keep this issue open, then split it into actionable tracks: close or mark addressed the current-main gateway status, stale-plugin warning, and exec-notification pieces; route WeChat and skill-interceptor behavior to the owning plugins or downstream package unless a missing OpenClaw plugin API is identified; and request a current-release Windows/QClaw reproduction or downstream packaging tracker for Crashpad upload and memory-growth evidence.
