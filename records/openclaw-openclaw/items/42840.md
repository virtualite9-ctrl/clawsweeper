## Summary

Keep this issue open. Current main still does not implement MathJax, KaTeX, or another LaTeX math rendering layer in the built-in Control UI markdown/chat path, and the open KaTeX PR #67103 remains an implementation candidate rather than shipped behavior. Prior PR #61510 was closed unmerged, and the closed duplicate issues point back to the same user-facing gap rather than proving it is fixed.

## Best Possible Solution

Keep this as the canonical Control UI math-rendering request. Review and land #67103 or an equivalent focused implementation in the shared Control UI markdown renderer, with tests for inline math, display math, code spans/fences, invalid formulas, sanitizer behavior, and ordinary dollar-sign text; close only after the implementation is merged to main.
