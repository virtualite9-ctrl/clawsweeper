## Summary

Keep open. Current main still lacks the pre-hello close tolerance in both the one-shot gateway call wrapper and `GatewayClient`, while the PR branch contains the focused fix and maintainer follow-up says this branch should remain canonical.

## Best Possible Solution

Review and land this PR if the branch validation remains green, then close or supersede same-author PR #54774 as duplicate. The implementation should stay in `src/gateway/call.ts` and `src/gateway/client.ts`, keep the structured close-error approach, and retain the focused regression tests in `src/gateway/call.test.ts` and `src/gateway/client.test.ts`. Broader cron retry work can remain separate under #49109 unless maintainers explicitly consolidate it.
