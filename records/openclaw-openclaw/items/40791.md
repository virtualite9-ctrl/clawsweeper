## Summary

Keep this PR open. Current main does not include the external rescue watchdog, Claude runner, systemd timer templates, or rescue-watchdog docs, and the existing health, in-process channel monitor, Crestodian rescue, and rescue-bot guidance are adjacent rather than equivalent. The related core watchdog PR #46502 is still open and unmerged, so it does not make this external shell/systemd approach implemented or clearly superseded yet.

## Best Possible Solution

Keep this PR open for maintainer judgment. The best path is to decide whether OpenClaw wants an external shell/systemd rescue path alongside or instead of the core watchdog-service proposal in #46502. If the external path is wanted, rebase it onto current main, drop unrelated test-only churn, fix the suppression/incident-ID/overlap issues, harden state and incident-file handling, add focused shell or unit coverage, and document the security model for captured diagnostics and Claude runner permissions. If the core service becomes the canonical accepted design, close this PR only after that path is merged or maintainers explicitly mark it superseded.

## Risks / Open Questions

- The proposed watchdog intentionally executes an operator-configured shell runner and launches Claude Code with edit permission; that command-execution surface needs explicit maintainer approval and clear docs.
- The script sources persistent `state.env` and writes state/incidents without obvious owner, permission, lock, or symlink hardening in the proposed diff.
- Incident bundles include health/status output, gateway log tails, git status/diff stats, and runner output, so the docs should warn operators about sensitive local diagnostics.
- Open review comments identify real race/correctness issues around suppression reset, incident ID uniqueness, and overlapping runs before state is persisted.
- The PR overlaps with open core-service watchdog work in #46502 and still carries unrelated test-only type churn, so maintainers need to choose the product direction and keep the branch scoped.
