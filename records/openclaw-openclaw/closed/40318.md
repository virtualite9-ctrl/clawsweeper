## Summary

Keep open. Current `main` does not contain the PR's change: the equivalent Kilocode discovery code now lives in `extensions/kilocode/provider-models.ts`, and it still uses `entry.context_length || KILOCODE_DEFAULT_CONTEXT_WINDOW`. The proposed zero-context regression test is also not present on `main`. There is downstream normalization that rejects non-positive `contextWindow` values, so the practical impact looks limited, but that is not enough evidence to close this PR as implemented or obsolete.
