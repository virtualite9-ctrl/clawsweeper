## Summary

Keep this PR open as useful bug-fix evidence, but it should not be merged as-is. Current main moved Bonjour/CIAO handling into the bundled Bonjour plugin, and the active classifier still misses the exact upstream `IPv4 address changed from undefined to defined` assertion variant described by this PR. The fix should be ported into `extensions/bonjour` with focused tests.

## Best Possible Solution

Keep this PR open as a reference patch. Port the missing `IPv4 address changed from undefined to defined` handling into `extensions/bonjour/src/ciao.ts`, add focused coverage in `extensions/bonjour/src/ciao.test.ts` and the advertiser handler test if needed, then run the Bonjour/plugin test lane before closing the linked crash work as implemented.
